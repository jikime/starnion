# Starnion - Your Stellar Companion in Every Task

**Last Updated:** 2026-03-13

## 1. 프로젝트 개요

Starnion은 초개인화 AI 에이전트 플랫폼이다. AI 컴패니언 **Nion**이 사용자의 일상 전반을 지원하며, 금융 관리부터 일기 작성, 목표 달성, 일상 생활까지 폭넓은 영역을 커버한다.

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Starnion |
| **슬로건** | Your Stellar Companion in Every Task |
| **AI 컴패니언** | Nion |
| **핵심 도메인** | 금융, 일기/저널링, 목표 관리, 일상 생활 |
| **내장 스킬** | 46개 |
| **접근 채널** | Web UI, Telegram Bot |

## 2. 주요 기능

### 금융 관리 (Finance)

- 수입/지출 추적 및 분류
- 예산 설정 및 모니터링
- 금융 리포트 생성
- 소비 패턴 분석

### 일기/저널링 (Diary & Journaling)

- AI 기반 감정 분석
- 자동 일기 요약
- 패턴 인식 및 인사이트 제공
- 멀티미디어 일기 지원

### 목표 관리 (Goals)

- 목표 설정 및 추적
- 마일스톤 관리
- 진행률 시각화
- AI 기반 동기 부여 및 조언

### 일상 생활 (Daily Life)

- 대화형 AI 어시스턴트
- 이미지 생성 및 분석
- 음성 처리 (TTS/STT)
- 웹 검색 및 정보 조회
- 일정 및 루틴 관리

### 리포트 & 분석

- 주간/월간 활동 리포트
- 데이터 기반 인사이트
- 커스텀 리포트 생성

### 메모리 시스템

- 4계층 RAG 기반 메모리
- 장기/단기 기억 관리
- 대화 맥락 유지
- 사용자 선호도 학습

## 3. 사용자 가치

### 초개인화 경험

Nion은 사용자의 대화 패턴, 선호도, 습관을 학습하여 시간이 지날수록 더 정확하고 개인화된 응답을 제공한다. 단순한 챗봇이 아니라, 사용자를 이해하는 AI 컴패니언이다.

### 올인원 생활 관리

금융, 일기, 목표, 일상 생활을 하나의 플랫폼에서 관리할 수 있다. 도메인 간 데이터가 연결되어 종합적인 인사이트를 제공한다.

### 자연스러운 대화 인터페이스

복잡한 UI 조작 없이 자연어 대화만으로 모든 기능에 접근할 수 있다. "이번 달 식비 얼마 썼어?", "오늘 일기 써줘" 같은 자연스러운 요청으로 작업을 처리한다.

### 멀티 채널 접근성

Web UI와 Telegram 봇을 통해 언제 어디서나 Nion에 접근할 수 있다. 데스크톱에서는 웹 대시보드로 상세한 데이터를 확인하고, 이동 중에는 Telegram으로 빠르게 소통한다.

## 4. 지원 플랫폼

| 플랫폼 | 설명 | 기술 |

- 목표 추적
- 사용량 모니터링
- 설정 (프로바이더, 페르소나, 모델 관리)

## 5. AI 기능

### 멀티 LLM 지원

| 프로바이더 | 모델 | 용도 |
|-----------|------|------|
| **Google Gemini** | Gemini 2.0 Flash 등 | 기본 추론 |
| **OpenAI** | GPT-4o 등 | 고품질 생성 |
| **Anthropic Claude** | Claude 3.5 등 | 분석/추론 |
| **Ollama** | 로컬 모델 | 프라이버시 우선 |

### 46개 내장 스킬

- **대화**: 일반 대화, 컨텍스트 유지, 페르소나 전환
- **금융**: 거래 기록, 예산 관리, 리포트 생성
- **일기**: 일기 작성, 감정 분석, 요약
- **목표**: 목표 설정, 진행 추적, 리마인더
- **이미지**: 이미지 생성, 분석, 편집
- **음성**: 텍스트-음성 변환, 음성 인식
- **검색**: 웹 검색, 정보 조회
- **메모리**: 기억 저장, 회상, 컴팩션
- **패턴**: 행동 패턴 분석, 인사이트 도출
- **리포트**: 주간/월간 리포트 자동 생성

### 4계층 메모리 시스템 (RAG)

1. **즉시 메모리**: 현재 대화 컨텍스트
2. **단기 메모리**: 최근 대화 히스토리
3. **장기 메모리**: 벡터 임베딩 기반 지식 저장
4. **요약 메모리**: 대화 컴팩션 및 핵심 정보 추출
