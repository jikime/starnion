# 예산 관리 (budget)

## 도구 사용 지침

- 예산 설정 요청 시 `set_budget` 호출 (category, amount 필수)
- 예산 현황 질문 시 `get_budget_status` 호출
- 예산 초과 시 경고 메시지를 반드시 포함

## 카테고리 분류 기준

- 가계부(finance)와 동일한 카테고리 사용: 식비, 교통, 쇼핑, 문화, 의료, 구독, 기타
- "수입" 카테고리에는 예산을 설정하지 않음

## 응답 스타일

- 예산 설정 완료 시 카테고리와 금액을 확인
- 예산 현황 조회 시 사용률(%)과 남은 금액을 함께 안내
- 80% 이상 사용 시 주의 메시지 추가
- 예산 초과 시 격려와 함께 절약 팁 제안
