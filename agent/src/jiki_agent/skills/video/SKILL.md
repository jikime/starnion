# 비디오 (video)

## 도구 목록

| 도구 | 설명 |
|------|------|
| `analyze_video` | 비디오를 분석하여 내용 요약, 장면 설명 |
| `generate_video` | AI 이미지 슬라이드쇼 영상 생성 (Imagen 3 + ffmpeg) |

## analyze_video 사용 지침

- 비디오가 첨부된 메시지에 호출
- 비디오가 첨부되지 않은 상태에서는 호출하지 않음
- `file_url`: 비디오 파일의 URL
- `user_query`: 비디오에 대한 질문이나 요청

## generate_video 사용 지침

- 사용자가 영상 생성을 요청하면 호출
- `prompt`: 생성할 영상에 대한 상세한 설명
- `frames`: 프레임 수 (1-10, 기본값 5)
- Imagen 3로 이미지 프레임 생성 후 ffmpeg로 슬라이드쇼 MP4 생성
- ffmpeg 실패 시 첫 번째 프레임을 이미지로 전달

## 주의사항

- 이 스킬은 옵트인 방식이며, 사용자가 활성화해야 사용 가능
- 대용량 비디오는 처리 시간이 길어질 수 있음
- 영상 생성은 프레임 수에 비례하여 시간 소요

## 응답 스타일

- analyze_video: 분석 결과를 시간순으로 정리하여 전달
- generate_video: "영상을 생성했어요" 간단히 안내 (파일이 자동 전달됨)
- 불확실한 부분은 명시
