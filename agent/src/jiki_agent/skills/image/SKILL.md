# 이미지 (image)

## 도구 목록

| 도구 | 설명 |
|------|------|
| `analyze_image` | 이미지를 분석하여 내용 설명, 텍스트 추출, 객체 인식 |
| `generate_image` | 요청한 설명으로 AI 이미지 생성 (Imagen 3) |

## analyze_image 사용 지침

- 이미지가 첨부된 메시지에 호출
- 이미지가 첨부되지 않은 상태에서는 호출하지 않음
- 영수증 이미지의 경우 추출된 금액을 가계부 스킬과 연계하여 자동 기록 가능

### 이미지 유형 분류 기준

- 영수증: 금액, 날짜, 항목 추출 → 가계부 연계
- 일반 사진: 내용 설명, 객체 인식
- 텍스트 이미지: OCR, 번역
- 스크린샷: 화면 내용 분석

## generate_image 사용 지침

- 사용자가 이미지 생성을 요청하면 호출
- `prompt`: 생성할 이미지에 대한 상세한 설명 (영어로 번역하여 전달하면 품질 향상)
- 생성된 이미지는 자동으로 사용자에게 전달됨

## 응답 스타일

- analyze_image: 분석 결과를 알기 쉽게 정리, 영수증의 경우 "가계부에 기록할까요?" 확인
- generate_image: "이미지를 생성했어요" 간단히 안내 (파일이 자동 전달됨)
- 분석이 불확실한 부분은 명시
