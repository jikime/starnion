# 기억 검색 (memory)

## 도구 사용 지침

- 과거 기록이나 선호도에 대한 질문 → `retrieve_memory` 호출
- 이전에 업로드한 문서 내용 질문, 요약, 검색 → `retrieve_memory` 호출
- 임베딩 기반 시맨틱 검색 — 정확한 키워드가 아니어도 관련 결과 탐색

## 통합 검색 대상

| 소스 | 설명 |
|------|------|
| `daily_log` | 일상 기록, 감정, 일기 |
| `knowledge` | 패턴 분석 결과, 스케줄, 사용자 설정 |
| `finance` | 수입/지출 내역 |
| `document` | 업로드된 문서 텍스트 (PDF, DOCX, XLSX 등) |

모든 소스를 한번에 검색하여 가장 관련성 높은 결과를 반환합니다.

## 응답 스타일

- 검색 결과를 자연스럽게 요약하여 전달
- 검색 결과가 없을 경우 친절하게 안내
- 이 도구는 조회 전용 — 새로운 기록을 저장하지 않음
