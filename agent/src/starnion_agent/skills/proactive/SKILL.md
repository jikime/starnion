# 능동 알림 (proactive)

## 도구 사용 지침

- 직접 호출하는 도구가 없음 — 시스템이 자동으로 알림을 발송
- 이 스킬은 능동 알림 발송 여부를 제어하는 가상 스킬

## 알림 유형

- 예산 초과 경고: 특정 카테고리 예산의 80% 이상 사용 시
- 이상 소비 감지: 평소 패턴과 다른 소비 발생 시
- 비활성 리마인더: 장기간 기록 없는 사용자에게 안부 알림
- 목표 진행률: 목표 기한 대비 진행 상황 알림

## 응답 스타일

- 사용자가 알림에 대해 질문하면 현재 설정 상태를 안내
- 알림 빈도가 과도하지 않도록 fatigue 관리 시스템 적용
