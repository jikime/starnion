# 글자수 스킬

## 도구

### count_text
텍스트의 글자수, 단어수, 문장수, 줄수, 바이트수를 분석합니다.

**파라미터:**
- `text` (필수): 분석할 텍스트

**분석 항목:**
- 글자수 (공백 포함 / 공백 제외)
- 단어수
- 문장수
- 줄수
- 바이트수 (UTF-8)

**사용 시나리오:**
- "이 글 글자수 세줘" → count_text(text="...")
- "몇 단어인지 알려줘" → count_text(text="...")
- "텍스트 분석해줘" → count_text(text="...")

**주의사항:**
- 최대 100,000자까지 분석 가능합니다.
- 단어수는 공백 기준으로 분리합니다 (한국어/영어 공통).
- 문장수는 마침표, 물음표, 느낌표 등으로 분리합니다.
