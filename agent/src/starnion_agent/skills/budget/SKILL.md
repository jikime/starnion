# 예산 관리 (budget)

## 트리거 패턴 — set_budget을 호출해야 하는 경우

다음 중 하나라도 해당하면 반드시 `set_budget`을 즉시 호출하세요:

- 예산 설정/변경: "식비 예산 30만원", "교통비 5만원으로 설정해줘"
- 전체 지출 한도: "전체 비용을 X원으로 제한", "이번 달 총 지출 X만원으로 해줘", "월 지출 한도 X원"
- 예산 변경/수정: "예산 변경해줘", "예산 수정해줘", "X 예산을 Y원으로 바꿔줘", "제한 변경해줘"
- 특정 월 언급: "3월 전체 비용 50만원으로", "이번 달 식비 30만원으로"

**중요**: "전체", "총", "모든 지출"은 category="전체"로, 금액은 amount에 숫자(원 단위)로 전달하세요.
별도 확인 없이 바로 `set_budget`을 호출하고 결과를 알려주세요.

## 도구 사용 지침

- 예산 설정/변경 요청 시 → `set_budget` 즉시 호출 (category, amount 필수)
  - "전체", "총 지출", "전체 비용" → category="전체"
  - "식비" → category="식비", "교통" → category="교통" 등
  - amount는 항상 원 단위 정수 (예: "50만원" → 500000)
- 예산 현황 질문 시 → `get_budget_status` 호출
- 예산 초과 시 경고 메시지를 반드시 포함

## 카테고리 분류 기준

- 가계부(finance)와 동일한 카테고리 사용: 식비, 교통, 쇼핑, 문화, 의료, 구독, 기타
- "전체" 카테고리는 월 총 지출 한도를 의미
- "수입" 카테고리에는 예산을 설정하지 않음

## 응답 스타일

- 예산 설정/변경 완료 시 카테고리와 금액을 확인
- 예산 현황 조회 시 사용률(%)과 남은 금액을 함께 안내
- 80% 이상 사용 시 주의 메시지 추가
- 예산 초과 시 격려와 함께 절약 팁 제안
