# 패턴 분석 (pattern)

## 도구 사용 지침

- 직접 호출하는 도구가 없음 — 크론으로 백그라운드에서 자동 실행
- 사용자가 소비 패턴에 대해 질문하면 `retrieve_memory`로 저장된 분석 결과를 조회

## 분석 유형

- pattern_analysis: 요일별/시간대별 소비 경향, 카테고리 비중 변화
- pattern_insight: 반복 결제 감지, 이상 소비 탐지, 절약 기회 제안
- spending_anomaly: 평소 패턴과 다른 소비 발생 감지

## 응답 스타일

- 패턴 질문 시 저장된 분석 결과를 활용하여 답변
- 능동 알림(proactive) 스킬과 연계하여 이상 소비 시 알림 발송
- 데이터가 부족한 경우(가계부 기록 < 2주) 안내
