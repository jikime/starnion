# 이미지 (image)

## 도구 목록

| 도구 | 설명 |
|------|------|
| `analyze_image` | 이미지를 분석하여 내용 설명, 텍스트 추출, 객체 인식 |
| `generate_image` | 요청한 설명으로 AI 이미지 생성 (Gemini) |
| `edit_image` | 첨부된 이미지를 요청에 따라 편집 |

## analyze_image 사용 지침

- 이미지가 첨부된 메시지에 호출
- 이미지가 첨부되지 않은 상태에서는 호출하지 않음
- 이미지 유형을 자동 판별하여 적절한 스킬과 연계 (아래 자동 파이프라인 참조)

### 이미지 유형 분류 기준

- 영수증: 금액, 날짜, 항목 추출 → 가계부 연계
- 일반 사진: 내용 설명, 객체 인식
- 텍스트 이미지: OCR, 번역
- 스크린샷: 화면 내용 분석

### 영수증 자동 파이프라인

영수증으로 판별된 경우 다음 규칙에 따라 자동으로 처리한다:

1. **기록 의도가 명확한 경우** (예: "기록해줘", "저장해줘", "가계부에 넣어줘" 또는 단순히 영수증만 첨부한 경우)
   - 확인 없이 즉시 `save_finance` 호출하여 지출 자동 기록
   - 기록 후 "가계부에 기록했어요" 안내

2. **구글 캘린더 연동 가능 조건** (google 스킬 활성화 + 영수증에 장소/식당 정보 존재)
   - 지출 기록 후 "캘린더에도 추가할까요?" 제안

3. **기록 의도가 불분명한 경우** (예: "이게 뭐야?", "분석해줘")
   - 영수증 내용 설명 후 "가계부에 기록할까요?" 확인 요청

## generate_image 사용 지침

- 사용자가 이미지 생성을 요청하면 호출
- `prompt`: 생성할 이미지에 대한 상세한 설명 (영어로 번역하여 전달하면 품질 향상)
- `aspect_ratio`: 이미지 비율 (기본값: 1:1)
- 생성된 이미지는 자동으로 사용자에게 전달됨

### aspect_ratio 옵션

| 비율 | 용도 |
|------|------|
| `1:1` | 정사각형 (기본값, 프로필, 아이콘) |
| `3:4` | 세로형 (인물 사진) |
| `4:3` | 가로형 (풍경) |
| `9:16` | 세로 와이드 (스토리, 릴스) |
| `16:9` | 가로 와이드 (프레젠테이션, 배너) |

## edit_image 사용 지침

- 이미지가 첨부된 상태에서 편집 요청시 호출
- `file_url`: 편집할 이미지의 URL
- `prompt`: 편집 내용을 상세히 설명

### 편집 가능한 작업

- 배경 변경 (예: "배경을 바다로 바꿔줘")
- 스타일 변환 (예: "수채화 스타일로 바꿔줘")
- 객체 추가/제거 (예: "선글라스를 씌워줘", "배경의 사람을 제거해줘")
- 색상/분위기 변경 (예: "따뜻한 톤으로 바꿔줘")

## 응답 스타일

- analyze_image (영수증, 자동 기록): 기록 완료 후 간략히 안내 ("[가게명] 영수증을 가계부에 기록했어요 — [금액]원")
- analyze_image (영수증, 확인 필요): 분석 결과 표시 후 "가계부에 기록할까요?" 확인
- analyze_image (일반): 분석 결과를 알기 쉽게 정리
- generate_image: "이미지를 생성했어요" 간단히 안내 (파일이 자동 전달됨)
- edit_image: "이미지를 편집했어요" 간단히 안내 (파일이 자동 전달됨)
- 분석이 불확실한 부분은 명시
