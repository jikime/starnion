# AI Memory

## Overview

AI Memory is a feature that visualizes the patterns and insights Nion has automatically learned through conversations. Unlike memos that users manually enter, AI Memory is a long-term memory system generated by the AI itself through analysis of conversation context.

**Key features:**
- Insights automatically extracted from conversation patterns
- Emotional and behavioral pattern analysis over time
- Weekly conversation summaries and topic trend tracking
- Direct deletion of stored memory entries

> AI Memory is different from Memo. A memo is information you explicitly save yourself, while AI Memory is long-term learning data that Nion automatically generates by analyzing conversations.

---

## Three Tabs

### Conversation Insights

A list of insights extracted by AI through analysis of recent conversations.

**Displayed information:**
- **Date list**: Dates when analysis was completed (left sidebar or dropdown)
- **Insight cards**: Summary content of each analysis entry
- **Mood indicator**: Overall emotional state at that point in time
- **Topic tags**: Key topics that frequently appeared in conversations

**Insight types:**
| Type | Description |
|------|-------------|
| Conversation patterns | Speaking style, question patterns, etc. |
| Topics of interest | Interests that come up repeatedly |
| Emotional flow | Emotional changes during conversations |
| Behavioral indicators | Routines, habits, frequent actions |

### Behavior Patterns

Behavioral and preference patterns analyzed from conversation data.

**Displayed information:**
- **Pattern cards**: Title and description of each behavioral pattern
- **Category badge**: Pattern classification (lifestyle, health, finance, relationships, etc.)
- **Confidence indicator**: Confidence level of the AI analysis

**Pattern categories:**
| Category | Examples |
|----------|----------|
| Lifestyle habits | Sleep patterns, meal times, etc. |
| Emotional patterns | Stress triggers, positive factors |
| Financial behavior | Spending tendencies, saving patterns |
| Social relationships | Conversation frequency, relationship types |

### Weekly Memory

Conversation summaries and key event records automatically generated each week.

**Displayed information:**
- **Weekly summary**: Main conversation content and activities for the week
- **Key events**: Important moments that were specially recorded
- **Delete function**: Select and delete specific weekly memory entries

## When AI Memory Is Generated

AI Memory data is generated automatically and analysis only runs when certain conditions are met.

### Conversation Insights Conditions

- Automatic analysis runs when there has been **no conversation for 30 minutes to 2 hours** in Telegram or chat
- **Diary skill** must be activated
- Analysis cycle: Conditions checked approximately every 10 minutes

### Behavior Patterns Conditions

- Analyzes data up to the previous day at **6:00 AM every day**
- Requires **7 or more days** of conversation data for meaningful pattern generation
- If financial data is available, spending patterns are also analyzed

### Weekly Memory Conditions

- Automatic cleanup of the previous week's data at **5:00 AM every Monday**
- Requires **30 or more days** of conversation history for weekly compaction to run

> **If you don't see any data:** The above conditions may not have been met yet. As you naturally have conversations in Telegram or chat, the AI will gradually start learning patterns.

## Deleting Data

Stored AI Memory entries can be deleted directly.

### How to Delete

**Deleting Weekly Memory:**
1. Click the **Weekly Memory** tab.
2. Click the **Delete** button on the week entry you want to remove.
3. Click **Delete** in the confirmation popup.

**Deleting Conversation Insights / Behavior Patterns:**
1. Click the **Delete** icon on the item you want to remove in the respective tab.
2. The item is deleted immediately.

> Deleted memories cannot be recovered. Please delete carefully.

## Tips & FAQ

**Q. What is the difference between AI Memory and Memo?**

| Item | AI Memory | Memo |
|------|-----------|------|
| How it's created | AI generates automatically | User enters manually |
| Data source | Conversation analysis results | User input |
| Purpose | Long-term pattern learning | Information storage and retrieval |
| Editing | Not possible (deletion only) | Editable on the web |

**Q. Why is there no data in AI Memory?**
AI Memory is generated after enough conversation data has accumulated. If you regularly have conversations in Telegram or chat, data should start appearing within 2–7 days.

**Q. Will Conversation Insights not be generated without the Diary skill?**
Correct. Conversation Insights are only generated for users with the Diary skill activated. Enable the Diary skill in Settings > Skills.

**Q. If I delete a memory, will the AI re-learn it?**
Deleted memories are not recoverable, and patterns based on that data will no longer be displayed. However, as new conversations accumulate, the AI will learn again and generate new memories.

**Q. Is AI Memory data safe?**
All AI Memory data is stored tied to your user account and is not shared with other users. You can delete it at any time.
