# Spending Analysis

## Overview

The Spending Analysis feature **visualizes your expense data by category and time period**, providing an analytics dashboard to understand your spending patterns. View trends through pie charts, line charts, and more, while the AI offers personalized saving insights.

> You need to know where your money goes before you can save. Try asking Nion "Analyze my spending pattern this month."

---

## Key Charts

### Spending by Category (Pie Chart)

Shows the proportion of total spending for each category.

| Item | Description |
|------|-------------|
| **Category colors** | Food (green), Transport (blue), Shopping (pink), Leisure (orange), etc. |
| **Percentage** | Each category's share (%) |
| **Amount** | Actual amount shown on hover |

### Monthly Spending Trend (Line Chart)

Displays total monthly spending for the last 6 months.

- X-axis: Month (e.g., Oct, Nov, Dec, Jan, Feb, Mar)
- Y-axis: Spending amount
- Month-over-month direction indicator

### Top 5 Spending Categories

The five categories with the highest spending this month, ranked.

| Rank | Information |
|------|-------------|
| 1st | Category + amount + month-over-month change |
| 2nd | Category + amount + month-over-month change |
| ... | ... |
| 5th | Category + amount + month-over-month change |

### Month-over-Month Change

Compares this month's spending with last month.

| Display | Meaning |
|---------|---------|
| **↑ 15%** (red) | 15% increase from last month |
| **↓ 8%** (green) | 8% decrease from last month |
| **— 0%** (gray) | No change |

## Period Filters

Use the top filter to change the analysis period.

| Period | Description |
|--------|-------------|
| **This week** | Last 7 days |
| **This month** | From the 1st to today |
| **Last month** | Full previous month |
| **Last 3 months** | Cumulative last 3 months |
| **Last 6 months** | Cumulative last 6 months |
| **This year** | From January 1st to today |

## FAQ

**Q. Where does the analysis data come from?**

All income/expense data recorded in the finance section is used for analysis. The more records you have, the more accurate the analysis.

**Q. Is income included in the analysis?**

Spending Analysis covers expenses only. Income data can be found in the finance section's income/expense summary.

**Q. Can I download chart data?**

Direct chart image or data download is not currently supported. You can ask Nion for a text-based analysis summary and copy it.

**Q. Can I view analysis for a specific period?**

Yes, select a period from the top filter to see results for that timeframe. You can also specify periods in chat, e.g., "Analyze my March spending."

**Q. Can I add custom categories?**

Categories are automatically classified by the AI when recording expenses. Custom category creation is not currently supported, but you can correct misclassified entries.
