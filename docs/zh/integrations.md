# 集成

通过连接外部服务扩展 Starnion 的 AI 功能。

| 集成服务 | 说明 |
|---------|------|
| [Google Workspace](integrations/google.md) | Gmail、日历、Drive |
| [Notion](integrations/notion.md) | 搜索、创建和编辑页面 |
| [GitHub](integrations/github.md) | 仓库、Issue、PR 管理 |
| [Tavily](integrations/tavily.md) | 实时网络搜索 |
| [Naver Search](integrations/naver-search.md) | 韩语新闻和博客搜索 |
| [Gemini](integrations/gemini.md) | Google Gemini AI 模型集成 |
