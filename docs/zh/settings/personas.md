# 角色

## 概述

角色功能是 Starnion 的 AI 个性化配置系统，可以自定义 AI 的名称、性格和角色。创建多个角色并在不同情况下切换使用，就像与具有不同性格的 AI 对话一样。

**主要功能：**
- 角色自定义：设置名称、职责、性格描述和问候语
- 多角色支持：创建多个角色按需切换
- 内置角色：预设 5 种角色，开箱即用
- 对话中切换：随时在对话中切换角色

---

## 什么是角色？

角色是**定义 AI 性格和职责的配置文件**。

| 组成部分 | 说明 | 示例 |
|---------|------|------|
| **名称** | 角色的称呼 | "Nion"、"理财教练"、"英语老师" |
| **职责** | 专业领域 | "个人理财顾问" |
| **性格描述** | 对话风格和态度 | "亲切、注重节约..." |
| **问候语** | 对话开始时发送的消息 | "你好！我们来检查一下今天的财务状况吧。" |

---

## 内置角色

Starnion 预设了 **5 种内置角色**，开箱即用，无需额外配置。

| 角色 | 职责 | 特点 |
|------|------|------|
| **心灵伙伴** | 情感陪伴助手 | 共情能力强，用温暖的语言分享感受 |
| **基础助手** | 通用 AI 助手 | 友好亲切，协助处理各类日常事务 |
| **金融专家** | 个人理财顾问 | 专注储蓄与投资，分析财务状况并提供建议 |
| **数据分析师** | 数据洞察专家 | 清晰解读数字与统计，支持数据可视化 |
| **人生教练** | 目标达成教练 | 系统性地支持动力提升与习惯养成 |

## 创建角色

1. 前往 **设置 > 角色**。
2. 点击 **+ 新建角色** 按钮。
3. 填写名称、职责、性格描述和问候语。
4. 点击 **保存**。

### 角色示例

**理财教练**
```
名称：理财教练
职责：个人理财顾问
性格描述：专注于储蓄和投资，温和地指出不必要的支出。
问候语：你好！准备好检查今天的财务状况了吗？
```

**日记伙伴**
```
名称：小暖
职责：日记助手
性格描述：富有同理心，深切感受用户情感。使用温暖的语言。
问候语：今天过得怎么样？跟我说说吧。
```

**英语老师**
```
名称：Emma 老师
职责：英语口语教师
性格描述：只用英语交流，温和纠正语法错误。
问候语：Hi! Ready for some English practice today?
```

## 切换角色

### 从 Web UI 切换

1. 在 **设置 > 角色** 中，点击要激活的角色旁边的 **激活** 按钮。
2. 或者点击聊天界面顶部的角色名称。

### 在对话中切换

```
用户：切换到英语老师模式
AI：  Hi! I'm Teacher Emma. Let's practice English together!

用户：切回 Nion
AI：  我回来了！有什么可以帮你的？
```

## 常见问题

**Q：可以创建多少个角色？**
数量不限。但每次只能激活一个角色。

**Q：切换角色会清除之前的对话上下文吗？**
不会。对话历史和记忆保持不变。只有 AI 的语气和回复风格会改变。

**Q：可以编辑内置角色吗？**
可以修改任意内置角色的性格描述和问候语，但名称不可更改。

**Q：创建好角色的技巧是什么？**
性格描述越具体，AI 就能越一致地保持角色特征。
