# 技能列表

Starnion 目前内置 **38** 个技能。每个技能都是一个独立模块，可以根据需要单独启用或禁用。

---

## 财务

| 技能 ID | 技能名称 | 说明 | 示例指令 | 默认启用 |
|---------|---------|------|---------|---------|
| `finance` | 消费追踪器 | 自动记录收入和支出，并检索月度总计。 | `午餐12,000韩元`、`我这个月花了多少？` | ✓ |
| `budget` | 预算管理器 | 按类别设置月度预算并查看使用情况。 | `设置餐饮预算300,000韩元`、`还剩多少预算？` | ✓ |
| `pattern` | 消费模式分析 | 在后台自动分析按星期几的消费倾向、定期费用和异常消费。 | `我的消费模式是什么？`、`有异常支出吗？` | ✓ |
| `proactive` | 主动通知 | AI 主动发出预算超支、异常消费等提醒。 | （自动发送，无需直接调用） | ✓ |
| `currency` | 汇率 | 查询实时汇率并换算货币。 | `告诉我美元汇率`、`100美元是多少韩元？` | ✓ |

## 个人

| 技能 ID | 技能名称 | 说明 | 示例指令 | 默认启用 |
|---------|---------|------|---------|---------|
| `diary` | 日记 | 记录日常生活、情绪和想法。响应非财务消息。 | `我今天真的很难受`、`会议拖堂了` | ✓ |
| `memo` | 备忘录 | 保存、查看和删除带有标题和标签的备忘录。 | `备忘录：买牛奶`、`只显示工作备忘录` | ✓ |
| `goals` | 目标管理器 | 设置财务目标、待办事项、习惯和个人目标；跟踪进度。 | `本月餐饮支出控制在300,000以内`、`我完成目标了！` | ✓ |
| `dday` | 倒计时 | 跟踪重要日期的剩余天数；支持每年重复。 | `设置圣诞节倒计时`、`显示我的倒计时` | ✓ |
| `reminder` | 提醒 | 在特定时间安排一次性提醒。 | `明天上午9点提醒我开会` | ✓ |
| `schedule` | 日程 | 创建和管理定期或一次性提醒计划。 | `每周五晚上8点提醒我查看每周消费` | ✓ |
| `memory` | 记忆搜索 | 对历史对话、消费记录、日记条目和文档进行统一语义搜索。 | `我上周吃了什么？`、`记得我上次说的吗？` | ✓ |

## 媒体

| 技能 ID | 技能名称 | 说明 | 示例指令 | 默认启用 |
|---------|---------|------|---------|---------|
| `image` | 图像分析与生成 | 分析图像，用 AI 生成和编辑；支持自动从收据提取消费金额。 | `分析这张图像`、`生成一张猫的图片`、`把背景换成大海` | ✓ |
| `audio` | 语音转换 | 语音转文字（STT）和文字转语音（TTS）转换。 | `将这段语音转为文字`、`大声朗读这句话` | ✓ |
| `video` | 视频分析与生成 | 摘要视频内容并生成 AI 图像幻灯片视频。 | `总结这个视频`、`制作一个关于太空旅行的视频` | - |
| `documents` | 文档管理器 | 解析 PDF、Word、Excel 等文档，存储到向量数据库，并生成新文档。 | `摘要这份合同`、`将这份报告生成为 PDF` | ✓ |
| `qrcode` | 二维码生成器 | 从 URL 或文本生成二维码图像。 | `为这个 URL 生成二维码`、`创建 Wi-Fi 二维码` | ✓ |

## 搜索

| 技能 ID | 技能名称 | 说明 | 示例指令 | 默认启用 |
|---------|---------|------|---------|---------|
| `websearch` | 网络搜索 | 通过 Tavily API 搜索互联网上的最新信息并获取网页内容。 | `告诉我今天的新闻`、`摘要这篇文章：https://...` | ✓ |

| `summarize` | 摘要 | AI 驱动的 URL 网页或文本摘要；支持简洁/详细/要点样式。 | `摘要这个链接：https://...`、`用3个要点摘要` | ✓ |

## 实用工具

| 技能 ID | 技能名称 | 说明 | 示例指令 | 默认启用 |
|---------|---------|------|---------|---------|
| `calculator` | 计算器 | 计算数学表达式；支持 sin/cos/log 等数学函数。 | `计算 2+3*4`、`144的平方根`、`sin(π/2) 是多少？` | ✓ |

| `timezone` | 世界时间 | 查看世界各城市的当前时间并在时区之间转换。 | `伦敦现在几点？`、`首尔14:30在纽约是几点？` | ✓ |

| `color` | 颜色换算器 | 相互转换 HEX、RGB 和 HSL 颜色代码。 | `告诉我颜色 #FF5733`、`红色的 RGB 值是多少？` | ✓ |
| `encode` | 编码 | Base64、URL 和 HTML 的编码与解码。 | `将 Hello World 编码为 Base64`、`解码这个 URL` | ✓ |

| `random` | 随机 | 随机选择、抽号、掷硬币、掷骰子和随机字符串生成。 | `在炸酱面和辣海鲜汤之间选一个`、`从1-45中抽6个号`、`生成一个密码` | ✓ |
| `ip` | IP 查询 | 查询 IP 地址或域名的位置和 ISP 信息。 | `8.8.8.8在哪里？`、`告诉我我的公网 IP`、`google.com 的服务器在哪里？` | ✓ |

## 各技能所需权限

某些技能需要外部 API 密钥或 OAuth 认证。

| 技能 | 所需配置 | 配置位置 |
|------|---------|---------|
| `weather` | 天气 API 密钥 | 服务器 `.env` |
| `websearch` | Tavily API 密钥 | **设置 > 集成** |
| `naver_search` | Naver 客户端 ID/密钥 | **设置 > 集成** |
| `google` | Google OAuth2 认证 | **设置 > 集成** |
| `image` | Gemini API 密钥（用于图像生成） | **设置 > 模型** |
| `audio` | Gemini API 密钥（用于 TTS） | **设置 > 模型** |
| `currency` | 汇率 API（Frankfurter，免费） | 自动 |
| `browser` | 需要安装 Playwright | 服务器环境 |
| `notion` | Notion 集成令牌 | **设置 > 集成** |
| `github` | GitHub 个人访问令牌 | **设置 > 集成** |
| `coding_agent` | 需要安装 Claude Code CLI | 服务器环境 |

## 使用自然语言调用技能

您不需要直接提及技能名称。只需自然地说话，AI 就会自动选择合适的技能。

```
我：50,000韩元是多少美元？
机器人：按当前汇率（1,340韩元/美元），50,000韩元 ≈ 37.31美元。

我：伦敦现在几点？
机器人：伦敦（GMT）当前时间为上午6:42。

我：今天真的很糟糕，被经理骂了。
机器人：听起来很辛苦。我已将其记录到您的日记中。

我：明天下午3点提醒我看医生。
机器人：已为"看医生"设置明天下午3点的提醒。

我：从1到45中抽6个彩票号码。
机器人：🎰 结果：7、14、23、31、38、42

我：这张图里有什么？（附加照片）
机器人：看起来是一张咖啡店收据——美式咖啡4,500韩元。是否将其记录到您的消费追踪器中？

我：总结 https://news.ycombinator.com。
机器人：我已总结了该页面。当前热门帖子：...
```

## 组合多个技能

单条消息可以按顺序触发多个技能。

| 示例 | 激活的技能 |
|------|-----------|
| 发送收据照片 | `image` → 提取金额 → `finance` → 自动记录 |
| 语音消息"午餐10,000韩元" | `audio` → 转录 → `finance` → 记录到消费追踪器 |
| "有异常消费模式吗？" | `pattern` 分析结果 → `memory` 搜索 → 响应 |
| "将这份报告生成为 PDF" | LLM 生成内容 → `documents` → 交付 PDF 文件 |
