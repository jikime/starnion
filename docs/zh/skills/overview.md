# 技能系统

技能是 Starnion AI 智能体的功能模块。每个技能都是一个独立的工具，负责特定功能，可以根据需要单独开启或关闭。

---

## 什么是技能？

技能是让 AI 能够真正**执行**操作（而不仅仅是对话）的功能单元。

例如：
- `今天天气怎么样？` → **weather**（天气）技能调用天气 API
- `午餐花了12,000韩元` → **finance**（财务）技能将其记录到消费追踪器
- `收据照片` → **image**（图像）技能提取金额 → **finance**（财务）技能自动记录

即使用户没有明确提到技能名称，AI 也会读取上下文并自动选择合适的技能。

---

## 启用/禁用技能

您可以在**设置 > 技能**下单独切换每个技能。

- 禁用的技能即使 AI 收到相关请求也不会执行。
- 某些核心技能（系统技能）无法禁用。
- 技能设置按账户保存。

## 使用自然语言调用技能

您不需要明确指定技能名称。只需自然地说话，AI 就会自动选择合适的技能。

```
我：50,000韩元是多少美元？
机器人：按当前汇率（1,340韩元/美元），50,000韩元 ≈ 37.31美元。

我：伦敦现在几点？
机器人：伦敦（GMT）当前时间为上午6:42。

我：今天真的很糟糕，被经理骂了。
机器人：听起来很辛苦。我已将其记录到您的日记中。

我：明天下午3点提醒我看医生。
机器人：已为"看医生"设置明天下午3点的提醒。

我：这张图里有什么？（附加照片）
机器人：我分析了图像。看起来是一张咖啡店收据——美式咖啡4,500韩元。

我：总结 https://news.ycombinator.com。
机器人：我已总结了该页面。当前热门帖子：...
```

## 技能权限

某些技能需要外部 API 密钥或 OAuth 认证。

| 技能 | 所需配置 |
|------|---------|
| `weather` | 天气 API 密钥（环境变量） |
| `websearch` | Tavily API 密钥 |
| `naver_search` | Naver 客户端 ID/密钥 |
| `google` | Google OAuth2 认证完成 |
| `image` | OpenAI API 密钥（用于图像生成） |
| `currency` | 汇率 API 密钥 |

API 密钥配置由服务器管理员在 `.env` 文件中处理。

## 常见问题

**Q：禁用技能会删除数据吗？**
不会。禁用技能不会删除现有数据。重新启用后，所有之前的数据都可以访问。

**Q：什么是系统技能？**
`memory` 和 `proactive` 等智能体核心操作所需的技能。这些技能无法禁用。

**Q：多个技能可以同时使用吗？**
可以。单条消息可以按顺序触发多个技能。例如，发送收据图片会使 `image` 技能识别金额，然后 `finance` 技能自动记录到消费追踪器。
