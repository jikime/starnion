# 消费分析

## 概述

消费分析功能将记账数据**按类别、按时间段进行可视化**，帮助您了解消费模式。通过饼图、折线图等多种图表直观展示支出趋势，AI还会提供个性化的节省建议。

> 了解钱花在哪里才能更好地节省。试着对Nion说"分析一下我这个月的消费模式"。

---

## 主要图表

### 按类别支出比例（饼图）

展示各类别在总支出中所占的比例。

| 项目 | 说明 |
|------|------|
| **类别颜色** | 餐饮（绿色）、交通（蓝色）、购物（粉色）、娱乐（橙色）等 |
| **比例显示** | 各类别的百分比（%） |
| **金额显示** | 悬停时显示实际金额 |

### 月度支出趋势（折线图）

展示最近6个月的月度总支出趋势。

- X轴：月份（如：10月、11月、12月、1月、2月、3月）
- Y轴：支出金额
- 显示环比增减方向

### 支出类别 TOP 5

本月支出最多的5个类别排名。

| 排名 | 显示信息 |
|------|---------|
| 第1名 | 类别名 + 金额 + 环比变化率 |
| 第2名 | 类别名 + 金额 + 环比变化率 |
| ... | ... |
| 第5名 | 类别名 + 金额 + 环比变化率 |

### 环比变化率

对比本月与上月的支出，显示变化率。

| 显示 | 含义 |
|------|------|
| **↑ 15%**（红色） | 比上月增加15% |
| **↓ 8%**（绿色） | 比上月减少8% |
| **— 0%**（灰色） | 无变化 |

## 时间段筛选

使用顶部筛选器可以更改分析时间段。

| 时间段 | 说明 |
|--------|------|
| **本周** | 最近7天 |
| **本月** | 本月1日至今天 |
| **上月** | 上月全月 |
| **最近3个月** | 最近3个月累计 |
| **最近6个月** | 最近6个月累计 |
| **今年** | 1月1日至今天 |

## AI分析洞察

Nion分析您的消费数据，提供个性化的洞察建议。

### 洞察示例

```
本月消费分析结果：

1. 总支出：15,200元（环比 ↑12%）
2. 最大支出：餐饮 4,800元（31.6%）
3. 增长项：购物 +45%（推测受促销季影响）
4. 下降项：交通 -20%（推测受居家办公影响）
5. 节省建议：每周在家做饭3次可节省约800元/月
```

## 常见问题

**Q. 分析使用的数据来自哪里？**

记账中记录的所有收支数据都会用于分析。记录越多，分析越准确。

**Q. 收入也包含在分析中吗？**

消费分析仅分析支出数据。收入数据可以在记账的收支汇总中查看。

**Q. 可以下载图表数据吗？**

目前不支持直接下载图表图片或数据。您可以在聊天中让Nion提供文字版分析结果，然后复制使用。

**Q. 可以只查看特定时间段的分析吗？**

可以，在顶部筛选器中选择时间段即可显示对应期间的分析结果。在聊天中也可以指定，如"分析我3月份的消费"。

**Q. 可以自定义添加类别吗？**

类别由AI在记录支出时自动分类。目前不支持用户自定义添加类别，但可以修正分类错误的条目。
