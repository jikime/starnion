# 搜索

Starnion 提供两种搜索模式。**本地搜索**快速查找您自己的数据（日记条目、备忘录、账单记录等），**网络搜索**从互联网获取最新信息。两种模式也都可以通过 AI 聊天中的自然语言使用。

---

## 顶部搜索栏

屏幕顶部的搜索栏让您可以随着输入实时搜索数据。

### 使用方法

1. 点击屏幕顶部的搜索图标（🔍）或搜索输入框。
2. 输入搜索查询。
3. 结果实时出现。
4. 点击结果跳转到该项目。

### 搜索快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+K`（Windows/Linux） | 打开搜索栏 |
| `Cmd+K`（macOS） | 打开搜索栏 |
| `ESC` | 关闭搜索栏 |
| `↑` `↓` | 浏览结果 |
| `Enter` | 跳转到所选项目 |

---

## 本地搜索

在您自己的数据中搜索。无需任何外部网络连接，结果快速出现。

### 搜索内容

| 数据类型 | 可搜索内容 | 示例 |
|----------|----------|------|
| **日记** | 日期、内容、心情 | "上周的日记"、"悲伤的一天" |
| **备忘录** | 标题、内容、标签 | "Python备忘录"、"#工作标签" |
| **账本** | 项目名称、金额、日期、分类 | "三层肉"、"5万韩元以上的支出" |
| **文档** | 文件名、文档内容 | "合同"、"2024年报告" |
| **聊天历史** | 之前的聊天内容 | "上次问的食谱" |

### 搜索示例

```
查询："三层肉"
结果：
  📒 账本    2024-03-15  三层肉（和朋友）  32,000韩元
  📒 账本    2024-02-28  深夜三层肉馆      18,000韩元
  📔 日记     2024-03-15  "和朋友吃了三层肉..."

查询："Python"
结果：
  📝 备忘录   Python基础摘要  #学习 #开发
  📄 文档     Python_tutorial.pdf
  💬 聊天     "告诉我如何使用Python列表推导式..."
```

## 网络搜索

Starnion 通过 **Tavily API** 和 **Naver搜索API** 从互联网搜索最新信息。

### Tavily网络搜索

AI友好型搜索API，即使对于复杂问题也能提供高度相关的结果。

**如何启用：** 服务器管理员在 `.env` 中设置 `TAVILY_API_KEY`，会自动激活 `websearch` 技能。

**使用示例：**

```
您：  比较2024年最新的AI模型。
机器人：[正在搜索网络...]
      以下是互联网搜索结果的摘要。

      GPT-4o（OpenAI）：多模态支持、快速响应...
      Gemini 1.5 Pro（Google）：100万token上下文...
      Claude 3.5 Sonnet（Anthropic）：出色的编程性能...

      来源：techcrunch.com, arxiv.org ...
```

### Naver搜索

针对韩语内容优化的搜索。搜索博客、新闻、购物和Naver知识iN（问答）。

**如何启用：** 在 `.env` 中设置 `NAVER_CLIENT_ID` 和 `NAVER_CLIENT_SECRET`，然后在 设置 > 技能 中激活 `naver_search` 技能。

**支持的搜索类型：**

| 类型 | 描述 | 示例请求 |
|------|------|----------|
| 博客 | Naver博客搜索 | "找一篇怎么做好泡菜汤的博客文章" |
| 新闻 | 最新新闻 | "今天的国内经济新闻" |
| 购物 | 商品和价格比较 | "告诉我AirPods Pro的最低价格" |
| 知识iN | 问答搜索 | "在Naver知识iN找签租房合同的注意事项" |
| 本地 | 本地信息 | "找江南的餐厅" |
| 图书 | 书目搜索 | "告诉我关于《Clean Code》这本书" |

## 搜索历史

您可以查看最近的搜索和结果。

- 使用顶部搜索栏搜索后，结果会自动记录。
- 可以在 **设置 > 搜索历史** 中查看或清除完整历史。
- 搜索历史也可以在AI聊天中引用。

## 通过 AI 聊天搜索

除了顶部搜索栏，您还可以在聊天窗口中直接用自然语言提出搜索请求。AI 会根据需要组合本地搜索和网络搜索来回答您的问题。

### 搜索本地数据（memory技能）

用自然语言查询您自己的记录。

```
您：  上个月我吃了什么？
机器人：以下是您上个月（2024年2月）的餐饮支出：
      - 2月3日：午餐（便利店）  8,500韩元
      - 2月8日：晚餐（三层肉馆）22,000韩元
      - 2月14日：情人节晚餐（意餐）67,000韩元
      ...
      餐饮总支出：312,400韩元

您：  找我的Python相关备忘录。
机器人：找到3条Python相关备忘录：
      1. "Python基础摘要"（2024-01-15，#学习）
      2. "如何使用FastAPI"（2024-02-03，#开发）
      3. "Pandas DataFrame笔记"（2024-02-18，#学习）

您：  我之前提到的天花板维修估价是多少？
机器人：在2024年1月15日的对话中找到：
      记录了"天花板维修估价：2,500,000韩元"。
```

## 语义搜索

Starnion 的搜索超越了简单的关键词匹配，支持**基于含义的（语义）**搜索。

### 什么是语义搜索？

- 即使您输入的**确切词语不存在**，只要含义相似，结果也会出现。
- 文本被转换为向量（嵌入）并存储在PostgreSQL（pgvector）中。搜索时检索相似的向量。

### 语义搜索示例

```
查询："吃饭的记录"
实际匹配："午餐"、"外出就餐"、"三层肉馆"、"意大利餐厅"
→ 即使没有确切词语，也能找到与食物相关的记录

查询："花出去的钱"
实际匹配："支出"、"付款"、"12,000韩元"、"购买"
→ 即使没有确切词语，也能找到财务记录

查询："心情好的那天"
实际匹配："今天真的很兴奋"、"快乐的一天"、"状态极佳"
→ 理解情感表达并找到相关日记条目
```

### 应用了语义搜索的数据

| 数据 | 已应用嵌入 | 备注 |
|------|:---------:|------|
| 日常对话记录 | ✓ | 每次对话后自动保存 |
| 账本项目 | ✓ | 项目名称+分类 |
| 日记条目 | ✓ | 全文嵌入 |
| 备忘录 | ✓ | 标题+内容 |
| 文档段落 | ✓ | 每个文档块 |
| 知识库 | ✓ | 模式分析结果 |

## 常见问题

**Q. 本地搜索有多快？**
大多数搜索在100毫秒内返回结果。语义向量搜索根据数据量可能稍慢，但通常在1秒内完成。

**Q. 搜索结果不准确时怎么办？**
输入更具体的关键词，或在AI聊天中指定日期和条件——例如"2024年2月记录的午餐餐饮支出"——以获得更准确的结果。

**Q. 网络搜索结果会保存吗？**
网络搜索结果本身不会保存。但它们会作为聊天历史的一部分保留。然后您可以向AI询问"显示你之前找到的防范租房诈骗技巧"，它会从聊天历史中检索出来。

**Q. 可以搜索旧数据吗？**
可以。所有数据都永久存储，可以搜索。但较旧的数据可能会获得较低的相关性分数。

**Q. 如何搜索文件内部的内容？**
将PDF或Word文件上传到Starnion，`documents`技能会解析其内容并存储到向量数据库中。然后您可以使用顶部搜索栏或AI聊天通过文件内容进行搜索。
