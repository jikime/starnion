# 备忘录

## 概述

备忘录功能是一个简单的笔记系统，可以快速保存您需要记住的信息，并用标签进行整理。如果日记是记录情感和日常生活的个人日志，那么备忘录就是存储信息和事实的个人数据库。

**主要功能：**
- 基于标签的组织方式，便于快速搜索
- 最多存储100条备忘录
- AI 自动为您找到相关备忘录
- 简单的标题 + 内容结构，管理清晰

> 备忘录与记忆技能不同。备忘录是您明确保存和检索的结构化信息存储。记忆是 AI 的长期学习系统，会自动吸收对话上下文。

## 写备忘录

### 在网页端写备忘录

1. 点击侧边栏中的 **备忘录**。
2. 点击右上角的 **+ 新建备忘录** 按钮。
3. 输入以下内容：
   - **标题**（必填）：快速识别备忘录的名称
   - **内容**（可选）：备忘录的详细内容
   - **标签**（可选）：分类类别（默认：个人）
4. 点击 **保存**。

保存的备忘录按最近修改时间排序显示在列表中。

### 通过聊天写备忘录

在聊天框中让 AI 保存备忘录，会立即存储。

**示例对话：**

> 用户："我需要买牛奶和鸡蛋，保存备忘录"
>
> AI："备忘录已保存！记录为'购物清单'。"

> 用户："保存会议记录——预定下周二上线，需要市场团队审批"
>
> AI："会议记录备忘录已保存。标签设为'工作'。"

> 用户："新想法备忘录——如果能给应用添加深色模式就太好了"
>
> AI："想法备忘录已保存！用'想法'标签记录了。"

**保存触发短语：**
- "记录这个"、"记下这个"、"保存这个"
- "留个备忘——"、"写下来"
- "保存会议记录"、"记录X"

## 标签系统

标签是对备忘录进行分类并便于日后快速查找的关键功能。

### 默认标签

保存备忘录时如果未指定标签，会自动应用 **个人** 标签。AI 会分析备忘录内容并自动建议适当的标签：

| 内容示例 | 自动标签 |
|----------|----------|
| 会议内容、项目相关 | 工作 |
| 购物、食物备忘录 | 个人 |
| 新想法、规划 | 想法 |
| 学习笔记、课堂记录 | 学习 |
| 旅行计划、想去的地方 | 旅行 |

### 如何添加标签

**在网页端：** 直接在标签字段中输入并保存
**通过聊天：** "用工作标签记录这个" / "用想法标签保存这个"

### 按标签搜索

**在网页端：** 点击备忘录列表上方的标签筛选按钮 → 选择所需标签
**通过聊天：**

> 用户："只显示工作备忘录"
>
> AI："工作标签备忘录：\n1. 会议记录（3/5）\n2. 项目计划（3/3）\n3. 截止日期安排（2/28）"

## 搜索备忘录

### 关键词搜索（网页端）

在备忘录页面的搜索栏中通过标题或内容中的关键词进行搜索。搜索不区分大小写，支持部分匹配。

**示例：** 搜索"市场"→ 显示标题或内容中含有"市场"的所有备忘录

### AI 辅助搜索（聊天）

向 AI 描述您要找的内容，它会找到相关备忘录。您不需要记住确切的关键词。

> 用户："找一下我上次保存的会议相关备忘录"
>
> AI："找到了会议相关备忘录：\n1. 3/5 会议记录——下周预定上线\n2. 2/28 周会——市场团队反馈"

> 用户："显示所有项目相关备忘录"
>
> AI："找到3条项目相关备忘录：\n..."

> 用户："有食物相关的备忘录吗？"
>
> AI："有食物相关备忘录：\n- 购物清单：牛奶、鸡蛋\n- 餐厅列表：意面馆（弘大）"

## AI 使用示例

以下是与 AI 一起使用备忘录功能的各种方式。

### 快速保存备忘录

```
用户："保存一个链接以后看——https://example.com/article"
AI: "链接备忘录已保存。标题设为'参考链接'。"

用户："把金哲洙的电话010-1234-5678保存为备忘录"
AI: "联系人备忘录已保存。用'个人'标签记录了。"
```

### 保存结构化备忘录

```
用户："保存这些会议记录：
  参会者：金组长、李副组长、朴组员
  议程：Q2市场策略
  决定：增加SNS广告预算200万韩元"
AI: "会议记录已保存！用'工作'标签记录了。"
```

### 检索备忘录

```
用户："显示我的备忘录"
AI: "以下是您最近的5条备忘录：
  1. 会议记录（工作）— 3/8
  2. 购物清单（个人）— 3/7
  3. 旅行清单（旅行）— 3/5
  4. 想法笔记（想法）— 3/3
  5. 学习总结（学习）— 3/1"

用户："详细显示旅行备忘录"
AI: "[旅行清单备忘录]
  护照、充电器、旅行保险证明、当地SIM卡..."
```

### 删除备忘录

```
用户："删除购物清单备忘录"
AI: "购物清单备忘录已删除。"
```

## 使用技巧与常见问题

**Q. 我应该如何在备忘录和日记之间选择？**

| 情况 | 推荐功能 |
|------|----------|
| "今天的感受和发生的事" | 日记 |
| "保存以后会参考的信息" | 备忘录 |
| "会议记录、待办事项" | 备忘录 |
| "回顾情绪化的一天" | 日记 |

**Q. 可以存储多少条备忘录？**
最多100条。达到100条后，需要删除旧备忘录才能保存新的。

**Q. 可以给一条备忘录添加多个标签吗？**
目前每条备忘录只能有一个标签。如果备忘录涉及多个主题，在内容中包含关键词可以更容易在搜索时找到。

**Q. 在聊天中保存的备忘录在网页端也能看到吗？**
可以，聊天和网页端共享同一数据。在聊天中保存的备忘录也会出现在网页端的备忘录列表中，您可以在那里查看和编辑它们。

**Q. 可以编辑备忘录吗？**
在网页端，点击备忘录会进入编辑页面。聊天目前不支持直接编辑，请删除现有备忘录后重新保存一条新的。

**Q. 备忘录内容也会保存到 AI 记忆中吗？**
备忘录本身不会直接保存到 AI 长期记忆中。但是，如果您检索备忘录或在对话中提到其内容，该对话上下文可能会影响 AI 记忆。要明确让 AI 记住某件事，请说"记住这个"。
