# 频道

## 概述

频道管理页面用于配置和管理与Starnion AI助手Nion进行对话的**通信频道**。目前支持**Telegram Bot**和**Web Chat**两种频道，您可以一目了然地查看各频道的连接状态。

> 频道是与Nion沟通的桥梁。无论在Telegram还是网页上，都能享受一致的AI体验。

---

## 支持的频道

| 频道 | 说明 | 状态显示 |
|------|------|---------|
| **Telegram Bot** | 通过Telegram聊天 | 已连接 / 已断开 |
| **Web Chat** | 网页仪表盘内聊天 | 始终可用 |

---

## 连接Telegram机器人

### 第1步：在BotFather中创建机器人

1. 在Telegram中搜索[@BotFather](https://t.me/BotFather)。
2. 输入`/newbot`命令。
3. 设置机器人名称和用户名。
4. 复制获得的**API Token**。

### 第2步：在Starnion中注册Token

1. 从侧边栏进入**设置**页面。
2. 在**Telegram**部分输入机器人Token。
3. 点击**连接**按钮。
4. 连接成功后，频道页面将显示"已连接"状态。

### 第3步：开始对话

在Telegram中搜索您创建的机器人，发送`/start`命令即可开始与Nion对话。

---

## 查看频道状态

频道页面实时显示各频道的当前状态。

| 状态 | 含义 |
|------|------|
| **已连接**（绿色） | 正常运行中 |
| **已断开**（红色） | Token过期或配置错误 |

如果显示断开，请在设置页面重新输入Token，或重新创建机器人。

## 常见问题

**Q. Telegram机器人没有响应。**

请确认机器人Token是否正确，并在设置页面检查连接状态。如果Token已过期，可以通过BotFather的`/token`命令重新获取。

**Q. 多个Telegram账户可以使用同一个机器人吗？**

每个Starnion账户只能连接一个Telegram机器人。其他Telegram账户向同一机器人发送消息时，只有首次关联的账户会正常工作。

**Q. Web Chat和Telegram的对话记录是共享的吗？**

是的，所有频道的对话都保存在同一历史记录中。您可以在Telegram上开始对话，然后在网页上继续。
