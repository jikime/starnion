# 数据花园

## 概述

数据花园是一个**将 Starnion 所有数据可视化为生态花园**的仪表板。您的财务、目标、日记情感、文件和技能分别表示为不断成长的花园元素。

> 储蓄率越高，树越健康。超支时，云会开始下雨。

## 天空背景 -- 心情调色板

天空颜色根据过去7天日记中记录的情感而变化。

| 心情 | 天空颜色 | 氛围 |
|------|---------|------|
| Joy（非常好） | 金色渐变 | 温暖的黄金时刻光芒 |
| Peace（好） | 蓝色渐变 | 澄澈平静的天空 |
| Calm（一般） | 紫色渐变 | 神秘的宇宙薄雾 |
| Sad（不好） | 深紫色 | 黑暗的夜空 |
| Down（很不好） | 深空黑色 | 漆黑的虚空 |

## 中央：资产树

花园的核心。树的健康状况反映您的储蓄率。

### 健康阶段

| 储蓄率 | 树的状态 | 树枝颜色 | 果实 |
|--------|---------|---------|------|
| 30% 及以上 | 健康 | 全彩色 | 最多5个星形果实 |
| 15-29% | 开始枯萎 | 饱和度降低 + 下垂 | 果实较少 |
| 低于 15% | 危急 | 灰褐色 + 严重下垂 | 无果实 |

### 人设光环

树的根部会出现与当前 AI 人设颜色匹配的光环。

| 人设 | 光环颜色 |
|------|---------|
| assistant | 银白色 |
| finance | 绿色 |
| buddy | 粉色 |
| coach | 橙色 |
| analyst | 蓝色 |
| counselor | 紫色 |

### 预算弹窗（点击树）

点击资产树会打开详细的预算弹窗。

| 显示信息 | 说明 |
|---------|------|
| **剩余预算** | 本月剩余预算（绿色/橙色/红色编码） |
| **预算星座** | 以圆形排列的类别星形 SVG |
| **总预算 / 支出** | 数值和进度条 |
| **支出率 / 储蓄率** | 百分比显示 |
| **状态消息** | 基于预算健康状况的上下文消息 |

## 支出云朵（右上）

当任一预算类别超支时，云朵会下雨。

- **无雨**：所有类别均在预算内
- **下雨**：对超支类别显示动画雨滴从云朵落下
- **剪刀动画**：超支时树上会出现剪刀

点击云朵可以看到超支类别及其统计数据的弹窗。

---

## 目标花园（左下）

进行中的目标以郁金香形式可视化。最多显示4朵花。

### 生长阶段

| 阶段 | 进度 | 外观 |
|------|------|------|
| 种子 | 0-10% | 单个闪烁的点 |
| 芽 | 10-30% | 茎 + 花苞 |
| 成长中 | 30-55% | 茎 + 2片叶子 |
| 开花中 | 55-80% | 3片花瓣打开 |
| 盛开 | 80-100% | 完全绽放 + 摇曳动画 |

点击花园可以在弹窗中查看所有目标及其完成率。

## 情感种子（左下）

过去7天日记的情感以小种子形式显示。

- 最多7颗种子
- 每颗种子按心情值着色
- 每颗种子有绽放动画

## 特殊效果

### 收入星尘

当记录收入时，金色星星从天空落下。

- 星星数量：`收入 / 200,000`，最多12颗
- 星尘落下动画
- 天空中也会出现与收入金额成比例的浮动收入星

### 流星

当 D-Day 在7天以内时，流星会斜穿屏幕。

- 最多3颗流星
- 35度角，4.5秒周期
- 数量随临近 D-Day 的数量而增加

## 数据流

数据花园加载时，会并行发起12个 API 调用。

| API 端点 | 用途 |
|---------|------|
| `/api/budget` | 预算类别和支出 |
| `/api/goals` | 目标进度 |
| `/api/diary?limit=7` | 过去7天的情感 |
| `/api/finance/summary` | 收入和储蓄率 |
| `/api/dday` | D-Day 列表 |
| `/api/images?limit=50` | 图片文件数 |
| `/api/audios?limit=50` | 音频文件数 |
| `/api/documents` | 文档文件数 |
| `/api/skills` | 活跃技能数 |
| `/api/usage?days=30` | 月度令牌使用量 |
| `/api/profile/persona` | 当前人设 |
| `/api/settings/providers` | 已连接的服务数 |

## 常见问题

**Q. 树变灰了。**

当储蓄率低于15%时，树会枯萎。减少支出或增加收入可以恢复其健康。

**Q. 云朵一直在下雨。**

当一个或多个预算类别超支时会发生这种情况。点击云朵查看哪些类别超支，并相应调整预算。

**Q. 只显示了4朵花。**

目标花园最多显示4个进行中的目标。点击花园或导航到目标菜单查看所有目标。

**Q. 能量充电器变红了。**

当月度令牌使用量超过1000万令牌限额的80%时，充电器变红。减少聊天频率有助于恢复充电水平。

**Q. 天空非常暗。**

如果过去7天的日记情感偏向 Sad 或 Down，天空会变暗。试着在日记中写下你的感受，或与 Nion 聊天。

**Q. 流星什么时候出现？**

当 D-Day 距今7天或更少时会出现流星。在 D-Day 菜单中注册即将到来的事件即可触发。
