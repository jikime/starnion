# 预算管理

## 概述

预算管理功能可以**按月、按类别设置预算并实时监控支出**。通过进度条直观展示实际支出与预算的比例，一目了然掌握消费状况。当支出超过预算的80%时，系统会自动发出警告，防止超支。

> 设置预算后会与记账数据自动关联。试着问Nion"这个月伙食费还剩多少？"

---

## 设置预算

### 在网页上设置

1. 点击侧边栏的**预算管理**菜单。
2. 点击**+ 添加预算**按钮。
3. 输入以下信息：
   - **类别**（必填）：餐饮、交通、购物、娱乐、医疗、教育等
   - **月预算金额**（必填）：该类别的月度预算
4. 点击**保存**。

总预算会自动计算为各类别预算的总和。

### 通过聊天设置

用自然语言告诉AI即可设置预算。

**对话示例：**

> 用户："把这个月的伙食费预算设为3000元"
>
> AI："已将伙食费预算设为3,000元！目前伙食费支出为1,280元，已使用预算的43%。"

## 查看预算状况

### 仪表盘页面

预算管理页面显示各类别的以下信息：

| 显示项目 | 说明 |
|---------|------|
| **类别名称** | 已设置预算的类别 |
| **进度条** | 支出与预算比例的可视化 |
| **支出 / 预算** | 当前支出 / 设定预算（如：1,280 / 3,000元） |
| **使用率（%）** | 支出占预算的百分比 |
| **状态图标** | 安全（绿色）、注意（橙色）、警告（红色） |

### 进度条颜色

| 使用率 | 颜色 | 含义 |
|--------|------|------|
| 0–60% | 绿色 | 安全区间 |
| 60–80% | 橙色 | 注意区间 |
| 80–100% | 红色 | 警告区间 |
| 100%+ | 深红色 | 超支 |

---

## 超预算警告

当支出超过预算的80%时，系统会自动发出警告。

### 警告方式

- **网页仪表盘**：对应类别的进度条变为红色
- **聊天通知**：记账时AI会提醒预算即将超支
- **周报**：周报中包含预算状况

## 常见问题

**Q. 预算每月会自动重置吗？**

是的，预算在每月1日自动重置。只有支出金额会重置，设定的预算金额保持不变。

**Q. 可以不按类别、只设置总预算吗？**

目前仅支持按类别设置预算。您可以使用"其他"类别来设置总预算。

**Q. 超出预算后会阻止记录支出吗？**

不会，即使超出预算也能正常记录支出。系统只会显示超支警告，不会限制记录。

**Q. 会与记账数据自动同步吗？**

是的，记账中记录的支出会自动反映到对应的预算类别中，无需额外设置。
