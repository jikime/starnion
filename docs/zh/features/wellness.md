# 心灵花园

## 概述

心灵花园是 Starnion 的情感健康仪表板。它可视化日记中记录的情感，并通过 AI 伙伴 Nion 传递与情境相关的信息。它不仅是简单的统计界面，更是一个培养心理健康的空间。

> 日记写得越多，情感层次越丰满，治愈之树也会不断成长。

---

## 布局

心灵花园分为三列。

```
┌─────────────────┬──────────────────┬─────────────────┐
│  治愈之树        │   Nion 中心      │ 咨询空间         │
│  情感图表        │  （今日心情）     │  （统计）        │
│  心情日历        │                  │  （心灵花朵）     │
│                 │                  │  （进行中的目标） │
└─────────────────┴──────────────────┴─────────────────┘
```

---

## 左列：治愈之树和心情日历

### 治愈之树

随着日记条目的积累，树会不断生长。

| 元素 | 说明 |
|------|------|
| **树枝** | 5种情感色彩的能量流动 |
| **心形果实** | 每种情感类型一个，最多5个，带跳动动画 |
| **记忆之叶** | 每10篇日记1片叶子，最多8片 |
| **中央宝珠** | 代表生命力的金色球体 |

### 情感层次图表

以水平条形图显示日记中的情感分布。

- 显示每种情感的记录次数
- 显示前5种情感
- 底部计数器：**"共 N 条记忆"**

### 月度心情日历

月度日历，每天以彩色圆点显示当天的情感。

| UI 元素 | 说明 |
|---------|------|
| **< / >** 按钮 | 导航到上个月/下个月（不能跳到未来） |
| **彩色圆点** | 以当天日记心情的颜色显示 |
| **今日高亮** | 金色边框 + 金色字体 |
| **星期徽章** | 周日（粉色）/ 周六（蓝色）/ 工作日（白色）圆形徽章 |
| **图例** | Joy・Peace・Calm・Sad・Down 的颜色参考 |

## 中列：Nion 中心

### Nion 角色

AI 伙伴 Nion 以与当前时间匹配的状态出现。

| 时间 | Nion 状态 |
|------|----------|
| 午夜（0时） | 治愈 |
| 每小时之后 | 照料植物 → 守望 → 擦拭 → 打扫 |

心形、星星和圆点围绕 Nion 旋转，持续动画。

### 今日心情

- 最近一篇日记的情感会自动显示。
- 也可以使用底部的 **5个心情选择按钮** 直接设置今日心情。
- Nion 的治愈消息会根据选择的心情更新。

### 治愈消息

每种心情对应一条安慰消息。

| 心情 | 消息示例 |
|------|---------|
| Joy（非常好） | "多么灿烂的一天！请珍惜这份喜悦" |
| Peace（好） | "内心感到平静。请细细品味这一刻" |
| Calm（一般） | "平凡的日子也是珍贵的记录。你做得很好" |
| Sad（不好） | "这样的感受没关系。Nion 就在你身边" |
| Down（很不好） | "你累了。让我们慢慢来，一步一步" |

## 右列：咨询空间

### 会话统计卡

| 项目 | 说明 |
|------|------|
| **环形图** | 目标完成率（%） |
| **本周记录** | 过去7天的日记条目数 |

### 心灵花朵

三个正念活动的快捷按钮。

| 按钮 | 跳转 |
|------|------|
| 冥想 | 日记页面 |
| 日记 | 日记页面 |
| 聊天 | 聊天页面 |

### 进行中的目标

显示最多3个当前状态为 `in_progress` 的目标。

- 图标 + 标题 + 进度条 + 百分比
- 紫色到粉色渐变进度条

## 心情系统

心灵花园使用5种基本情感值。

| 键值 | 标签 | 颜色 | 表情 |
|------|------|------|------|
| `매우좋음` | Joy | 金色 `#fbbf24` | ✨ |
| `좋음` | Peace | 绿色 `#34d399` | 🌿 |
| `보통` | Calm | 紫色 `#a78bfa` | 💜 |
| `나쁨` | Sad | 蓝色 `#60a5fa` | 💙 |
| `매우나쁨` | Down | 靛蓝 `#818cf8` | 🌙 |

当 AI 通过聊天保存自由格式的情感（"兴奋"、"疲惫"、"糟糕"等）时，会自动归一化为这5个值之一。

## 数据流

```
聊天 / 写日记
    ↓
保存到 diary_entries 表（心情已归一化）
    ↓
/api/diary?limit=100  <── 心灵花园加载时调用
/api/goals
    ↓
情感计数汇总      → 情感层次图表
日期→心情映射     → 月度心情日历
最近的心情        → Nion 治愈消息
进行中的目标      → 咨询空间目标区域
```

## 常见问题

**Q. 心情日历上没有显示圆点。**

圆点仅在心情为5种基本值（`매우좋음 / 좋음 / 보통 / 나쁨 / 매우나쁨`）之一的条目上显示。v1.3.3 之前通过聊天保存的现有条目会通过该迁移自动归一化。

**Q. Nion 的图片一直在变。**

Nion 的状态根据当前时间每小时轮换一次。这是预期行为。

**Q. 我点了心情按钮但没有保存。**

心情选择按钮仅更改**当前屏幕上显示的心情**。要永久保存心情，请通过聊天或日记菜单写一篇日记。

**Q. 没有显示目标。**

在目标菜单中添加目标，并将状态设置为"进行中"，即可在咨询空间中看到。
