# 记账 & 预算

## 概述

Starnion 的记账功能是**用自然语言记录的 AI 驱动账本**。输入"午餐泡菜汤9000韩元"之类的内容，AI 会自动分类、解析日期和金额并立即记录。您也可以附上收据照片。

无需学习复杂的应用操作。用说话的方式记录支出，只需问"这个月我花了多少钱？"就能获得分析。设置分类预算可在超出时收到警告，并从消费模式分析中获得储蓄建议。

## 分类系统

记录支出时，AI 会分析并自动分配分类。

### 默认分类

| 分类 | 示例项目 |
|------|----------|
| 餐饮 | 午餐、晚餐、超市购物、便利店食品、外卖 |
| 咖啡馆 | 咖啡店、咖啡馆、饮品、甜点 |
| 交通 | 地铁、公交、出租车、油费、过路费 |
| 购物 | 服装、电子产品、家居用品、网购 |
| 订阅 | Netflix、YouTube Premium、音乐流媒体、应用订阅 |
| 医疗 | 就诊、药房、体检、医疗用品 |
| 文化 | 电影、演出、展览、健身房、爱好 |
| 其他 | 不适合上述分类的项目 |

### 自定义分类

您可以指定默认分类以外的分类。

```
用户：宠物粮食35000韩元，使用"宠物"分类
AI：  宠物/宠物粮食/35000韩元 已记录 🐾

用户：英语学校月学费150000韩元，分类：教育
AI：  教育/英语学校/150000韩元 已记录 📚
```

### 自动分类工作原理

```
输入："星巴克美式5500韩元"
  ↓
AI 分析：
  - "星巴克" → 识别为咖啡品牌
  - "美式" → 识别为饮品类型
  → 分类：咖啡馆（置信度：高）

输入："药房泰诺8200韩元"
  ↓
AI 分析：
  - "药房" → 医疗相关场所
  - "泰诺" → 药品
  → 分类：医疗（置信度：高）

输入："在XX超市买了东西"
  ↓
AI 分析：
  - "超市" → 购物相关
  - 具体商品不明确
  → 分类：购物（置信度：中）
  AI："您买了什么？我可以更准确地分类。"
```

```

AI：  预算已设置！
      咖啡馆：         100,000韩元
      交通：           150,000韩元

用户：显示整体预算状态
AI：  🔧 正在运行 `budget_status`...

      📋 3月预算状态
      ──────────────────────────────────

      餐饮    400,000  342,500  57,500  85.6% ⚠️
```

### 在网页端设置预算

进入 设置 > 预算管理，直接为每个分类输入预算。

- **警告阈值**：支出超过该百分比时显示警告标志（默认：70%）
- **危险阈值**：支出超过该百分比时显示危险标志（默认：90%）

```
预算示例：

──────────────────

```

## 预算提醒

超出预算或接近阈值时收到提醒。

### 提醒工作原理

```
[70%警告 — 注意阶段]
AI 提醒："您已使用餐饮预算的70%。
           本月剩余预算：120,000韩元
           未来22天每天可花费约5,455韩元。"

[90%警告 — 危险阶段]
AI 提醒："⚠️ 您已使用交通预算的92%！
           仅剩12,000韩元。
           请尝试减少本月交通支出。"

[100%超支]
AI 提醒："❌ 您已超出咖啡馆预算。
           预算：100,000韩元 / 已用：127,000韩元（超出27,000韩元）
           建议重新审视本月的咖啡馆预算。"
```

### Telegram 提醒集成

连接 Telegram 频道，即可在手机上接收预算提醒。即使外出时也能实时掌握消费情况。

### 充分利用记账功能的技巧

1. **立即记录**：支出发生后马上录入，保持记录准确，不遗漏任何内容。

2. **使用收据照片**：对于大额购物或多行项目，拍下收据让 AI 一次自动录入多条记录。

3. **月初设置预算**：每月初设置分类预算，使消费管理更加容易。

4. **养成每周回顾的习惯**：每周问一次"总结本周支出"，快速了解消费模式。

5. **使用 AI 洞见**：定期问"分析我的消费模式"，发现可以节省的地方。

## 常见问题

**Q. 我输入了错误的金额，如何修正？**

A. 直接说"把我刚输入的[项目][金额]改成正确的数字"。也可以在网页端的交易列表中直接点击该条目。

```
用户：把刚才输入的便利店从5200韩元改为4200韩元
AI：  便利店支出已从5,200韩元更新为4,200韩元。
```

**Q. 自动分类错误了。**

A. 说"把分类从[原分类]改为[目标分类]"，会立即更新。如果 AI 对某类记录持续误分类，欢迎提供反馈——这有助于改进。

**Q. 可以输入上个月的支出吗？**

A. 可以，只需指定日期即可。例如，输入"2月15日超市32,000韩元"会记录在该日期。

**Q. 可以将支出数据导出到Excel吗？**

A. CSV导出功能目前正在开发中。目前可以请求"发给我本月的支出列表"以文本格式接收。

**Q. 可以管理共同支出（家庭、伴侣）吗？**

A. 目前系统基于账户，供个人使用。共享账本功能计划在未来更新中推出。

**Q. 超出预算时会收到什么提醒？**

A. 您会在使用70%（警告）、90%（危险）和100%超支（紧急）时收到提醒。连接Telegram可在手机上接收提醒。

**Q. 如何记录收入？**

A. 输入"工资300万韩元到账"或"收入3,000,000韩元"之类的内容，会自动分类为收入。正数视为收入，负数视为支出。

**Q. 固定订阅付款可以自动记录吗？**

A. 默认为手动录入，但对于固定重复金额，可以说"每月自动记录Netflix 17,000韩元"，之后每月会自动记录。
