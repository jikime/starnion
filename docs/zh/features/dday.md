# 倒计时

## 概述

倒计时功能提供实时倒计时，显示距重要日期还有多少天。注册您想记住的日期——生日、纪念日、考试、旅行出发日——AI 会立即回答"距我的考试还有多少天？"之类的问题。

您最多可以注册30个倒计时，支持未来日期（D-N）和过去日期（D+N）。

---

## 注册倒计时

### 在网页端注册

1. 点击侧边栏中的 **倒计时**。
2. 点击 **+ 添加倒计时** 按钮。
3. 输入以下信息：
   - **名称**（必填）：倒计时标题（如生日、托业考试）
   - **日期**（必填）：目标日期（从日历中选择）
   - **图标**（可选）：代表性表情符号（默认：📅）
   - **描述**（可选）：额外备注
4. 点击 **保存**。

倒计时列表按日期升序排列（最近日期在前）。

### 通过聊天注册

用自然语言告诉 AI，它会自动设置倒计时。

**示例对话：**

> 用户："设置圣诞节倒计时"
>
> AI："圣诞节倒计时已设置！距2025年12月25日还有D-292天 🎄"

> 用户："把每年5月20日的结婚纪念日设为年度循环倒计时"
>
> AI："结婚纪念日倒计时已设为年度循环！距下次纪念日（5/20）还有D-73天 💍"

> 用户："我的托业考试是6月15日。设置倒计时。"
>
> AI："托业考试倒计时已设置！距考试（6/15）还有D-99天。祝好运！📝"

**自然语言日期表达：**
AI 自动将自然语言日期表达转换为具体日期：

| 表达 | 转换为 |
|------|--------|
| "下周一" | 下周一的日期 |
| "本月底" | 当月最后一天 |
| "圣诞节" | 12月25日 |
| "6月15日" | YYYY-06-15 |
| "我的生日3月10日" | YYYY-03-10 |

## D-N 与 D+N

了解倒计时的计数方式让您更好地使用该功能。

### D-N（剩余天数）

显示距未来日期还剩多少天。

```
今天：2025年3月8日
目标日期：2025年6月15日
显示：D-99
```

### D-Day!（当天）

当今天与注册日期相同时显示。

```
今天：2025年6月15日
目标日期：2025年6月15日
显示：D-Day!
```

### D+N（已过天数）

显示自过去日期以来已经过了多少天。注册纪念日等过去日期时很有用。

```
今天：2025年3月8日
目标日期：2024年3月8日
显示：D+365
```

### 使用示例

| 用途 | 日期示例 | 显示示例 |
|------|----------|----------|
| 考试倒计时 | 2025-06-15 | D-99 |
| 结婚纪念日（今天） | 2025-03-08 | D-Day! |
| 相识纪念日 | 2024-01-01 | D+431 |
| 旅行出发 | 2025-04-01 | D-24 |

## 通知

倒计时通知目前通过聊天主动查询或查看每周报告来工作。

### 包含在每周报告中

即将到来的倒计时信息包含在每周自动生成的报告中：

```
[即将到来的倒计时]
- D-7: 托业考试（6/15）
- D-30: 朋友生日（4/7）
- D-93: 济州岛旅行（6/9）
```

### 通过聊天查询

随时向 AI 询问当前的倒计时状态：

> 用户："我下一个倒计时是什么时候？"
>
> AI："您最近的倒计时是**托业考试**。D-7，6月15日。最后冲刺，加油！"

> 用户："本月有什么倒计时？"
>
> AI："本月有2个倒计时：\n- 3/10（D-2）：父母结婚纪念日\n- 3/20（D-12）：项目截止日期"

## 常见问题

**Q. 可以注册多少个倒计时？**
最多30个。达到30个后，需要删除旧倒计时才能添加新的。

**Q. 如何设置年度循环纪念日？**
在聊天中说"设为年度循环"，或在网页注册页面启用 **循环** 选项。设定日期过后，自动更新为次年的日期。

**Q. 倒计时会根据今天的日期自动重新计算吗？**
是的，倒计时值每天会自动重新计算。如果昨天是D-10，今天就会显示D-9。

**Q. 可以编辑倒计时吗？**
点击网页端的倒计时可以打开编辑页面，在那里可以更改日期、名称、图标和描述。

**Q. 注册过去日期时显示什么？**
过去的日期以D+N格式显示。当您想纪念经过的时间时（如与伴侣相识的日子或公司入职日）非常有用。

**Q. 倒计时会自动包含在每周报告中吗？**
是的，D-30以内即将到来的倒计时会在生成每周报告时自动包含。这让您可以在新一周开始前回顾即将到来的重要日期。
