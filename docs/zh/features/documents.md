# 文档管理

## 概述

Starnion 的文档管理功能让您可以上传多种格式的文档——PDF、Word、Excel、PowerPoint、纯文本等——并与 AI 一起使用。文档上传后，AI 会自动读取并分析其内容，您可以在聊天中用自然语言提问，如"找出合同中的交货截止日期"。

上传的文档会经过文字提取 → 分块 → 向量嵌入的流程，以可搜索的状态存储。AI 随后检索相关段落以提供准确的答案。

---

## 支持格式

| 格式 | 扩展名 | 文字提取 | AI 搜索 | AI 聊天 |
|------|--------|----------|---------|---------|
| PDF | .pdf | 是 | 是 | 是 |
| Word | .doc, .docx | 是 | 是 | 是 |
| Excel | .xls, .xlsx | 是 | 是 | 是 |
| PowerPoint | .ppt, .pptx | 是 | 是 | 是 |
| 纯文本 | .txt | 是 | 是 | 是 |
| Markdown | .md | 是 | 是 | 是 |
| CSV | .csv | 是 | 是 | 是 |

> **HWP（韩国文字处理器）文件**目前不直接支持。请将 HWP 文件导出为 PDF 后再上传。

## 上传文档

### 在聊天中附加文件

点击聊天输入框底部的文件附件按钮并选择您的文档，或将文件直接拖放到聊天窗口中。

```
用户：[附加 合同.pdf]
AI：  合同.pdf 已上传。正在分析内容... 完成！
      您现在可以提问关于此文档的问题。

用户：这份合同的交货截止日期是什么？
AI：  根据合同第4条，交货截止日期为2025年6月30日。
      具体内容为："乙方应于2025年6月30日前交付产品。"
```

### 从文档菜单上传

进入顶部菜单 > **文档** 标签，然后点击 **上传文档** 按钮或将文件拖放到拖放区域。

## 文档处理流程

文档上传后，以下步骤会在后台自动运行。

```
1. 上传     → 文件保存到 MinIO 存储
2. 解析     → 根据文件格式提取文字
3. 分块     → 长文档分割成适当大小的片段
4. 嵌入     → 每个片段转换为 AI 可理解的向量
5. 存储     → 以可搜索状态保存到数据库
6. 完成     → 可立即在 AI 聊天中使用
```

> 处理时间因文件大小而异。1 MB 以下的文档通常在10秒内完成。

## 搜索文档并进行问答

您可以向 AI 提出关于任何已上传文档内容的自然语言问题。

**合同相关：**
```
用户：找出合同中的违约条款。
AI：  根据第8条（违约条款）：若乙方无法在交货截止日期前完成，
      乙方须按合同金额的0.1%支付每天的违约金。
```

**会议纪要摘要：**
```
用户：只摘要上次会议纪要中做出的决定。
AI：  3月5日会议决定：
      1. 新功能上线日期：4月15日
      2. 市场营销预算：500万韩元已批准
      3. 负责人：金组长（开发）、李总监（市场）
```

**Excel数据分析：**
```
用户：告诉我1月销售数据中销量前5的产品。
AI：  按1月销量前5的产品：
      第1名：产品A — 1,250件（375万韩元）
      第2名：产品B — 980件（294万韩元）
      ...
```

## 在 AI 聊天中使用文档

### 附加并提问

在聊天中直接附加文档，可以基于该文档进行对话。

```
用户：[附加 员工手册.pdf] 总结一下年假政策。
AI：  手册第3章（假期政策）摘要：
      - 年假累计：工龄满1年后15天，之后每2年增加1天（最多25天）
      - 使用期限：须在累计日期起1年内使用
      - 申请方式：须提前3天获得团队负责人批准
      - 半天假：可分为上午（09:00-13:00）或下午（14:00-18:00）
```

### 比较多份文档

```
用户：比较两份报价单，哪个更有利？
AI：  供应商A和供应商B报价比较：
      供应商A：总计450万韩元，30天交货，1年保修
      供应商B：总计480万韩元，25天交货，2年保修
      → 供应商B贵30万韩元，但交货提前5天，保修期是前者的两倍。
```

## AI 文档生成

您也可以让 AI 起草新文档或报告。生成的文档会自动保存到您的文档列表中。

```
用户：起草3月工作报告。本月完成了3项新功能开发和12个bug修复。
AI：  您的3月工作报告已创建。
      [已生成 report.docx]
      您可以从文档标签中下载。
```

## 管理文档

### 查看文档列表

在 **文档** 标签中，您可以按时间顺序查看所有已上传的文档。最多显示100份文档。

### 删除文档

点击列表中文档旁的删除按钮，或在聊天中请求删除。

```
用户：删除"2024年年度报告.pdf"
AI：  2024年年度报告.pdf 已删除。
```

## 使用技巧与常见问题

**Q. 我上传了文档，但 AI 说不知道内容。**
A. 文件处理可能需要一点时间。请等待片刻后重试问题。大型文件可能需要1-2分钟处理。

**Q. 可以读取扫描的PDF吗？**
A. 扫描的PDF由图片组成，难以提取文字。请使用OCR处理过的PDF或包含可选择文本的PDF。

**Q. 可以只针对Excel文件中的特定工作表提问吗？**
A. 可以——只需在问题中提到工作表名称即可。例如："告诉我销售工作表中3月的总计。"

**Q. 我上传的文档安全存储吗？**
A. 所有文档都存储在加密的MinIO存储中，只有您能访问。
