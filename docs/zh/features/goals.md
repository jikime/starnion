# 目标管理

## 概述

Starnion 的目标管理功能是一个 AI 驱动的系统，帮助您系统地追踪并实现大大小小的日常目标。AI 不仅仅是简单地列出任务，还会全程陪伴您追踪进度，给予鼓励并分析进展情况。

支持的目标类型：
- **任务/项目** — 有截止日期的一次性项目（提交报告、完成项目）
- **习惯** — 每日重复的行动（运动、阅读、冥想）
- **储蓄目标** — 积累特定金额（旅行基金、应急基金）
- **消费限制** — 将支出控制在预算内（本月餐饮预算30万韩元）
- **通用目标** — 不属于上述类别的个人目标

---

## 创建目标

### 在网页端创建

1. 点击侧边栏中的 **目标**。
2. 点击 **+ 新建目标** 按钮。
3. 填写以下字段：
   - **目标名称**（必填）：您想要实现的目标简短描述
   - **图标**：代表目标的表情符号（默认 🎯）
   - **分类**：健康、财务、自我提升、工作等
   - **目标值**：定义完成的数字（如30次、10万韩元）
   - **单位**：次、天、韩元、公里等
   - **开始日期/结束日期**：设置时间段
   - **描述**：关于目标的额外说明（可选）

### 通过聊天创建

自然地说话，AI 会自动设置目标。无需确认步骤，立即保存。

**示例对话：**

> 用户："设置本月读5本书的目标"
>
> AI："目标已保存！您设置了本月读5本书的目标。加油 📚"

> 用户："我决定每天早晨锻炼"
>
> AI："多棒的习惯目标！我已保存'每天早晨锻炼'目标。从今天开始 💪"

> 用户："我要把本月餐饮支出控制在30万韩元以内"
>
> AI："财务目标已保存！您的目标是本月餐饮控制在30万韩元以内。一起管理好它 👍"

**触发模式** — 出现以下任何短语时会自动创建目标：

| 示例短语 | 目标类型 |
|----------|----------|
| "我决定做X"、"设置目标做X" | 通用/习惯 |
| "我需要每天做X" | 习惯 |
| "我需要本周完成X" | 任务 |
| "我要存X"、"我花费不超过X" | 财务 |
| "我决定存X" | 储蓄 |

## 打卡系统

打卡对于习惯类目标尤为重要。每天记录是否完成目标，建立连续打卡记录。

### 如何打卡

**在网页端：**
1. 在目标列表中找到目标卡片。
2. 点击卡片底部的 **今天打卡** 按钮。
3. 确认打卡，连续记录随之更新。

**通过聊天：**

> 用户："我今天锻炼了"
>
> AI："太棒了！今天的锻炼目标已打卡。当前连续：连续7天 🔥"

> 用户："今天阅读目标打卡"
>
> AI："已打卡！本月阅读进度：3/5本（60%）"

### 连续打卡计数

连续打卡是您连续打卡的天数。即使昨天错过了，今天打卡也能保持连续（一天宽限期）。

| 连续天数 | 状态 |
|----------|------|
| 0天 | 未开始或已重置 |
| 1-6天 | 刚刚起步 |
| 7-29天 | 势头正好 🔥 |
| 30天以上 | 习惯已养成 🏆 |

连续缺卡两天或以上会将连续打卡重置为零。

## 评估目标

### 更新进度

对于数字目标（储蓄、消费等），您可以直接更新进度。

> 用户："旅行储蓄目标已完成50%"
>
> AI："已过半！我已将旅行储蓄目标更新为50%。距终点线还有一半 ✈️"

> 用户："运动目标进度70%"
>
> AI："令人印象深刻！我已将运动目标进度更新为70%。看来本月内就能完成！"

### 更改目标状态

目标可以有三种状态：

| 状态 | 含义 | 如何更改 |
|------|------|----------|
| `进行中` | 正在努力中 | 默认状态 |
| `已达成` | 目标已完成 | "我完成了"、"我成功了"、"搞定" |
| `已放弃` | 不再追求 | "我要放弃"、"我要停止"、"取消它" |

> 用户："我完成了阅读目标的全部5本书！"
>
> AI："恭喜！🎉 您已实现本月阅读目标！读完了全部5本书。太了不起了！"

## 通知

与目标相关的通知通过每周报告发送。当设定的结束日期临近时，AI 会在聊天中自动提醒您。

**自动通知触发条件：**
- 距结束日期7天：" 距您的目标截止日期还有7天"
- 距结束日期1天："您的目标截止日期是明天"
- 连续打卡有风险："您昨天错过了打卡——今天打卡以保持连续记录"

---

## 与每周报告集成

每周自动生成的报告包含目标状态部分：

```
[本周目标状态]
进行中的目标：3个
- 每日锻炼（连续：5天）— 本周打卡5/7天
- 读5本书（完成60%）— 已完成3/5本，截止日期还有18天
- 餐饮消费限制（完成45%）— 预算剩余135,000韩元

本周已达成：无
本周新增：读5本书（3/1开始）
```

## 使用技巧与常见问题

**Q. 同时可以有多少个目标？**
最多可以同时管理20个进行中的目标。已达成或已放弃的目标没有数量限制。

**Q. 连续打卡断了，需要从零开始吗？**
是的，连续打卡断了会重置为零。但是，如果您错过了昨天的打卡但今天打卡了，明天打卡将保持连续（一天宽限期政策）。

**Q. 通过聊天创建目标时，数字值会自动设置吗？**
习惯类目标（如"每天锻炼"）会自动设置单位为"天"。如果提到了具体数字——例如"读5本书"——目标值也会自动设置。

**Q. 如何查看已完成的目标？**
从目标页面顶部的筛选器中选择 **已完成** 可以看到您的已达成目标。在聊天中，只需说"显示我已达成的目标"。

**Q. 财务目标和记账功能会重叠吗？**
财务目标用于设定目标金额并追踪大方向。记账功能用于详细记录每日支出。两个功能相辅相成——记账数据可用于分析财务目标的进展。
