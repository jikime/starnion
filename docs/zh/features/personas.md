# 人格设置

## 概述

人格（Persona）功能让您可以自定义 Nion 的名称、性格和角色。创建多个人格并根据场景切换，就像与不同性格的 AI 对话一样。

**主要特点：**
- 人格自定义：设置名称、角色、性格描述和问候语
- 多人格支持：创建多个人格，按需切换
- 默认人格：Nion — 内置默认人格
- 对话中切换：在聊天过程中即可更换人格

---

## 什么是人格？

人格是 **定义 AI 性格和角色的配置文件**。同一个 AI 会根据不同人格以不同方式回应。

| 组成部分 | 说明 | 示例 |
|---------|------|------|
| **名称** | 人格的显示名 | "Nion"、"理财教练"、"英语老师" |
| **角色** | 专业领域 | "个人理财顾问" |
| **性格描述** | 对话风格和态度 | "亲切友善，注重节约……" |
| **问候语** | 对话开始时发送的消息 | "你好！今天一起看看你的财务状况吧。" |

---

## 默认人格

Starnion 内置了名为 **Nion** 的默认人格。

- **名称**：Nion
- **角色**：个人 AI 助手
- **性格**：友好温暖的对话风格，善用表情符号，关心用户的日常生活
- **特点**：均衡支持所有功能 — 日记、记账、备忘录、目标等

## 创建人格

### 从 Web 界面创建

1. 进入 **设置 > 人格** 菜单。
2. 点击 **+ 新建人格** 按钮。
3. 填写以下信息：
   - **名称**（必填）：人格的称呼
   - **角色**（选填）：该人格的专业领域
   - **性格描述**（必填）：AI 参考的对话风格和态度说明
   - **问候语**（选填）：对话开始时自动发送的消息
4. 点击 **保存**。

### 人格示例

以下是一些可创建的人格参考：

#### 理财教练

```
名称：理财教练
角色：个人理财顾问
性格描述：专注于节约和投资，发现不必要的支出时温和地指出。
         从不吝啬鼓励，即使是小额节约也会表扬。为财务目标的达成加油打气。
问候语：你好！今天一起检查一下你的财务状况吧？
```

**对话示例：**
```
用户：今天在咖啡店花了 30 元
AI：  一杯咖啡呢！你这个月咖啡消费已经累计到 200 元了。
      小提示：很多咖啡店自带杯子可以打折哦！
```

#### 日记伙伴

```
名称：暖暖
角色：日记助手
性格描述：感性且富有同理心。深深共情用户的感受，
         帮助回顾一天。使用温暖的语气。
问候语：今天过得怎么样？跟我说说吧。
```

**对话示例：**
```
用户：今天演讲搞砸了……
AI：  真的吗……准备了那么久一定很沮丧吧。
      哪个部分觉得遗憾？说出来的话，我们一起想想下次怎么做得更好。
```

#### 英语老师

```
名称：Teacher Emma
角色：英语会话教师
性格描述：只用英语对话，温和地纠正语法错误。
         根据用户水平调整对话难度，自然地教授新表达方式。
问候语：Hi! Ready for some English practice today?
```

**对话示例：**
```
用户：I goed to the store yesterday
AI：  Great effort! Just a small correction:
      "goed" → "went" (go-went-gone is an irregular verb)
      "I went to the store yesterday." Perfect otherwise!
      What did you buy there?
```

## 切换人格

### 从 Web 界面切换

1. 在 **设置 > 人格** 中，点击想要激活的人格旁的 **激活** 按钮。
2. 或点击聊天页面顶部的人格名称进行更改。

### 在聊天中切换

对话中可以用自然语言请求切换人格。

```
用户：切换到英语老师模式
AI：  Hi! I'm Teacher Emma. Let's practice English together!

用户：切回 Nion
AI：  我回来啦！有什么可以帮你的？
```

## 提示与常见问题

**Q：最多可以创建多少个人格？**
没有数量限制，可以自由创建。但同一时间只能激活一个人格。

**Q：切换人格后之前的对话上下文会消失吗？**
不会。切换人格后对话历史和记忆都会保留。只有 AI 的语气和回复风格会随新人格改变。

**Q：可以编辑默认的 Nion 人格吗？**
可以修改默认人格的性格描述和问候语。名称不可更改。

**Q：可以为不同人格使用不同的模型吗？**
目前人格和模型是分别配置的。模型设置在 **设置 > 模型** 菜单中管理。

**Q：有创建好人格的技巧吗？**
性格描述越具体，AI 就越能保持一致的角色。与其只写"友善"，不如写"总是不吝鼓励，即使是小小的进步也会夸奖，使用温暖亲切的语气"。
