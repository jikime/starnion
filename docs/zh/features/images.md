# 图片

## 概述

Starnion 的图片功能让您可以上传照片进行 AI 分析，自动识别收据并记录到账本中，以及仅凭文字描述生成新图片。

在聊天中附加图片时，您可以使用自然语言命令，如"把这张收据记录到我的账本"、"这张照片里是什么？"或"分析这个图表"。所有上传的图片都收集在图库中，AI 分析结果也随之保存。

## 上传图片

### 在聊天中附加图片

点击聊天输入框中的文件附件按钮，或将图片拖放到聊天窗口中进行上传。

```
用户：[附加图片] 这是什么，能描述一下吗？
AI：  这张照片展示的是一座传统韩国民居（韩屋）。它有瓦片屋顶
      和木柱，屋檐弧线优美。传统建筑与背景中山景的
      和谐共融尤为引人注目。
```

### 从图库上传

进入顶部菜单 > **图片** 标签，然后点击上传按钮或通过拖放添加图片。

## 图片分析

AI 可以描述图片中的内容并回答关于图片的各种问题。

**分析照片内容：**
```
用户：[附加餐厅菜单照片] 我应该点什么？
AI：  看这份菜单，这家餐厅主打韩国料理。推荐：
      - 大酱汤，8000韩元：可靠的经典之选
      - 辣炒猪肉，9000韩元：看起来是受欢迎的菜品
      - 拌饭，8500韩元：健康之选
      11,000韩元的午市套餐很划算。
```

**图表和图形分析：**
```
用户：[附加销售图表图片] 分析这个图表。
AI：  这是一个显示月度销售趋势的柱状图。
      - 1月-3月：上升趋势（平均230万韩元）
      - 4月：急剧下降（180万韩元，比上月-22%）
      - 5月-6月：恢复趋势
      4月下降的原因似乎需要调查。
```

**文字提取（OCR）：**
```
用户：[附加名片照片] 提取联系方式。
AI：  从名片提取的信息：
      姓名：金敏俊
      职务：市场团队组长
      公司：Starnion株式会社
      电子邮件：minjun@starnion.com
      电话：010-1234-5678
```

## 收据OCR——自动记账功能集成

上传收据照片，AI 会自动识别金额和项目，然后记录到账本中。

```
用户：[附加便利店收据照片] 保存到我的账本。
AI：  收据已识别：
      - 三角饭团：1,200韩元
      - 美式咖啡：1,500韩元
      - 饮料：1,800韩元
      合计：4,500韩元（2025-03-08 14:32）

      是否将4,500韩元记录为餐饮支出？是这样吗？

用户：是的，没错。
AI：  便利店/4,500韩元 已记录！今日餐饮支出合计：12,800韩元
```

**批量收据处理：**
```
用户：[附加多张收据照片] 记录本周所有的收据。
AI：  识别到3张收据：
      1. 超市收据 — 45,200韩元（3月5日）
      2. 咖啡馆收据 — 8,500韩元（3月6日）
      3. 餐厅收据 — 32,000韩元（3月7日）
      合计85,700韩元已记录到账本中。
```

在 **图片** 标签中，您可以一目了然地查看所有使用过的图片。

### 图库筛选

| 筛选 | 描述 |
|------|------|

| 已生成 | AI创建的图片 |

| 已分析 | 上传并由AI分析的图片 |

- 查看AI分析结果文字

## AI 图片生成

您可以从文字描述（提示词）生成图片。生成的图片会自动保存到您的图库中。

```
用户：画一幅从汉江看到的首尔夜景。
AI：  正在生成图片...
      [图片已生成]
      我已创建了从汉江看到的首尔夜景图片。
      您也可以在图库中找到它。
```

**指定不同风格：**
```
用户：用水彩风格画济州偶来小路的风景。
AI：  [已生成水彩风格济州偶来小路图片]

用户：创作一个可爱的猫咪角色贴纸。
AI：  [已生成猫咪贴纸图片]
```

## 图片编辑

您可以用 AI 修改现有图片。编辑后的图片以 `edited_` 前缀保存到图库中。

```
用户：[附加照片] 把这张照片的背景虚化（焦外虚化效果）。
AI：  我已创建了背景虚化的图片。
      [编辑后的图片]

用户：[附加照片] 去掉这张照片中的人。
AI：  我已去掉了人并恢复了背景。
      [编辑后的图片]
```

## 使用技巧与常见问题

**Q. 收据识别效果不好。**
A. 在光线充足的环境中正面拍摄，确保整张收据在画面内。拍摄前将皱巴巴的收据展平会显著提高识别准确率。

**Q. 可以一次上传多张图片吗？**
A. 可以，在聊天中可以一次附加多张图片。但由于AI分析会一次性处理所有图片，数量非常多时响应可能较慢。

**Q. 可以下载生成的图片吗？**
A. 在图库中点击图片查看全尺寸，然后右键或长按即可保存。

**Q. 包含个人信息的图片安全吗？**
A. 上传的图片静态加密存储，只有您能访问。但在上传包含敏感信息的图片时请谨慎。
