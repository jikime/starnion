# Tavily 搜索集成

将 Starnion 连接到 Tavily，让 AI 智能体能够执行实时网络搜索，提供最新信息。当 Ninon 需要最新信息时，会自动执行 Tavily 搜索。

---

## 概述

使用 Tavily 集成您可以：

- **实时网络搜索**: 搜索最新新闻、信息和数据
- **新闻搜索**: 查找最近的新闻文章
- **学术搜索**: 搜索学术论文和研究资料

> **可选功能：** Tavily 集成默认处于禁用状态。您需要注册 API 密钥并启用技能才能使用。

---

## 支持的功能

| 功能 | 说明 |
|------|------|
| 网络搜索 | 实时网络搜索提供最新信息 |
| 新闻搜索 | 最近新闻文章搜索 |
| 学术搜索 | 学术论文和研究资料搜索 |

---

## 前置准备：获取 Tavily API 密钥

### 步骤 1：创建 API 密钥

1. 访问 [tavily.com](https://tavily.com)。
2. 创建账户或登录。
3. 在控制面板中找到 **API Keys** 区域。
4. 点击 **Create API Key** 按钮。
5. 复制生成的 API 密钥（`tvly-...` 格式）。

> **免费额度：** Tavily 每月提供 1,000 次免费搜索。

---

## 设置方法

### 在 Web UI 中注册 API 密钥

1. 登录 Starnion Web UI。
2. 左侧菜单 → **Settings** → **Integrations** 标签页。
3. 找到 **Tavily** 区域的 **API Key** 输入字段。
4. 粘贴复制的 API 密钥（`tvly-...`）。
5. 点击 **保存** 按钮。
6. 开启 **Tavily 技能启用** 开关。

## 自动搜索行为

Ninon 在以下情况下自动触发 Tavily 搜索：

- 请求最新新闻或实时信息时
- 询问实时数据（股价、汇率、天气等）时
- 需要 AI 训练数据中未包含的最新信息时

> **提示：** 使用"搜索"、"查找"等词语会增加触发 Tavily 搜索的可能性。

---

## 断开连接

1. Settings → Integrations → Tavily 区域。
2. 点击 **断开连接** 按钮。
3. 存储的 API 密钥将立即删除。

---

## 故障排除

### "Tavily 集成未配置"

请在 Settings → Integrations → Tavily 中确认是否已注册 API 密钥。

### "Tavily API 密钥无效"（401 错误）

- 确认 API 密钥是否正确。
- 在 [tavily.com](https://tavily.com) 控制面板中确认密钥是否处于活跃状态。

### "搜索次数已超过限制"（429 错误）

- 您可能已超过免费额度的每月搜索限制（1,000 次）。
- 升级到付费计划或等到下个月。

## 常见问题

**Q：Tavily 搜索结果总是准确的吗？**
A：结果基于网络搜索，建议始终验证来源。Ninon 会尽可能显示来源。

**Q：没有 Tavily 也能获取最新信息吗？**
A：没有 Tavily 集成时，AI 仅基于训练数据进行回答。如需实时信息，建议使用 Tavily 集成。

**Q：免费额度够用吗？**
A：对于个人使用，每月 1,000 次搜索通常足够。团队使用请考虑付费计划。
