# Gemini 集成

将 Google Gemini API 连接到 Starnion，让您能够在对话中使用 Gemini 模型。利用 Gemini 强大的图像分析和多模态处理等功能。

---

## 概述

使用 Gemini 集成您可以：

- **对话模型**: 使用 Gemini 2.0 Flash、Gemini 1.5 Pro、Gemini 1.5 Flash
- **嵌入向量**: 支持 gemini-embedding-001（768 维）
- **图像分析**: 上传图像进行内容分析和描述
- **多模态处理**: 文本和图像的综合理解

> **默认提供商：** Gemini 是 Starnion 的默认提供商。无需额外 API 密钥设置即可使用免费额度开始。

---

## 支持的模型

| 模型 | 特点 | 推荐用途 |
|------|------|----------|
| Gemini 2.0 Flash | 快速响应、最新模型 | 日常对话、快速问答 |
| Gemini 1.5 Pro | 长上下文（最多 1M token） | 长文档分析、复杂推理 |
| Gemini 1.5 Flash | 均衡性能 | 通用对话 |
| gemini-embedding-001 | 768 维嵌入向量 | 文档向量化、相似度搜索 |

## 前置准备：获取 Google AI Studio API 密钥

### 步骤 1：创建 API 密钥

1. 访问 [Google AI Studio](https://aistudio.google.com/)。
2. 使用 Google 账户登录。
3. 点击 **Get API key** 按钮。
4. 点击 **Create API key** → 选择或创建项目。
5. 复制生成的密钥（`AIza...` 格式）。

> **免费额度：** 每分钟 15 次请求，每天 1,500 次请求（2025 年标准）。

---

## 设置方法

### 在 Web UI 中注册 API 密钥

1. 登录 Starnion Web UI。
2. 左侧菜单 → **Settings** → **Models** 标签页。
3. 选择 **Google Gemini** 提供商。
4. 在 **API Key** 字段粘贴复制的密钥（`AIza...`）。
5. 勾选要使用的模型（可多选）。
6. 点击 **保存** 按钮。

API 密钥保存时自动验证。

## 使用方法

### 在对话中选择模型

开始对话时从模型下拉菜单中选择 Gemini 模型，或将 Gemini 模型绑定到角色以自动应用。

## 嵌入向量配置

Gemini 的嵌入向量模型用于文档搜索、相似度分析等。

- **模型**: gemini-embedding-001
- **维度**: 768
- **用途**: 对话历史向量化、文档搜索、模式分析

嵌入向量设置可在 `starnion.yaml` 或环境变量中配置。

## 故障排除

### "Gemini API 密钥无效"

- 确认 API 密钥是否为 `AIza...` 格式。
- 在 [Google AI Studio](https://aistudio.google.com/) 确认密钥是否处于活跃状态。

### "请求次数已超过限制"（429 错误）

- 您已超过免费额度（每分钟 15 次，每天 1,500 次）。
- 稍等片刻或切换到其他提供商。

### 图像分析无法工作

- 确认是否正在使用 Gemini 2.0 Flash 或 Gemini 1.5 Pro 模型。
- 确认图像格式是否受支持（JPEG、PNG、GIF、WebP）。

## 常见问题

**Q：可以一直免费使用 Gemini 吗？**
A：是的，在 Google AI Studio 免费额度内可以持续免费使用。超过限制可能会产生费用。

**Q：可以同时配置 Gemini 和其他提供商吗？**
A：是的，可以同时注册多个提供商的 API 密钥，并在每次对话中选择模型。

**Q：使用 Gemini 嵌入向量需要单独设置吗？**
A：注册 Gemini API 密钥后，嵌入向量将自动可用。可在 `starnion.yaml` 中调整详细设置。
