# ドキュメント管理

## 概要

Starnionのドキュメント管理機能では、PDF、Word、Excel、PowerPoint、プレーンテキストなど様々なフォーマットのドキュメントをアップロードし、AIと組み合わせて活用できます。ドキュメントをアップロードすると、AIが自動的に内容を読み取って分析するため、「契約書の納期はいつ？」のような自然言語での質問をチャットで行えます。

アップロードされたドキュメントは、テキスト抽出 → チャンク分割 → ベクターエンベディングのパイプラインを経て、検索可能な状態で保存されます。AIは関連するセクションを取得して正確な回答を提供します。

## 対応フォーマット

| フォーマット | 拡張子 | テキスト抽出 | AI検索 | AIチャット |
|------------|-------|------------|--------|-----------|
| PDF | .pdf | 対応 | 対応 | 対応 |
| Word | .doc, .docx | 対応 | 対応 | 対応 |
| Excel | .xls, .xlsx | 対応 | 対応 | 対応 |
| PowerPoint | .ppt, .pptx | 対応 | 対応 | 対応 |
| プレーンテキスト | .txt | 対応 | 対応 | 対応 |
| Markdown | .md | 対応 | 対応 | 対応 |
| CSV | .csv | 対応 | 対応 | 対応 |

> **HWP（ハングルワードプロセッサ）ファイル**は現在直接サポートされていません。HWPファイルはPDFに書き出してからアップロードしてください。

## ドキュメントをアップロードする

### チャットでファイルを添付する

チャット入力欄の下部にあるファイル添付ボタンをクリックしてドキュメントを選択するか、ファイルをチャットウィンドウに直接ドラッグ＆ドロップします。

```
ユーザー：[contract.pdf を添付]
AI:   contract.pdf をアップロードしました。内容を分析中... 完了！
      このドキュメントについて質問できます。

ユーザー：この契約書の納期はいつですか？
AI:   契約書の第4条によると、納期は2025年6月30日です。
      具体的には「甲は2025年6月30日までに製品を納品するものとする。」と記載されています。
```

### ドキュメントメニューからアップロードする

上部メニュー > **「ドキュメント」** タブに移動し、**「ドキュメントをアップロード」** ボタンをクリックするか、ドラッグ＆ドロップエリアにファイルを投下します。

## ドキュメント処理パイプライン

ドキュメントがアップロードされると、以下のステップがバックグラウンドで自動的に実行されます。

```
1. アップロード   → ファイルがMinIOストレージに保存される
2. パース        → ファイル形式に応じてテキストが抽出される
3. チャンク分割   → 長いドキュメントを適切なサイズに分割
4. エンベディング → 各チャンクをAIが理解できるベクターに変換
5. 保存          → 検索可能な状態でデータベースに保存
6. 完了          → AIチャットですぐに利用可能
```

> 処理時間はファイルサイズによって異なります。1 MB未満のドキュメントは通常10秒以内に完了します。

## ドキュメントでの検索と質問応答

アップロードされたドキュメントの内容についてAIに自然言語で質問できます。

**契約書関連：**
```
ユーザー：契約書の違約条項を見つけて。
AI:   第8条（違約金）によると：甲が納期に間に合わない場合、
      甲は遅延1日につき契約金額の0.1%に相当する違約金を
      乙に支払うものとします。
```

**議事録の要約：**
```
ユーザー：先日の議事録で決定した事項だけをまとめて。
AI:   3月5日のミーティングでの決定事項：
      1. 新機能のローンチ日：4月15日
      2. マーケティング予算：5,000,000円承認
      3. 担当者：田中チームリーダー（開発）、李部長（マーケティング）
```

**Excelデータ分析：**
```
ユーザー：1月の売上データで上位5製品を教えて。
AI:   1月の売上上位5製品：
      1位：製品A — 1,250個（375,000円）
      2位：製品B — 980個（294,000円）
      ...
```

## AIチャットでドキュメントを活用する

### 添付して質問する

チャットで直接ドキュメントを添付すると、そのドキュメントに基づいた会話ができます。

```
ユーザー：[社員マニュアル.pdf を添付] 有給休暇の規定をまとめて。
AI:   マニュアルの第3章（休暇規定）の要約：
      - 有給休暇の付与：1年勤務後15日、以降2年ごとに1日増加（最大25日）
      - 使用期間：付与日から1年以内に使用
      - 申請方法：3日前までに上長の承認が必要
      - 半日休暇：午前（09:00〜13:00）または午後（14:00〜18:00）に分割可能
```

### 複数のドキュメントを比較する

```
ユーザー：2つの見積書を比較して、どちらが有利か教えて。
AI:   ベンダーAとベンダーBの見積書の比較：
      ベンダーA：合計4,500,000円、納期30日、保証期間1年
      ベンダーB：合計4,800,000円、納期25日、保証期間2年
      → ベンダーBは300,000円高いですが、5日早く納品でき、保証期間が2倍です。
```

## AIドキュメント生成

AIに新しいドキュメントやレポートの作成を依頼することもできます。生成されたドキュメントはドキュメントリストに自動的に保存されます。

```
ユーザー：3月の業務報告書を作成して。今月は新機能開発3件とバグ修正12件を完了した。
AI:   3月の業務報告書を作成しました。
      [report.docx を生成]
      ドキュメントタブからダウンロードできます。
```

## ドキュメントの管理

### ドキュメントリストの確認

**「ドキュメント」** タブで、アップロードされたすべてのドキュメントを新着順で確認できます。最大100件が表示されます。

### ドキュメントの削除

リストのドキュメントの横にある削除ボタンをクリックするか、チャットで削除をリクエストします。

```
ユーザー：「2024年度年次報告書.pdf」を削除して
AI:   2024年度年次報告書.pdf を削除しました。
```

## ヒントとよくある質問

**Q. ドキュメントをアップロードしたのに、AIが内容を知らないと言います。**
A. ファイルの処理には少し時間がかかる場合があります。しばらく待ってから再度質問してください。大きなファイルは処理に1〜2分かかることがあります。

**Q. スキャンされたPDFは読み取れますか？**
A. スキャンPDFは画像で構成されているため、テキスト抽出が難しい場合があります。OCR処理済みのPDF、またはテキストが選択できる状態のPDFを使用してください。

**Q. Excelの特定のシートだけについて質問できますか？**
A. はい — 質問にシート名を明記してください。例：「売上シートの3月合計を教えて」

**Q. アップロードしたドキュメントは安全に保管されますか？**
A. すべてのドキュメントは暗号化されたMinIOストレージに保存され、あなただけがアクセスできます。
