# 家計簿 & 予算

## 概要

Starnionの家計簿は、**自然言語で記録するAI搭載の家計管理ツール**です。「ランチにキムチチゲ900円」と入力するだけで、AIが自動的にカテゴリを分類し、日付と金額を解析して即座に記録します。レシートの写真を添付することもできます。

複雑なアプリの使い方を覚える必要はありません。話すように支出を記録し、「今月いくら使いましたか？」と聞くだけで分析結果が得られます。カテゴリ別の予算を設定して上限超過時に警告を受け取り、支出パターン分析から節約のヒントを得ることができます。

## 収支の記録

### 自然言語で入力する

チャット欄に自然に入力するだけです。特別なフォーマットは不要です。

**支出の入力例：**

```
ユーザー: ランチ キムチチゲ 900円
AI:   飲食 / キムチチゲランチ / 900円を記録しました 🍜（3月8日）

ユーザー: カフェ アメリカーノ 450円 カード
AI:   カフェ / アメリカーノ / 450円を記録しました ☕（3月8日 · カード）

ユーザー: バス 130円
AI:   交通 / バス / 130円を記録しました 🚌（3月8日）

ユーザー: ネットで シャンプー 1,280円を購入
AI:   ショッピング / シャンプー / 1,280円を記録しました 🛒（3月8日）

ユーザー: Netflix 月額 1,700円
AI:   サブスク / Netflix / 1,700円を記録しました 📺（3月8日）

ユーザー: クリニック 受診料 500円
AI:   医療 / クリニック受診 / 500円を記録しました 🏥（3月8日）

ユーザー: 昨日 友達と焼き鳥 食べた 3,200円 払った
AI:   飲食 / 焼き鳥 / 3,200円を記録しました 🍗（3月7日）
```

**収入の入力例：**

```
ユーザー: 給料が入った 285,000円
AI:   収入 / 給与 / 285,000円を記録しました 💰（3月8日）

ユーザー: フリーランスの報酬 45,000円を受け取った
AI:   収入 / フリーランス / 45,000円を記録しました（3月8日）

ユーザー: フリマで椅子を売って 3,000円もらった
AI:   収入 / フリマ売上 / 3,000円を記録しました（3月8日）
```

**複数の項目をまとめて入力する：**

```
ユーザー: 今日の支出をまとめて記録して
      - コンビニ 520円
      - ランチ 850円
      - コーヒー 400円
      - 電車 130円

AI:   4件を記録しました ✅
      飲食 / コンビニ / 520円
      飲食 / ランチ / 850円
      カフェ / コーヒー / 400円
      交通 / 電車 / 130円
      ────────────────
      本日の合計支出：1,900円
```

**レシートの写真から入力する：**

```
[レシート画像を添付]
ユーザー: このレシートを家計簿に追加して

AI:   レシートを分析しました 🧾
      - スターバックス トール アメリカーノ: 550円
      - ブルーベリーマフィン: 480円
      合計1,030円をカフェとして記録しますか？

ユーザー: はい
AI:   カフェ / スターバックス / 1,030円を記録しました！（3月8日）
```

### 日付を指定する

日付を含めると、その日付で記録されます。

```
ユーザー: 一昨日 スーパーで食材の買い物をした 6,740円
AI:   ショッピング / スーパー / 6,740円を3月6日に記録しました 🛒

ユーザー: 先週の月曜日に3ヶ月のジムの会費を払った 27,000円
AI:   カルチャー / ジム会費 / 27,000円を3月3日に記録しました 💪

ユーザー: 2月28日に外食 4,500円
AI:   飲食 / 外食 / 4,500円を2月28日に記録しました 🍽️
```

## カテゴリシステム

支出を記録すると、AIが内容を分析して自動的にカテゴリを割り当てます。

### デフォルトカテゴリ

| カテゴリ | 例 |
|----------|-----|
| 飲食 | ランチ、夕食、食材の買い物、コンビニ食品、デリバリー |
| カフェ | カフェ、コーヒーショップ、飲み物、スイーツ |
| 交通 | 電車、バス、タクシー、ガソリン、高速料金 |
| ショッピング | 衣類、電子機器、日用品、ネットショッピング |
| サブスク | Netflix、YouTube Premium、音楽ストリーミング、アプリ購読 |
| 医療 | 病院、薬局、健康診断、医療用品 |
| カルチャー | 映画、コンサート、展覧会、ジム、趣味 |
| その他 | 上記カテゴリに当てはまらない項目 |

### カスタムカテゴリ

デフォルト以外のカテゴリを指定することもできます。

```
ユーザー: ペットフード 3,500円、「ペット」カテゴリで
AI:   ペット / ペットフード / 3,500円を記録しました 🐾

ユーザー: 英会話スクール月謝 15,000円、カテゴリ：教育
AI:   教育 / 英会話スクール / 15,000円を記録しました 📚
```

### 自動カテゴリ分類の仕組み

```
入力：「スターバックス アメリカーノ 550円」
  ↓
AI分析：
  - 「スターバックス」→ カフェブランドと認識
  - 「アメリカーノ」→ 飲み物の種類と認識
  → カテゴリ：カフェ（確信度：高）

入力：「薬局 タイレノール 820円」
  ↓
AI分析：
  - 「薬局」→ 医療関連の場所
  - 「タイレノール」→ 医薬品
  → カテゴリ：医療（確信度：高）

入力：「XX マートで買い物した」
  ↓
AI分析：
  - 「マート」→ ショッピング関連
  - 具体的な商品が不明
  → カテゴリ：ショッピング（確信度：中）
  AI：「何を購入しましたか？より正確にカテゴリ分けできます。」
```

## 予算設定

カテゴリ別の月間支出上限を設定して家計を管理します。

### チャットで予算を設定する

```
ユーザー: 飲食の予算を40,000円に設定して
AI:   飲食の予算を40,000円に設定しました。
      今月の飲食費現在：34,250円（85.6%使用）
      残り予算：5,750円

ユーザー: カフェを10,000円、交通を15,000円に設定して
AI:   予算を設定しました！
      カフェ：        10,000円
      交通：          15,000円

ユーザー: 全体の予算状況を見せて
AI:   🔧 `budget_status` を実行中...

      📋 3月の予算状況
      ──────────────────────────────────
      カテゴリ    予算     使用済   残り    %
      飲食       40,000  34,250   5,750  85.6% ⚠️
      カフェ     10,000  12,700  -2,700 127.0% ❌ 超過！
      交通       15,000   8,940   6,060  59.6% ✅
      ショッピング 20,000  6,740  13,260  33.7% ✅
      ──────────────────────────────────
      合計       85,000  62,630  22,370  73.7%
```

### Web UIで予算を設定する

設定 > 予算管理でカテゴリごとの予算を直接入力できます。

- **警告しきい値**: 支出が予算のこの割合を超えると警告が表示されます（デフォルト：70%）
- **危険しきい値**: 支出が予算のこの割合を超えると危険表示が出ます（デフォルト：90%）

```
予算の例：

カテゴリ       月間予算
──────────────────
飲食           40,000
カフェ         10,000
交通           15,000
ショッピング    20,000
サブスク        7,000
医療            5,000
カルチャー      8,000
その他         10,000
──────────────────
合計          115,000
```

## 予算アラート

予算を超過したり、しきい値に近づいたりすると通知を受け取ります。

### アラートの仕組み

```
[70%警告 — 注意段階]
AIアラート：「飲食の予算を70%使用しました。
            今月の残り予算：12,000円
            残り22日間で1日5,455円まで使えます。」

[90%警告 — 危険段階]
AIアラート：「⚠️ 交通費の予算を92%使用しました！
            残り1,200円のみです。
            今月は交通費を控えるようにしましょう。」

[100%超過]
AIアラート：「❌ カフェの予算を超過しました。
            予算：10,000円 / 使用：12,700円（2,700円超過）
            今月のカフェ予算の見直しを検討してください。」
```

### Telegramアラート連携

Telegramチャンネルを接続すると、外出先でもスマートフォンで予算アラートを受け取れます。外出中もリアルタイムで支出を把握できます。

## データ管理

### 記録の編集

```
ユーザー: 今日入力した交通費130円を145円に変更して
AI:   交通 / バス / 130円 → 145円に更新しました ✏️

ユーザー: 最後の記録のカテゴリを飲食からカフェに変えて
AI:   カフェカテゴリに更新しました ✅
```

Web UIで直接編集する場合：
1. 取引リストで編集したい項目をクリック
2. 金額、カテゴリ、説明、日付を変更
3. 保存ボタンをクリック

```
ユーザー: 今日入力したコンビニ 520円の記録を削除して

      再度入力しましょうか？
```

## AIチャットで家計簿を照会する

自然言語での照会で幅広い検索が可能です。

### 期間別照会

```
ユーザー: 今月の飲食費はいくら？
AI:   3月の飲食費：34,250円（29件）
      予算40,000円の85.6%を使用。

ユーザー: 先月の合計支出は？
AI:   2月の合計支出：79,240円
      収入285,000円に対して貯蓄率72.2%

ユーザー: 今年1月から3月の累計支出を見せて
AI:   2025年1〜3月の累計支出：
      1月：75,820円
      2月：79,240円
      3月：84,730円（進行中）
      合計：239,790円
```

### カテゴリ別照会

```
ユーザー: 過去3ヶ月のサブスク支出はいくら？
AI:   3ヶ月間のサブスク支出：
      1月：  6,200円
      2月：  6,200円
      3月：  6,200円
      合計：18,600円
      有効なサブスク：Netflix、YouTube Premium、Spotify

ユーザー: 今月の支出上位3カテゴリは？
AI:   今月の上位3カテゴリ：
      #1  飲食       34,250  (40.4%)
      #2  カフェ     12,700  (15.0%)
      #3  交通        8,940  (10.6%)
```

### 比較照会

```
ユーザー: 今月は先月より多く使っていますか？
AI:   3月の支出：84,730円（1日〜8日現在）
      2月の支出：79,240円（全体）

      このペースだと3月の合計は
      約317,800円になる見込みです（日平均ベース）。
      先月を超えそうです — 注意が必要です ⚠️

ユーザー: 昨年の同時期と比較して
AI:   2024年3月1〜8日の支出：73,120円
      2025年3月1〜8日の支出：84,730円
      前年比+15.9%の増加。
      主な要因：飲食+22.4%、カフェ+18.7%
```

### 分析照会

```
ユーザー: どの曜日が一番支出が多いですか？
AI:   過去30日の曜日分析：
      金曜日が平均1,730円で最多です。 💸
      主に外食（52%）と交通費（18%）。

ユーザー: 今月の1日当たりの平均支出は？
AI:   3月1〜8日ベース：
      84,730円 ÷ 8日 = 10,591円/日平均
```

### 家計簿をもっと活用するヒント

1. **すぐに記録する**：支出直後に入力することで記録の正確性が保たれ、忘れ防止になります。

2. **レシートの写真を活用する**：大きな買い物や多品目の場合は、レシートを撮影するとAIが複数の項目を一度に入力してくれます。

3. **月初めに予算を設定する**：月の始まりにカテゴリ別の予算を設定すると、支出管理がずっと楽になります。

4. **週次レビューを習慣にする**：週に一度「今週の支出をまとめて」と聞くことで、支出パターンをすぐに把握できます。

5. **AIインサイトを活用する**：定期的に「支出パターンを分析して」と聞くことで、節約できる箇所を発見できます。

## よくある質問

**Q. 金額を間違えて入力しました。どう修正しますか？**

A. 「さっき入力した〔商品名〕〔金額〕を修正して」と言うだけです。Web UIの取引リストで直接クリックして編集することもできます。

```
ユーザー: さっき入力したコンビニの520円を420円に変更して
AI:   コンビニの支出を520円 → 420円に更新しました。
```

**Q. 自動カテゴリ分類が間違っていました。**

A. 「〔元のカテゴリ〕から〔希望のカテゴリ〕に変更して」と言うと即座に更新されます。AIが特定の種類の入力を継続的に誤分類する場合は、フィードバックをお送りください。

**Q. 前の月の支出を入力できますか？**

A. はい、日付を指定するだけです。例えば「2月15日 スーパー 3,200円」と入力するとその日付で記録されます。

**Q. 支出データをExcelに書き出せますか？**

A. CSV書き出し機能は現在開発中です。現時点では「今月の支出リストを送って」とリクエストするとテキスト形式で受け取れます。

**Q. 共有家計（家族、パートナー）を管理できますか？**

A. 現在はアカウント単位の個人利用向けです。共有家計帳機能は今後のアップデートで予定されています。

**Q. 予算を超えた場合、どのようなアラートが届きますか？**

A. 予算の70%（警告）、90%（危険）、100%超過（緊急）でアラートが届きます。Telegramを接続すると、スマートフォンでもアラートを受け取れます。

**Q. 収入はどう記録しますか？**

A. 「給料300,000円が入った」や「収入 300,000円」のように入力すると、自動的に収入として分類されます。プラスの金額は収入、マイナスの金額は支出として扱われます。

**Q. サブスクの定期支払いを自動記録できますか？**

A. デフォルトは手動入力ですが、毎月固定の金額がある場合は「Netflixの1,700円を毎月自動記録して」と言うと毎月自動で記録されます。
