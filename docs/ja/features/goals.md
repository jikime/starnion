# 目標管理

## 概要

Starnionの目標管理機能は、大きな目標から日常の小さな目標まで、体系的に追跡・達成をサポートするAI搭載システムです。単純なタスクリストにとどまらず、AIがあなたの進捗を一緒に追いかけながら励ましの言葉をかけ、状況を分析してくれます。

サポートされる目標タイプ：
- **タスク / プロジェクト** — 期限のある一回限りの項目（レポートの提出、プロジェクトの完了）
- **習慣** — 毎日繰り返す行動（運動、読書、瞑想）
- **貯蓄目標** — 特定の金額を貯める（旅行資金、緊急資金）
- **支出制限** — 支出を予算内に収める（今月の飲食費30,000円以内）
- **一般目標** — 上記カテゴリに当てはまらない個人的な目標

---

## 目標を作成する

### Webで作成する

1. サイドバーの **「目標」** をクリックします。
2. **「+ 新規目標」** ボタンをクリックします。
3. 以下のフィールドを入力します：
   - **目標名**（必須）：達成したいことの短い説明
   - **アイコン**：目標を表す絵文字（デフォルト：🎯）
   - **カテゴリ**：健康、財務、自己成長、仕事など
   - **目標値**：完了を定義する数値（例：30回、100,000円）
   - **単位**：回、日、円、kmなど
   - **開始日 / 終了日**：期間を設定する
   - **説明**：目標に関する追加メモ（任意）

### チャットで作成する

自然に話しかけるだけで、AIが自動的に目標を設定します。確認ステップなしに即座に保存されます。

**会話例：**

> ユーザー：「今月の読書目標を5冊に設定して」
>
> AI：「目標を保存しました！今月5冊読む目標を設定しましたね。応援しています 📚」

> ユーザー：「毎朝運動すると決めた」
>
> AI：「素晴らしい習慣の目標ですね！「毎朝運動する」という目標を保存しました。今日から始めましょう 💪」

> ユーザー：「今月の飲食費を30,000円以内に収めたい」
>
> AI：「財務目標を保存しました！今月の飲食費を30,000円以内に収める目標ですね。一緒に管理しましょう 👍」

**トリガーパターン** — 以下のフレーズが含まれると自動的に目標が作成されます：

| フレーズ例 | 目標タイプ |
|------------|-----------|
| 「Xをすると決めた」「Xを目標に設定して」 | 一般 / 習慣 |
| 「毎日Xする必要がある」 | 習慣 |
| 「今週中にXを終わらせないといけない」 | タスク |
| 「Xを貯める」「X以内に収める」 | 財務 |
| 「Xを貯金すると決めた」 | 貯蓄 |

## チェックインシステム

チェックインは特に習慣タイプの目標において重要です。毎日目標を達成したかどうかを記録して連続ストリークを築きましょう。

### チェックインの方法

**Webの場合：**
1. 目標リストの目標カードを見つけます。
2. カードの下部にある **「今日チェックイン」** ボタンをクリックします。
3. チェックインが確認され、ストリークが更新されます。

**チャットの場合：**

> ユーザー：「今日運動した」
>
> AI：「すごい！今日の運動目標のチェックインが完了しました。現在のストリーク：7日連続 🔥」

> ユーザー：「今日の読書目標をチェックイン」
>
> AI：「チェックイン完了！今月の読書進捗：3/5冊（60%）」

### ストリークカウンター

ストリークとは連続してチェックインした日数です。昨日を逃しても今日チェックインすればストリークは維持されます（1日の猶予期間）。

| ストリーク | ステータス |
|-----------|-----------|
| 0日 | 未開始またはリセット |
| 1〜6日 | スタートしたばかり |
| 7〜29日 | 波に乗っています 🔥 |
| 30日以上 | 習慣が身についた 🏆 |

2日以上連続でチェックインを逃すとストリークはゼロにリセットされます。

## 目標の評価

### 進捗の更新

数値目標（貯蓄、支出など）の場合、進捗を直接更新できます。

> ユーザー：「旅行貯金の目標を50%達成した」
>
> AI：「折り返し地点です！旅行貯金の目標を50%に更新しました。あとひと頑張りですね ✈️」

> ユーザー：「運動目標の進捗が70%になった」
>
> AI：「素晴らしい！運動目標の進捗を70%に更新しました。今月中に達成できそうですね！」

### 目標ステータスの変更

目標は3つのステータスのいずれかになります：

| ステータス | 意味 | 変更方法 |
|----------|------|---------|
| `進行中` | 現在取り組んでいる | デフォルトのステータス |
| `達成` | 目標を完了した | 「達成した」「成功した」「完了」 |
| `中止` | もう追わない | 「やめる」「中止する」「キャンセル」 |

> ユーザー：「読書目標の5冊を全部読み終えた！」
>
> AI：「おめでとうございます！🎉 今月の読書目標を達成しました！5冊すべてを読み終えましたね。素晴らしい！」

## 通知

目標関連の通知は週次レポートで配信されます。終了日が近づくと、AIがチャットで自動的にリマインドします。

**自動通知のトリガー：**
- 終了日の7日前：「目標の締め切りまで7日です」
- 終了日の前日：「目標の締め切りは明日です」
- ストリークの危機：「昨日のチェックインを逃しました — 今日チェックインしてストリークを守りましょう」

---

## 週次レポートとの連携

毎週自動生成されるレポートには目標ステータスのセクションが含まれます：

```
[今週の目標ステータス]
進行中の目標：3件
- 毎日運動（ストリーク：5日）— 今週5/7日チェックイン済み
- 5冊読書（60%完了）— 3/5冊完了、締め切りまで18日
- 飲食費制限（45%完了）— 予算残り13,500円

今週の達成：なし
今週の新規：5冊読書（3/1開始）
```

```
単位：日
開始日：2025-03-01
```


チャット：「今月50,000円貯金した」→ 進捗更新

## 💬 テレグラム使用例

> ニオンに自然な言葉で話しかけるだけでOKです。特別なコマンドは不要 — 普段どおりにチャットしてください！

### 目標を作成する

```
毎日運動する目標を作って
```
→ 習慣トラッキング目標が作成されます

```
今月本を3冊読む目標を設定して
```
→ 数値ベースの目標が作成されます

### 進捗を更新する

```
今日の運動完了！
```
→ 運動目標に今日のチェックインが記録されます

```
「デミアン」読み終わった
```
→ 読書目標の進捗が更新されます

### 目標を確認する

```
目標の進捗状況を教えて
```
→ 現在のアクティブな目標の進捗を表示します

```
達成済みの目標を見せて
```
→ 完了した目標の一覧を表示します

## ヒントとよくある質問

**Q. 目標はいくつまで同時に持てますか？**
進行中の目標は最大20件まで管理できます。達成済みや中止した目標の数に制限はありません。

**Q. ストリークが途切れたら最初からやり直しですか？**
はい、ストリークが途切れるとゼロにリセットされます。ただし、昨日のチェックインを逃しても今日チェックインすれば、翌日のチェックインでストリークが維持されます（1日の猶予期間ポリシー）。

**Q. チャットで目標を作成したとき、数値は自動設定されますか？**
「毎日運動する」のような習慣目標は自動的に単位が「日」に設定されます。「5冊読む」のように特定の数字が言及された場合は、目標値も自動で設定されます。

**Q. 完了した目標を見るにはどうすればいいですか？**
目標画面の上部にあるフィルターから **「完了済み」** を選択すると、達成した目標が表示されます。チャットでは「達成済みの目標を見せて」と言うだけです。

**Q. 財務目標と家計簿が重複しませんか？**
財務目標は目標金額を設定して大きな流れを追うためのものです。家計簿は毎日の支出を詳細に記録するためのものです。2つの機能は補完的な関係にあり、家計簿のデータを財務目標への進捗分析に活用できます。
