# カウントダウン

## 概要

カウントダウン機能は、重要な日付まで何日残っているかをリアルタイムで表示するカウントダウンを提供します。誕生日、記念日、試験、旅行の出発日など覚えておきたい日付を登録しておくと、「試験まで何日？」のような質問にAIがすぐに答えてくれます。

最大30件のカウントダウンを登録でき、将来の日付（D-N）と過去の日付（D+N）の両方に対応しています。

---

## カウントダウンを登録する

### Webで登録する

1. サイドバーの **「カウントダウン」** をクリックします。
2. **「+ 追加」** ボタンをクリックします。
3. 以下の情報を入力します：
   - **名前**（必須）：カウントダウンのタイトル（例：誕生日、TOEIC試験）
   - **日付**（必須）：対象の日付（カレンダーから選択）
   - **アイコン**（任意）：代表する絵文字（デフォルト：📅）
   - **説明**（任意）：追加のメモ
4. **「保存」** をクリックします。

カウントダウンリストは日付の昇順（最も近い日付が先頭）に並びます。

### チャットで登録する

AIに自然な言葉で伝えると自動的に設定されます。

**会話例：**

> ユーザー：「クリスマスのカウントダウンを設定して」
>
> AI：「クリスマスのカウントダウンを設定しました！2025年12月25日まであと D-292日 🎄」

> ユーザー：「5月20日を毎年繰り返す結婚記念日のカウントダウンに設定して」
>
> AI：「結婚記念日のカウントダウンを毎年繰り返しに設定しました！次の記念日（5/20）まで D-73日 💍」

> ユーザー：「6月15日にTOEICの試験がある。カウントダウンを設定して。」
>
> AI：「TOEIC試験のカウントダウンを設定しました！試験（6/15）まで D-99日。頑張ってください！ 📝」

**自然言語の日付表現：**
AIが自然言語の日付表現を自動的に変換します：

| 表現 | 変換後 |
|------|-------|
| 「来週の月曜日」 | 来週月曜日の日付 |
| 「今月末」 | 今月の末日 |
| 「クリスマス」 | 12月25日 |
| 「6月15日」 | YYYY-06-15 |
| 「誕生日 3月10日」 | YYYY-03-10 |

## D-N と D+N

カウントダウンの仕組みを理解すると、機能をより活用できます。

### D-N（残り日数）

将来の日付までの残り日数を表示します。

```
今日：2025年3月8日
対象日：2025年6月15日
表示：D-99
```

### D-Day!（当日）

今日が登録した日付と一致する場合に表示されます。

```
今日：2025年6月15日
対象日：2025年6月15日
表示：D-Day!
```

### D+N（経過日数）

過去の日付からの経過日数を表示します。記念日など過去の日付を登録した場合に便利です。

```
今日：2025年3月8日
対象日：2024年3月8日
表示：D+365
```

### 使用例

| 用途 | 日付の例 | 表示例 |
|------|---------|--------|
| 試験カウントダウン | 2025-06-15 | D-99 |
| 結婚記念日（今日） | 2025-03-08 | D-Day! |
| パートナーと出会った日 | 2024-01-01 | D+431 |
| 旅行出発日 | 2025-04-01 | D-24 |

## 通知

カウントダウンの通知は現在、チャットで能動的に確認するか、週次レポートを確認することで行います。

### 週次レポートへの組み込み

毎週自動生成されるレポートには、近日のカウントダウン情報が含まれます：

```
[近日のカウントダウン]
- D-7: TOEIC試験（6/15）
- D-30: 友達の誕生日（4/7）
- D-93: 沖縄旅行（6/9）
```

### チャットで確認する

いつでもAIに聞いて現在のカウントダウン状況を確認できます：

> ユーザー：「次のカウントダウンはいつ？」
>
> AI：「最も近いカウントダウンは **TOEIC試験** です。6月15日まで D-7日。最後の追い込み、頑張ってください！」

> ユーザー：「今月のカウントダウンはある？」
>
> AI：「今月のカウントダウンは2件あります：\n- 3/10（D-2）：両親の結婚記念日\n- 3/20（D-12）：プロジェクトの締め切り」

## さまざまなカウントダウンの使用例

### 誕生日

```
名前：お母さんの誕生日
日付：2025-04-15
アイコン：🎂
説明：プレゼントの準備を忘れずに！
```

> ユーザー：「お母さんの誕生日はいつ？」
> AI：「お母さんの誕生日は4月15日です。あと D-38日。事前にプレゼントを用意しましょう！ 🎂」

### 記念日

```
名前：出会った日
日付：2023-11-25
アイコン：💕
```

> ユーザー：「付き合って何日になった？」
> AI：「2023年11月25日から今日まで D+468日。1年以上一緒にいますね！ 💕」

### 試験

```
名前：IELTS試験
日付：2025-05-10
アイコン：📝
説明：目標スコア：7.0
```

> ユーザー：「試験までどのくらい？」
> AI：「IELTS試験まで D-63日。5月10日です。準備の具合はどうですか？ 📝」

### 旅行

```
名前：日本旅行出発
日付：2025-05-01
アイコン：✈️
説明：大阪、4泊5日
```

> ユーザー：「旅行まであと何日？」
> AI：「日本旅行出発まで D-54日。楽しみですね！そろそろ荷物の準備を始めましょう ✈️」

### プロジェクトの締め切り

```
名前：Q2レポート提出期限
日付：2025-03-31
アイコン：📊
説明：チームリーダーへの提出
```

## AIでの照会

チャットからさまざまな方法でカウントダウンを確認できます。

```
ユーザー：「次の記念日はいつ？」
AI：「最も近く登録されている記念日は **結婚記念日** です。5月20日まで D-73日 💍」

ユーザー：「試験まで何日？」
AI：「TOEIC試験まで D-99日。6月15日です。毎日少しずつ準備すれば十分です！」

ユーザー：「カウントダウンの全リストを見せて」
AI：「登録されているカウントダウン：
  1. D-7: TOEIC試験（6/15）📝
  2. D-30: 友達の誕生日（4/7）🎂
  3. D-54: 日本旅行（5/1）✈️
  4. D-73: 結婚記念日（5/20）💍
  5. D+5: 会社の設立記念日（3/3）🏢」

ユーザー：「過去のカウントダウンも見せて」
AI：「過去のカウントダウンを含む全リスト：
  [今後]
  - D-7: TOEIC試験
  ...
  [過去]
  - D+5: 会社の設立記念日（3/3）」

ユーザー：「クリスマスのカウントダウンを削除して」
AI：「クリスマスのカウントダウンを削除しました。」
```

## よくある質問

**Q. カウントダウンは何件まで登録できますか？**
最大30件です。30件に達したら、古いカウントダウンを削除してから新しいものを追加してください。

**Q. 毎年繰り返す記念日はどう設定しますか？**
チャットで「毎年繰り返しに設定して」と伝えるか、Webの登録画面で **「繰り返し」** オプションを有効にしてください。設定した日付が過ぎると、自動的に翌年の日付に更新されます。

**Q. カウントダウンは今日の日付に基づいて自動的に再計算されますか？**
はい、カウントダウンの値は毎日自動的に再計算されます。昨日 D-10だったものは、今日は D-9と表示されます。

**Q. カウントダウンを編集できますか？**
Webでカウントダウンをクリックすると編集画面が開き、日付、名前、アイコン、説明を変更できます。

**Q. 過去の日付を登録するとどう表示されますか？**
過去の日付は D+N 形式で表示されます。パートナーと出会った日や会社の開業日など、経過した時間を記念したい場合に便利です。

**Q. 週次レポートにカウントダウンが自動的に含まれますか？**
はい、D-30以内の近日のカウントダウンは週次レポート生成時に自動的に含まれます。週が始まる前に重要な近日の日程を確認できます。
