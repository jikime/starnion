# AI メモリ

## 概要

AI メモリは、Nion が会話を通じて自動的に学習したパターンやインサイトを可視化する機能です。ユーザーが手動で入力するメモとは異なり、AI が会話のコンテキストを分析して自動生成する長期記憶システムです。

**主な機能：**
- 会話パターンから自動抽出されたインサイト
- 時系列での感情・行動パターン分析
- 週次の会話サマリーとトレンド追跡
- 保存されたメモリエントリの直接削除

> AI メモリはメモとは異なります。メモはユーザーが明示的に保存する情報ですが、AI メモリは Nion が会話を分析して自動生成する長期学習データです。

---

## 3つのタブ

### 会話インサイト

AI が最近の会話を分析して抽出したインサイトの一覧です。

**表示情報：**
- **日付リスト**：分析が完了した日付（左サイドバーまたはドロップダウン）
- **インサイトカード**：各分析エントリの要約内容
- **気分インジケーター**：その時点での全体的な感情状態
- **トピックタグ**：会話で頻繁に登場したキーワード

**インサイトの種類：**
| 種類 | 説明 |
|------|------|
| 会話パターン | 話し方、質問のパターンなど |
| 興味のあるトピック | 繰り返し言及される関心事 |
| 感情の流れ | 会話中の感情の変化 |
| 行動指標 | ルーティン、習慣、よくする行動 |

### 行動パターン

会話データをもとに分析された行動・嗜好のパターンです。

**表示情報：**
- **パターンカード**：各行動パターンのタイトルと説明
- **カテゴリバッジ**：パターンの分類（生活習慣、健康、財務、人間関係など）
- **信頼度インジケーター**：AI 分析の信頼度レベル

**パターンカテゴリ：**
| カテゴリ | 例 |
|----------|-----|
| 生活習慣 | 睡眠パターン、食事時間など |
| 感情パターン | ストレス要因、プラス要因 |
| 財務行動 | 支出傾向、貯蓄パターン |
| 社会関係 | 会話頻度、関係の種類 |

### 週次メモリ

毎週自動生成される会話サマリーと主要イベントの記録です。

**表示情報：**
- **週次サマリー**：その週の主な会話内容とアクティビティ
- **主要イベント**：特別に記録された重要な瞬間
- **削除機能**：特定の週次メモリを選択して削除可能

## AI メモリが生成される条件

AI メモリのデータは自動生成されますが、特定の条件が満たされた場合にのみ分析が実行されます。

### 会話インサイトの生成条件

- テレグラムまたはチャットで **30分〜2時間** 会話がない場合に自動分析が実行される
- **日記スキル**が有効になっていること
- 分析サイクル：約10分ごとに条件を確認

### 行動パターンの生成条件

- **毎日午前6時**に前日までのデータを分析
- 意味のあるパターン生成には **7日以上**の会話データが必要
- 財務データがある場合、支出パターンも合わせて分析

### 週次メモリの生成条件

- **毎週月曜日の午前5時**に前週のデータを自動整理
- 週次圧縮には **30日以上**の会話履歴が必要

> **データが表示されない場合：** 上記の条件がまだ満たされていない可能性があります。テレグラムやチャットで自然に会話を続けると、AI が徐々にパターンを学習し始めます。

## データの削除

保存された AI メモリエントリは直接削除できます。

### 削除方法

**週次メモリの削除：**
1. **週次メモリ**タブをクリックします。
2. 削除したい週のエントリの**削除**ボタンをクリックします。
3. 確認ポップアップで**削除**ボタンをクリックします。

**会話インサイト／行動パターンの削除：**
1. 該当タブで削除したい項目の**削除**アイコンをクリックします。
2. 即座に削除されます。

> 削除されたメモリは復元できません。慎重に削除してください。

## ヒントとよくある質問

**Q. AI メモリとメモの違いは何ですか？**

| 項目 | AI メモリ | メモ |
|------|-----------|------|
| 作成方法 | AI が自動生成 | ユーザーが手動入力 |
| データの出所 | 会話分析結果 | ユーザー入力内容 |
| 用途 | 長期パターン学習 | 情報の保存と取得 |
| 編集 | 不可（削除のみ） | Web で編集可能 |

**Q. AI メモリにデータが何もないのはなぜですか？**
AI メモリは十分な会話データが蓄積された後に生成されます。テレグラムやチャットで定期的に会話をすると、2〜7日後からデータが生成され始めます。

**Q. 日記スキルがないと会話インサイトは生成されませんか？**
はい、会話インサイトは日記スキルが有効になっているユーザーにのみ生成されます。設定 > スキルから日記スキルを有効にしてください。

**Q. メモリを削除すると AI が再学習しますか？**
削除されたメモリは復元できず、そのデータに基づくパターンは表示されなくなります。ただし、新しい会話が蓄積されると AI が再び学習してメモリを生成します。

**Q. AI メモリのデータは安全ですか？**
すべての AI メモリデータはユーザーアカウントに紐づけられて保存され、他のユーザーとは共有されません。いつでも直接削除できます。
