# マインドガーデン

## 概要

マインドガーデンは、Starnionの感情ウェルネスダッシュボードです。日記に記録された感情を可視化し、AIコンパニオンのNionを通じてコンテキストに応じたメッセージを届けます。単なる統計画面ではなく、心の健康を育む空間として設計されています。

> 日記を書くほど、感情レイヤーが満たされ、ヒーリングツリーが育ちます。

---

## レイアウト

マインドガーデンは3列で構成されています。

```
┌─────────────────┬──────────────────┬─────────────────┐
│  ヒーリングツリー │   Nionセンター    │ カウンセリング    │
│  感情チャート     │  （今日の気分）    │  スペース        │
│  ムードカレンダー │                  │  （統計）        │
│                 │                  │  （マインド花）   │
│                 │                  │  （進行中の目標） │
└─────────────────┴──────────────────┴─────────────────┘
```

---

## 左列：ヒーリングツリー & ムードカレンダー

### ヒーリングツリー

日記のエントリーが蓄積されるにつれてツリーが成長します。

| 要素 | 説明 |
|------|------|
| **枝** | 5つの感情色でエネルギーが流れる |
| **ハートの実** | 感情タイプごとに1つ、最大5つ、鼓動アニメーション付き |
| **記憶の葉** | 日記エントリー10件ごとに1枚、最大8枚 |
| **中央のオーブ** | 生命力を表すゴールドの球体 |

### 感情レイヤーチャート

日記の感情分布を横棒グラフで表示します。

- 感情ごとの記録回数を表示
- 上位5つの感情を表示
- 下部のカウンター：**「合計N件の記憶」**

### 月間ムードカレンダー

各日にその日の感情を色分けしたドットで表示する月間カレンダーです。

| UI要素 | 説明 |
|--------|------|
| **< / >** ボタン | 前月 / 翌月に移動（未来には移動不可） |
| **色付きドット** | その日の日記の気分の色で表示 |
| **今日のハイライト** | ゴールドの枠線 + ゴールドのフォント |
| **曜日バッジ** | 日（ピンク）/ 土（青）/ 平日（白）の円形バッジ |
| **凡例** | Joy・Peace・Calm・Sad・Downの色参照 |

## 中央列：Nionセンター

### Nionキャラクター

AIコンパニオンのNionが、現在の時刻に合わせた状態で表示されます。

| 時間 | Nionの状態 |
|------|-----------|
| 深夜（0時） | ヒーリング |
| 1時間ごと | 植物の世話 → 見守り → 磨き → 掃除 |

ハート、星、ドットがNionの周りを周回し、連続的にアニメーションします。

### 今日の気分

- 最新の日記エントリーの感情が自動的に表示されます。
- 下部の**5つの気分選択ボタン**で、今日の気分を直接設定することもできます。
- 選択した気分に基づいてNionのヒーリングメッセージが更新されます。

### ヒーリングメッセージ

各気分に応じた癒しのメッセージが表示されます。

| 気分 | メッセージ例 |
|------|------------|
| Joy（とても良い） | 「輝かしい一日ですね！その喜びを大切に」 |
| Peace（良い） | 「心が穏やかですね。この瞬間を味わいましょう」 |
| Calm（普通） | 「平凡な日も大切な記録です。よくやっています」 |
| Sad（悪い） | 「そう感じても大丈夫。Nionがそばにいます」 |
| Down（とても悪い） | 「疲れましたね。ゆっくり、一歩ずつ進みましょう」 |

## 右列：カウンセリングスペース

### セッション統計カード

| 項目 | 説明 |
|------|------|
| **ドーナツチャート** | 目標達成率（%） |
| **今週の記録** | 過去7日間の日記エントリー数 |

### マインドフラワー

マインドフルネス活動への3つのクイックアクセスボタン。

| ボタン | 遷移先 |
|--------|--------|
| 瞑想 | 日記ページ |
| 日記 | 日記ページ |
| チャット | チャットページ |

### 進行中の目標

現在 `in_progress` ステータスの目標を最大3件表示します。

- アイコン + タイトル + プログレスバー + パーセンテージ
- パープルからピンクのグラデーションプログレスバー

## ムードシステム

マインドガーデンは5つの基本感情値を使用します。

| キー | ラベル | 色 | 絵文字 |
|------|--------|-----|--------|
| `매우좋음` | Joy | ゴールド `#fbbf24` | ✨ |
| `좋음` | Peace | グリーン `#34d399` | 🌿 |
| `보통` | Calm | パープル `#a78bfa` | 💜 |
| `나쁨` | Sad | ブルー `#60a5fa` | 💙 |
| `매우나쁨` | Down | インディゴ `#818cf8` | 🌙 |

AIがチャットを通じて自由形式の感情（「ワクワク」「疲れた」「最悪」など）を保存する際、自動的にこれら5つの値のいずれかに正規化されます。

## データフロー

```
チャット / 日記エントリー作成
    ↓
diary_entries テーブルに保存（ムード正規化済み）
    ↓
/api/diary?limit=100  <── マインドガーデン読み込み時に呼び出し
/api/goals
    ↓
感情カウント集計    → 感情レイヤーチャート
日付→ムードマッピング → 月間ムードカレンダー
最新のムード        → Nionヒーリングメッセージ
進行中の目標        → カウンセリングスペース目標セクション
```

Nionとのチャットへのショートカットが画面下部に常に表示されています。

> *「Nionに気持ちを共有しましょう...」*

クリックするとチャットページに移動します。

## FAQ

**Q. ムードカレンダーにドットが表示されません。**

ドットは、ムードが5つの基本値（`매우좋음 / 좋음 / 보통 / 나쁨 / 매우나쁨`）のいずれかであるエントリーにのみ表示されます。v1.3.3より前にチャットで保存された既存のエントリーは、そのマイグレーションで自動的に正規化されます。

**Q. Nionの画像が変わり続けます。**

Nionの状態は現在の時刻に基づいて1時間ごとにローテーションします。これは期待される動作です。

**Q. 気分ボタンをクリックしましたが、保存されませんでした。**

気分選択ボタンは、**現在の画面に表示されている気分のみ**を変更します。気分を永続的に保存するには、チャットまたは日記メニューから日記エントリーを書いてください。

**Q. 目標が表示されません。**

目標メニューで目標を追加し、ステータスを「進行中」に設定すると、カウンセリングスペースに表示されます。
