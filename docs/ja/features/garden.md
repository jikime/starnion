# データガーデン

## 概要

データガーデンは、**Starnionのすべてのデータを生きた庭園のエコシステムとして可視化する**ダッシュボードです。家計、目標、日記の感情、ファイル、スキルがそれぞれ成長する庭園の要素として表現されます。

> 貯蓄率が高いほどツリーは健康に。使いすぎると雲から雨が降り始めます。

## 空の背景 -- ムードパレット

空の色は、過去7日間の日記エントリーに記録された感情に基づいて変化します。

| 気分 | 空の色 | 雰囲気 |
|------|--------|--------|
| Joy（とても良い） | ゴールデングラデーション | 暖かいゴールデンアワーの輝き |
| Peace（良い） | ブルーグラデーション | 澄んだ穏やかな空 |
| Calm（普通） | パープルグラデーション | 神秘的なコスミックヘイズ |
| Sad（悪い） | ディープパープル | 暗い夜空 |
| Down（とても悪い） | ディープスペースブラック | 真っ暗な虚空 |

## 中央：アセットツリー

庭園の中心。ツリーの健康状態は貯蓄率を反映しています。

### 健康段階

| 貯蓄率 | ツリーの状態 | 枝の色 | 果実 |
|--------|------------|--------|------|
| 30%以上 | 健康 | フルカラー | 最大5つのスター果実 |
| 15-29% | 枯れ始め | 彩度低下 + 垂れ下がり | 果実が少ない |
| 15%未満 | 危険 | グレーブラウン + 重度の垂れ下がり | 果実なし |

### ペルソナオーラリング

ツリーの根元に、アクティブなAIペルソナに合わせた色のオーラリングが表示されます。

| ペルソナ | オーラの色 |
|---------|-----------|
| assistant | シルバーホワイト |
| finance | グリーン |
| buddy | ピンク |
| coach | オレンジ |
| analyst | ブルー |
| counselor | パープル |

### 予算ポップアップ（ツリーをクリック）

アセットツリーをクリックすると、詳細な予算ポップアップが開きます。

| 表示情報 | 説明 |
|---------|------|
| **残り予算** | 今月の残り予算（緑/オレンジ/赤でカラーコード） |
| **予算コンステレーション** | カテゴリーが円形に星として配置されたSVG |
| **総予算 / 支出** | 数値とプログレスバー |
| **支出率 / 貯蓄率** | パーセンテージ表示 |
| **ステータスメッセージ** | 予算の健全性に基づくコンテキストメッセージ |

---

## 支出クラウド（右上）

いずれかの予算カテゴリーが超過すると、雲から雨が降ります。

- **雨なし**：すべてのカテゴリーが予算内
- **雨**：超過カテゴリーに対してアニメーション付きの雨粒が雲から落ちる
- **ハサミのアニメーション**：超過時にツリーにハサミが表示される

クラウドをクリックすると、超過カテゴリーとその統計を一覧表示するポップオーバーが表示されます。

---

## ゴールフラワー園（左下）

進行中の目標がチューリップとして可視化されます。最大4本の花が表示されます。

### 成長段階

| 段階 | 進捗 | 外見 |
|------|------|------|
| 種 | 0-10% | 単一の輝くドット |
| 芽 | 10-30% | 茎 + つぼみ |
| 成長中 | 30-55% | 茎 + 2枚の葉 |
| 開花中 | 55-80% | 3枚の花弁が開く |
| 満開 | 80-100% | 完全に開花 + 揺れるアニメーション |

フラワー園をクリックすると、すべての目標とその達成率をポップオーバーで確認できます。

---

## エモーションシード（左下）

過去7日間の日記エントリーの感情が小さな種として表示されます。

- 最大7個の種
- 各種はムード値の色で着色
- 各種にブルームアニメーション

## Phase 2 要素

| 要素 | 位置 | データソース | 説明 |
|------|------|-------------|------|
| **風鈴** | 左上 | 音声ファイル数 | アップロードされた音声ファイルの数 |
| **知識のリュック** | 中央左 | ドキュメント数 | アップロードされたドキュメントの数 |
| **マジックレイク** | 下部中央 | 画像数 | アップロードされた画像の数、波紋アニメーション |
| **マジックワンド** | 右下 | 有効スキル数 | 有効なスキルの数、宝石アニメーション |
| **エナジーチャージャー** | 右下 | 月間トークン使用量 | 1000万トークン月間制限に対する使用率% |
| **スカウトスター** | 上部中央 | Web検索ステータス | 常に表示（静的要素） |
| **ギャラクシーパス** | 下部ストライプ | インテグレーション数 | APIキーが設定されたプロバイダーの数 |

各要素はクリックすると詳細な統計ポップオーバーが開きます。

## 特殊エフェクト

### 収入スターダスト

収入が記録されると、ゴールドの星が空から降ります。

- 星の数：`収入 / 200,000`、最大12個
- スターダストの落下アニメーション
- 収入額に比例して浮遊する収入スターも空に表示

### 流星

D-Dayが7日以内の場合、流星が画面を斜めに横切ります。

- 最大3つの流星
- 35度の角度、4.5秒サイクル
- 近づくD-Dayの数に応じて数がスケール

## データフロー

データガーデンの読み込み時に、12のAPIコールが並列で行われます。

| APIエンドポイント | 目的 |
|-----------------|------|
| `/api/budget` | 予算カテゴリーと支出 |
| `/api/goals` | 目標の進捗 |
| `/api/diary?limit=7` | 過去7日間の感情 |
| `/api/finance/summary` | 収入と貯蓄率 |
| `/api/dday` | D-Dayリスト |
| `/api/images?limit=50` | 画像ファイル数 |
| `/api/audios?limit=50` | 音声ファイル数 |
| `/api/documents` | ドキュメントファイル数 |
| `/api/skills` | 有効スキル数 |
| `/api/usage?days=30` | 月間トークン使用量 |
| `/api/profile/persona` | 現在のペルソナ |
| `/api/settings/providers` | 接続サービス数 |

## FAQ

**Q. ツリーが灰色になりました。**

貯蓄率が15%を下回るとツリーは枯れます。支出を減らすか収入を増やすと健康が回復します。

**Q. 雲から雨が降り続けています。**

1つ以上の予算カテゴリーが超過している場合に発生します。クラウドをクリックしてどのカテゴリーが超過しているか確認し、予算を調整してください。

**Q. 花が4本しか表示されません。**

ゴールフラワー園は進行中の目標を最大4件表示します。園をクリックするか目標メニューに移動して、すべての目標を確認してください。

**Q. エナジーチャージャーが赤くなっています。**

月間トークン使用量が1000万トークンの制限の80%を超えると、チャージャーが赤くなります。チャットの頻度を減らすと充電レベルが回復します。

**Q. 空がとても暗いです。**

過去7日間の日記の感情がSadやDownに偏っている場合、空が暗くなります。日記に気持ちを書いたり、Nionとチャットしてみてください。

**Q. 流星はいつ出ますか？**

D-Dayが7日以内の場合に流星が表示されます。D-Dayメニューで今後のイベントを登録すると表示されます。
