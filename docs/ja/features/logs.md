# ログ

## 概要

ログ機能は、Starnionシステム内のすべての活動記録を閲覧・分析できるモニタリングツールです。ニオンとの会話履歴、スキル実行の詳細、システムエラーなどを一箇所で確認でき、トラブルシューティングや利用パターンの把握に役立ちます。

**主な特徴：**
- 会話ログ：すべてのチャット履歴の検索・閲覧
- スキル実行ログ：どのスキルがいつ実行されたかの追跡
- エラーログ：システムエラー発生時の詳細情報の確認
- 多様なフィルター：日付、ログレベル、スキル種別、チャネル別のフィルタリング
- エクスポート：CSV/JSON形式でのログダウンロード

---

## ログの種類

### 会話ログ

ニオンとやり取りしたすべてのメッセージ記録です。

| 項目 | 説明 |
|------|------|
| タイムスタンプ | メッセージ送信の正確な時刻 |
| 送信者 | ユーザーまたはAI |
| チャネル | Telegram、WebChatなど |
| 内容 | メッセージ本文 |
| 会話ID | 会話スレッドの識別子 |

### スキル実行ログ

AIが内部的に呼び出したスキル（機能）の実行記録です。

```
[2024-03-15 14:23:45] SKILL  finance.add_expense  SUCCESS  "豚バラ 3,200円を記録"
[2024-03-15 14:25:12] SKILL  diary.create         SUCCESS  "3/15の日記を保存"
[2024-03-15 14:30:01] SKILL  websearch.search     SUCCESS  "トヨタ株価を検索"
[2024-03-15 14:35:22] SKILL  memo.create          FAILED   "保存失敗：メモ上限超過"
```

### エラーログ

システムエラー発生時に記録される詳細ログです。

| レベル | 説明 | 例 |
|--------|------|-----|
| ERROR | 機能実行失敗 | スキル呼び出しエラー、APIタイムアウト |
| WARN | 警告（機能は正常） | 応答遅延、リトライ発生 |
| INFO | 一般情報 | スキル実行完了、ユーザーログイン |
| DEBUG | デバッグ情報 | 詳細実行過程（開発者向け） |

## ログの閲覧

### Webから閲覧

1. サイドバーの**ログ**メニューをクリックします。
2. 上部のフィルターを設定します：
   - **日付範囲**：開始日〜終了日
   - **ログレベル**：ERROR、WARN、INFO、DEBUG
   - **スキル種別**：特定スキルのみフィルタリング
   - **チャネル**：Telegram、WebChatなど
3. 結果が時系列順に表示されます。

### フィルター活用例

**エラーのみ表示：**
- ログレベルをERRORに設定すると、失敗した操作だけを素早く確認できます。

**特定スキルの追跡：**
- スキルフィルターで`finance`を選択すると、家計簿関連のログのみ表示されます。

**特定期間の照会：**
- 「過去7日間」「今月」などのクイック日付選択で簡単に範囲を指定できます。

## ログのエクスポート

ログデータをファイルとしてダウンロードし、外部で分析できます。

### 対応形式

| 形式 | 特徴 | 用途 |
|------|------|------|
| CSV | Excelで直接開ける | スプレッドシート分析 |
| JSON | 構造化データ | プログラミング分析、外部ツール連携 |

### エクスポート方法

1. 希望するフィルターを適用します。
2. 右上の**エクスポート**ボタンをクリックします。
3. 形式（CSVまたはJSON）を選択します。
4. ファイルが自動的にダウンロードされます。

## 活用例

### トラブルシューティング

AIが特定のリクエストに対して予期しない応答をしたり、機能が動作しなかった場合、ログを確認すると原因が分かります。

```
問題：「家計簿に記録して」と言ったのに保存されなかった

ログ確認：
[14:35:22] SKILL  finance.add_expense  FAILED  "カテゴリ解析失敗：入力値不足"

原因：金額を指定しなかったため解析に失敗
解決：「昼食 1,200円記録して」のように金額を含めて再リクエスト
```

### 利用パターン分析

どのスキルをどの程度の頻度で使用しているかを把握し、活用度を高められます。

### セキュリティ監査

アカウントへの不正アクセスや、予期しない時間帯の活動がなかったか確認できます。

## ヒント & FAQ

**Q. ログはどのくらい保存されますか？**
すべてのログは永続的に保存されます。非常に古いログを検索する場合は、日付範囲を絞ると結果がより速く表示されます。

**Q. ログに機密情報が表示されますか？**
会話内容はログに含まれますが、APIキーやパスワードなどのシステム認証情報はマスキング処理されています。

**Q. 個別のログを削除できますか？**
現在、個別ログの削除機能は提供していません。データの整合性と監査証跡のためにログは保存されます。

**Q. リアルタイムでログを監視できますか？**
Webログ画面で自動更新を有効にすると、最新のログをほぼリアルタイムで確認できます。
