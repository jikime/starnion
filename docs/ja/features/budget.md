# 予算管理

## 概要

予算管理機能は、**月別・カテゴリ別の予算を設定し、リアルタイムで支出をモニタリング**する機能です。予算に対する実際の支出をプログレスバーで視覚化し、一目で消費状況を把握できます。予算の80%を超えると警告通知で使いすぎを防ぎます。

> 予算を設定すると家計簿の記録と自動的に連動します。ニオンに「今月の食費はあとどれくらい？」と聞いてみてください。

---

## 予算の設定

### ウェブで設定

1. サイドバーの**予算管理**メニューをクリックします。
2. **+ 予算追加**ボタンをクリックします。
3. 以下の情報を入力します：
   - **カテゴリ**（必須）：食費、交通費、ショッピング、娯楽、医療、教育など
   - **月間予算額**（必須）：そのカテゴリの月別予算
4. **保存**ボタンをクリックします。

総予算は各カテゴリ予算の合計として自動計算されます。

### チャットで設定

AIに自然言語で話しかけると予算を設定できます。

**会話例：**

> ユーザー：「今月の食費予算を5万円に設定して」
>
> AI：「食費予算を50,000円に設定しました！現在の食費支出は21,500円で、予算の43%を使用しています。」

## 予算状況の確認

### ダッシュボード画面

予算管理ページでは、各カテゴリについて以下の情報を確認できます：

| 表示項目 | 説明 |
|---------|------|
| **カテゴリ名** | 予算が設定されたカテゴリ |
| **プログレスバー** | 予算に対する支出比率の視覚化 |
| **支出額 / 予算** | 現在の支出 / 設定予算（例：21,500 / 50,000円） |
| **使用率（%）** | 予算に対する支出のパーセンテージ |
| **ステータスアイコン** | 安全（緑）、注意（オレンジ）、警告（赤） |

### プログレスバーの色

| 使用率 | 色 | 意味 |
|--------|-----|------|
| 0–60% | 緑 | 安全圏 |
| 60–80% | オレンジ | 注意圏 |
| 80–100% | 赤 | 警告圏 |
| 100%+ | 濃い赤 | 超過 |

---

## 予算超過警告

予算の80%以上を使用すると自動的に警告が表示されます。

### 警告方法

- **ウェブダッシュボード**：該当カテゴリのプログレスバーが赤色に変更
- **チャット通知**：家計簿記録時にAIが予算超過の可能性を案内
- **週間レポート**：週間レポートに予算状況を含む

## 予算の修正と削除

### チャットで修正

> ユーザー：「食費予算を6万円に上げて」
>
> AI：「食費予算を60,000円に修正しました。使用率が36%に下がりました。」

### 削除

> ユーザー：「交通費の予算を削除して」
>
> AI：「交通費予算を削除しました。総予算は50,000円に変更されました。」

## FAQ

**Q. 予算は毎月自動的にリセットされますか？**

はい、予算は毎月1日に自動リセットされます。支出額のみリセットされ、設定した予算額は維持されます。

**Q. カテゴリ別ではなく総予算だけ設定できますか？**

現在はカテゴリ別予算設定のみ対応しています。「その他」カテゴリに総予算を設定する方法で活用できます。

**Q. 予算を超過すると支出記録がブロックされますか？**

いいえ、予算超過時も支出記録は正常に保存されます。超過警告のみ表示され、記録自体は制限しません。

**Q. 家計簿データと自動連動しますか？**

はい、家計簿に記録された支出は該当カテゴリ予算に自動的に反映されます。別途の連動設定は不要です。
