# ペルソナ

## 概要

ペルソナ機能は、ニオンの名前、性格、役割をカスタマイズできるAIプロフィールシステムです。さまざまなペルソナを作成し、状況に応じて切り替えることで、異なる性格のAIと会話しているように活用できます。

**主な特徴：**
- ペルソナカスタマイズ：名前、役割、性格説明、挨拶メッセージの設定
- 複数ペルソナ：いくつでもペルソナを作成し、状況に応じて切り替え可能
- デフォルトペルソナ：Nion（ニオン）— 内蔵のデフォルトペルソナ
- チャット中の切り替え：会話中にペルソナを変更可能

---

## ペルソナとは？

ペルソナは、**AIの性格と役割を定義するプロフィール**です。同じAIでも、ペルソナによって異なる応答スタイルで会話します。

| 構成要素 | 説明 | 例 |
|---------|------|-----|
| **名前** | ペルソナの表示名 | 「ニオン」「ファイナンスコーチ」「英語の先生」 |
| **役割** | 専門分野 | 「パーソナルファイナンスアドバイザー」 |
| **性格説明** | 会話スタイルと態度 | 「親切で節約を重視し…」 |
| **挨拶** | 会話開始時に送るメッセージ | 「こんにちは！今日の家計を一緒に見てみましょう。」 |

---

## デフォルトペルソナ

Starnionには**Nion（ニオン）**というデフォルトペルソナが内蔵されています。

- **名前**：Nion
- **役割**：パーソナルAIアシスタント
- **性格**：フレンドリーで温かい会話スタイル、絵文字を活用、ユーザーの日常に関心を持つ
- **特徴**：日記、家計簿、メモ、目標など、すべての機能をバランスよくサポート

## ペルソナの作成

### Webから作成

1. **設定 > ペルソナ** メニューに移動します。
2. **＋ 新しいペルソナ** ボタンをクリックします。
3. 以下の項目を入力します：
   - **名前**（必須）：ペルソナを呼ぶ名前
   - **役割**（任意）：このペルソナの専門分野
   - **性格説明**（必須）：AIが参考にする会話スタイルと態度の説明
   - **挨拶**（任意）：会話開始時に自動送信されるメッセージ
4. **保存**をクリックします。

### ペルソナの例

活用できるペルソナのアイデアを紹介します。

#### ファイナンスコーチ

```
名前：ファイナンスコーチ
役割：パーソナルファイナンスアドバイザー
性格説明：節約と投資に集中し、不必要な支出を見つけるとやんわりと指摘する。
         励ましを惜しまず、小さな節約も褒める。財務目標の達成を応援する。
挨拶：こんにちは！今日の家計状況を一緒にチェックしましょう。
```

**会話例：**
```
ユーザー：今日カフェで700円使ったよ
AI：      コーヒー一杯ですね！今月のカフェ支出は合計4,500円になりました。
          マイタンブラーを持っていくと割引になるお店もありますよ。
```

#### 日記の友達

```
名前：ゆうこ
役割：日記ヘルパー
性格説明：感性的で共感力が豊か。ユーザーの感情に深く共感し、
         一日を振り返ることを手伝う。温かい言葉遣いを使う。
挨拶：今日はどんな一日だった？聞かせて。
```

**会話例：**
```
ユーザー：今日プレゼンを失敗しちゃった…
AI：      そうだったんだ…準備頑張ったのに悔しかったよね。
          どの部分が心残りだった？話してくれたら、次にもっとうまくいく方法を一緒に考えるよ。
```

#### 英語の先生

```
名前：Teacher Emma
役割：英会話教師
性格説明：英語でのみ会話し、文法の間違いを優しく訂正する。
         ユーザーのレベルに合わせて会話の難易度を調整し、新しい表現を自然に教える。
挨拶：Hi! Ready for some English practice today?
```

**会話例：**
```
ユーザー：I goed to the store yesterday
AI：      Great effort! Just a small correction:
          "goed" → "went" (go-went-gone is an irregular verb)
          "I went to the store yesterday." Perfect otherwise!
          What did you buy there?
```

## ペルソナの切り替え

### Webから切り替え

1. **設定 > ペルソナ** で、有効にしたいペルソナの横にある**有効化**ボタンをクリックします。
2. またはチャット画面上部のペルソナ名をクリックして変更します。

### チャットで切り替え

会話中に自然言語でペルソナの切り替えをリクエストできます。

```
ユーザー：英語の先生モードに変えて
AI：      Hi! I'm Teacher Emma. Let's practice English together!

ユーザー：ニオンに戻して
AI：      戻ってきたよ！何をお手伝いしましょうか？
```

## ヒント & FAQ

**Q. ペルソナは最大いくつまで作れますか？**
制限なく自由に作成できます。ただし、有効にできるペルソナは一度に1つです。

**Q. ペルソナを切り替えると以前の会話の文脈は消えますか？**
いいえ。ペルソナを切り替えても会話履歴とメモリは維持されます。AIの口調と応答スタイルのみが新しいペルソナに合わせて変更されます。

**Q. デフォルトのNionペルソナを編集できますか？**
デフォルトペルソナの性格説明と挨拶は変更可能です。名前は変更できません。

**Q. ペルソナごとに異なるモデルを使用できますか？**
現在、ペルソナとモデルは別々に設定されます。モデル設定は**設定 > モデル**メニューで管理します。

**Q. 良いペルソナを作るコツはありますか？**
性格説明を具体的に書くほど、AIはより一貫したキャラクターを維持します。「優しい」だけでなく、「常に励ましを惜しまず、小さな成果も褒め、温かい言葉遣いをする」のように詳しく書きましょう。
