# 画像

## 概要

Starnionの画像機能では、写真をアップロードしてAIによる分析を行ったり、レシートを自動認識して家計簿に記録したり、テキストの説明だけから新しい画像を生成したりすることができます。

チャットで画像を添付すると、「このレシートを家計簿に記録して」「この写真に何が写っていますか？」「このグラフを分析して」のような自然言語コマンドを使えます。アップロードされた画像はすべてギャラリーにまとめられ、AI分析の結果も一緒に保存されます。

## 画像をアップロードする

### チャットで画像を添付する

チャット入力欄のファイル添付ボタンをクリックするか、チャットウィンドウに画像をドラッグ＆ドロップしてアップロードします。

```
ユーザー：[画像を添付] これは何ですか？
AI:   この写真は伝統的な日本家屋（古民家）です。瓦屋根と木の柱が特徴で、
      美しい曲線を描く軒が印象的です。
      伝統的な建築様式と背景の山の景色との調和が
      特に際立っています。
```

### 画像ギャラリーからアップロードする

上部メニュー > **「画像」** タブに移動し、アップロードボタンをクリックするか、ドラッグ＆ドロップで画像を追加します。

## 画像分析

AIは画像に何が写っているかを説明し、様々な質問に答えることができます。

**写真の内容を分析する：**
```
ユーザー：[レストランのメニュー写真を添付] ここで何を注文すればいい？
AI:   メニューを見ると、このレストランは和食が中心のようです。おすすめ：
      - 味噌汁定食 800円：定番の安心感
      - 豚の生姜焼き 900円：人気メニューのようです
      - ビビンバ 850円：ヘルシーな選択肢
      ランチセット1,100円もお得です。
```

**グラフや図の分析：**
```
ユーザー：[売上グラフの画像を添付] このグラフを分析して。
AI:   この棒グラフは月次売上のトレンドを示しています。
      - 1〜3月：上昇傾向（平均230,000円）
      - 4月：急落（180,000円、前月比-22%）
      - 5〜6月：回復傾向
      4月の落ち込みの原因について調査が必要なようです。
```

**ドキュメント/テキスト抽出（OCR）：**
```
ユーザー：[名刺の写真を添付] 連絡先情報を抽出して。
AI:   名刺から情報を抽出しました：
      名前：田中 健二
      役職：マーケティングチームリーダー
      会社：Starnion株式会社
      メール：kenji@starnion.com
      電話：090-1234-5678
```

## レシートOCR — 家計簿との自動連携

レシートの写真をアップロードすると、AIが自動的に金額と品目を認識し、家計簿に記録します。

```
ユーザー：[コンビニのレシート写真を添付] 家計簿に保存して。
AI:   レシートを認識しました：
      - おにぎり（ツナマヨ）: 130円
      - アメリカーノ: 150円
      - ペットボトル飲料: 180円
      合計：460円（2025-03-08 14:32）

      460円を飲食費として記録しますか？

ユーザー：はい、そうです。
AI:   コンビニ / 460円を記録しました！本日の飲食費合計：1,280円
```

**複数レシートの一括処理：**
```
ユーザー：[複数のレシート写真を添付] 今週のレシートをすべて記録して。
AI:   3件のレシートを認識しました：
      1. スーパーのレシート — 4,520円（3月5日）
      2. カフェのレシート — 850円（3月6日）
      3. レストランのレシート — 3,200円（3月7日）
      合計8,570円を家計簿に記録しました。
```

## 画像ギャラリー

**「画像」** タブで、これまで使用したすべての画像を一覧で確認できます。

### ギャラリーフィルター

| フィルター | 説明 |
|----------|------|
| すべて | すべての画像 |
| 生成 | AIが生成した画像 |
| 編集 | AIが編集した画像 |
| 分析 | アップロードしてAIが分析した画像 |
| Webチャット | Web経由で添付した画像 |
| Telegram | Telegram経由で送信した画像 |

### ギャラリーでできること

- 画像をクリックして全画面で表示
- AI分析の結果テキストを確認
- 生成に使用したプロンプトを確認
- 画像を削除する

## AI画像生成

テキストの説明（プロンプト）から画像を生成できます。生成された画像は自動的にギャラリーに保存されます。

```
ユーザー：隅田川から見た東京の夜景を描いて。
AI:   画像を生成中...
      [画像生成完了]
      隅田川から見た東京の夜景の画像を作成しました。
      ギャラリーにも保存されています。
```

**異なるスタイルを指定する：**
```
ユーザー：桜並木の風景を水彩画風に描いて。
AI:   [水彩画風の桜並木の画像を生成]

ユーザー：かわいい猫のキャラクターのステッカーを作って。
AI:   [猫のステッカー画像を生成]
```

## 画像の編集

AIを使って既存の画像を修正できます。編集された画像はギャラリーに `edited_` というプレフィックスを付けて保存されます。

```
ユーザー：[写真を添付] この写真の背景をぼかして（ボケエフェクト）。
AI:   背景をぼかした画像を作成しました。
      [編集済み画像]

ユーザー：[写真を添付] この写真から人物を消して。
AI:   人物を消して背景を復元しました。
      [編集済み画像]
```

## ヒントとよくある質問

**Q. レシートの認識がうまくいきません。**
A. 明るい環境で正面から撮影し、レシート全体がフレーム内に収まるようにしてください。しわくちゃのレシートを伸ばしてから撮影すると、認識精度が大幅に向上します。

**Q. 複数の画像を一度にアップロードできますか？**
A. はい、チャットで複数の画像を一度に添付できます。ただし、AI分析がすべてを一度に処理するため、非常に多くの画像を添付すると応答が遅くなる場合があります。

**Q. 生成した画像をダウンロードできますか？**
A. ギャラリーで画像をクリックして全画面表示し、右クリック（またはスマートフォンでは長押し）して保存してください。

**Q. 個人情報が含まれる画像は安全ですか？**
A. アップロードした画像は暗号化されて保存され、あなただけがアクセスできます。ただし、機密情報が含まれる画像のアップロードには注意が必要です。
