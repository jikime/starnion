# 消費分析

## 概要

消費分析機能は、家計簿に記録された支出データを**カテゴリ別・期間別に視覚化**し、消費パターンを把握できる分析ダッシュボードです。円グラフやラインチャートなど様々なチャートで支出トレンドを一目で確認でき、AIが節約のインサイトを提供します。

> お金の流れを知ることが節約の第一歩です。ニオンに「今月の消費パターンを分析して」と聞いてみてください。

---

## 主要チャート

### カテゴリ別支出比率（円グラフ）

全体の支出に対する各カテゴリの比率を円グラフで表示します。

| 項目 | 説明 |
|------|------|
| **カテゴリ色** | 食費（緑）、交通費（青）、ショッピング（ピンク）、娯楽（オレンジ）など |
| **比率表示** | 各カテゴリのパーセンテージ（%） |
| **金額表示** | ホバー時に実際の金額を表示 |

### 月別支出トレンド（ラインチャート）

直近6ヶ月間の月別総支出をラインチャートで表示します。

- X軸：月（例：10月、11月、12月、1月、2月、3月）
- Y軸：支出金額
- 前月比の増減方向を表示

### 支出カテゴリ TOP 5

今月最も支出が多かったカテゴリ5つをランキング形式で表示します。

| 順位 | 表示情報 |
|------|---------|
| 1位 | カテゴリ名 + 金額 + 前月比変化率 |
| 2位 | カテゴリ名 + 金額 + 前月比変化率 |
| ... | ... |
| 5位 | カテゴリ名 + 金額 + 前月比変化率 |

### 前月比変化率

今月と先月の支出を比較し、変化率を表示します。

| 表示 | 意味 |
|------|------|
| **↑ 15%**（赤） | 先月比15%増加 |
| **↓ 8%**（緑） | 先月比8%減少 |
| **— 0%**（グレー） | 変化なし |

---

## 期間フィルター

上部フィルターで分析期間を変更できます。

| 期間 | 説明 |
|------|------|
| **今週** | 直近7日間 |
| **今月** | 当月1日から今日まで |
| **先月** | 前月全体 |
| **直近3ヶ月** | 直近3ヶ月累計 |
| **直近6ヶ月** | 直近6ヶ月累計 |
| **今年** | 1月1日から今日まで |

## AI分析インサイト

ニオンが消費データを分析し、パーソナライズされたインサイトを提供します。

### インサイトの例

```
今月の消費分析結果：

1. 総支出：152,000円（前月比 ↑12%）
2. 最多支出：食費 48,000円（31.6%）
3. 増加項目：ショッピング +45%（セールシーズンの影響と推定）
4. 減少項目：交通費 -20%（在宅勤務の影響と推定）
5. 節約提案：週3回の自炊に切り替えると月約8,000円節約可能
```

## FAQ

**Q. 分析に使用されるデータはどこから来ますか？**

家計簿に記録されたすべての収入/支出データが分析に使用されます。記録が多いほど、より正確な分析が可能です。

**Q. 収入も分析に含まれますか？**

消費分析は支出データのみを分析します。収入データは家計簿の収入/支出サマリーで確認できます。

**Q. チャートデータをダウンロードできますか？**

現在、チャート画像やデータの直接ダウンロードには対応していません。チャットでニオンにテキスト形式の分析結果を依頼し、コピーして活用できます。

**Q. 特定期間の分析だけを見ることはできますか？**

はい、上部フィルターで期間を選択すると、その期間の分析結果のみ表示されます。チャットでも「3月の消費を分析して」のように特定期間を指定できます。
