# Naver検索連携

StarnionをNaver検索APIに接続すると、AIエージェントが韓国語コンテンツに特化した検索を実行できるようになります。ブログ、ニュース、カフェ、ショッピング、知識iNなどNaverの多様な検索サービスを自然言語で活用できます。

---

## 概要

Naver検索連携を使用すると：

- **ブログ検索**: Naverブログの投稿を検索
- **ニュース検索**: 韓国語ニュース記事を検索
- **カフェ検索**: Naverカフェ（コミュニティフォーラム）の投稿を検索
- **ショッピング検索**: 商品情報を検索
- **知識iN検索**: 質問と回答を検索

> **オプトイン機能：** Naver検索連携はデフォルトで無効になっています。Naverデベロッパーセンターでアプリを作成し、APIキーを登録した後、スキルを有効化する必要があります。

---

|-----------|------|

| ショッピング | 商品名、価格などのショッピング情報検索 |

## 事前準備：Naverデベロッパーセンターでアプリ作成

### ステップ1：アプリの登録

1. [Naverデベロッパーセンター](https://developers.naver.com/)にアクセスします。
2. **Application** → **アプリケーション登録** をクリック。
3. **アプリケーション名** を入力します（例：`Starnion`）。
4. **使用API** で **検索** を選択します。
5. 適切な環境を追加します。
6. **登録** をクリック。

### ステップ2：認証情報の確認

1. 登録完了後、**マイアプリケーション** 一覧から作成したアプリをクリックします。
2. **Client ID** と **Client Secret** をコピーします。

> **無料利用：** Naver検索APIは1日25,000回まで無料で呼び出せます。

---

## 設定方法

### Web UIでAPIキーを登録

1. Starnion Web UIにログインします。
2. 左メニュー → **Settings** → **Integrations** タブをクリック。
3. **Naver検索** セクションで以下のフィールドを入力します：
   - **Client ID**: NaverデベロッパーセンターのClient ID
   - **Client Secret**: NaverデベロッパーセンターのClient Secret
4. **保存** ボタンをクリック。
5. **Naver検索スキル有効化** トグルをオンにします。

## 韓国語コンテンツに特化している理由

Naver検索APIは韓国語コンテンツに最適化されています：

- **韓国語形態素解析**: 韓国語の検索キーワードを正確に理解
- **国内コンテンツ**: 韓国のウェブサイト、ブログ、ニュースに特化
- **ローカル情報**: 韓国のショッピング、地域情報、国内コミュニティ検索
- **カフェ/知識iN**: グローバル検索エンジンでは見つけにくい韓国コミュニティ情報

> **ヒント：** 韓国語検索にはNaver検索、グローバル検索にはTavilyを併用すると、より豊富な情報を得られます。

## 接続解除方法

1. Settings → Integrations → Naver検索 セクション。
2. **接続解除** ボタンをクリック。
3. 保存されたClient IDとClient Secretが即座に削除されます。

---

## トラブルシューティング

### 「Naver検索連携が設定されていません」

Settings → Integrations → Naver検索でClient IDとClient Secretが登録されているか確認してください。

### 「Naver API認証に失敗しました」（401エラー）

- Client IDとClient Secretが正しいか確認します。
- [Naverデベロッパーセンター](https://developers.naver.com/)でアプリがアクティブか確認します。

### 「1日の呼び出し上限を超えました」（429エラー）

- 1日のAPI呼び出し上限（25,000回）を超えています。
- 翌日に自動的にリセットされます。

## FAQ

**Q: Naver検索とTavily検索を同時に使えますか？**
A: はい、両方を同時に有効化できます。ニノンが検索内容に応じて適切なサービスを自動選択します。

**Q: 英語の検索もNaverでできますか？**
A: 可能ですが、英語コンテンツの検索にはTavilyの使用をお勧めします。Naverは韓国語コンテンツに最適化されています。

**Q: Naver検索APIは無料ですか？**
A: 1日25,000回まで無料です。一般的な個人利用には十分な量です。
