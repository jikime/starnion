# Tavily検索連携

StarnionをTavilyに接続すると、AIエージェントがリアルタイムのウェブ検索を実行して最新情報を提供できるようになります。ニノンが最新情報を必要とする際に自動的にTavily検索を実行します。

---

## 概要

Tavily連携を使用すると：

- **リアルタイムウェブ検索**: 最新ニュース、情報、データを検索
- **ニュース検索**: 最近のニュース記事を検索
- **学術検索**: 学術論文や研究資料を検索

> **オプトイン機能：** Tavily連携はデフォルトで無効になっています。APIキーを登録し、スキルを有効化する必要があります。

---

## 対応機能一覧

| 機能 | 説明 |
|------|------|
| ウェブ検索 | リアルタイムウェブ検索で最新情報を提供 |
| ニュース検索 | 最近のニュース記事検索 |
| 学術検索 | 学術論文や研究資料の検索 |

---

## 事前準備：Tavily APIキーの発行

### ステップ1：APIキーの作成

1. [tavily.com](https://tavily.com)にアクセスします。
2. アカウントを作成またはログインします。
3. ダッシュボードで **API Keys** セクションを見つけます。
4. **Create API Key** ボタンをクリック。
5. 生成されたAPIキーをコピーします（`tvly-...` 形式）。

> **無料枠：** Tavilyは月1,000回の検索まで無料で提供しています。

---

## 設定方法

### Web UIでAPIキーを登録

1. Starnion Web UIにログインします。
2. 左メニュー → **Settings** → **Integrations** タブをクリック。
3. **Tavily** セクションの **API Key** 入力フィールドを見つけます。
4. コピーしたAPIキー（`tvly-...`）を貼り付けます。
5. **保存** ボタンをクリック。
6. **Tavilyスキル有効化** トグルをオンにします。

## 自動検索動作

ニノンは以下のような場合に自動的にTavily検索を実行します：

- 最新ニュースやリアルタイム情報がリクエストされた場合
- リアルタイムデータ（株価、為替、天気など）が質問された場合
- AI学習データに含まれない最近の情報が必要な場合

> **ヒント：**「検索して」「調べて」などの表現を使うと、Tavily検索がトリガーされやすくなります。

---

## 接続解除方法

1. Settings → Integrations → Tavily セクション。
2. **接続解除** ボタンをクリック。
3. 保存されたAPIキーが即座に削除されます。

---

## トラブルシューティング

### 「Tavily連携が設定されていません」

Settings → Integrations → Tavily でAPIキーが登録されているか確認してください。

### 「Tavily APIキーが無効です」（401エラー）

- APIキーが正しいか確認します。
- [tavily.com](https://tavily.com)のダッシュボードでキーがアクティブか確認します。

### 「検索上限を超えました」（429エラー）

- 無料枠の月間検索上限（1,000回）を超えた可能性があります。
- 有料プランにアップグレードするか、翌月までお待ちください。

## FAQ

**Q: Tavily検索結果は常に正確ですか？**
A: ウェブ検索結果に基づいているため、常に出典を確認することをお勧めします。ニノンは可能な限り出典を表示します。

**Q: Tavilyなしでも最新情報を得られますか？**
A: Tavily連携なしでは、AI学習データに基づいた回答のみとなります。リアルタイム情報にはTavily連携をお勧めします。

**Q: 無料枠で十分ですか？**
A: 個人利用であれば月1,000回の検索で十分です。チーム利用の場合は有料プランをご検討ください。
