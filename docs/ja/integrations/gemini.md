# Gemini連携

Google Gemini APIをStarnionに接続すると、会話でGeminiモデルを使用できるようになります。画像分析やマルチモーダル処理など、Geminiの強力な機能を活用しましょう。

---

## 概要

Gemini連携を使用すると：

- **対話モデル**: Gemini 2.0 Flash、Gemini 1.5 Pro、Gemini 1.5 Flashが利用可能
- **エンベディング**: gemini-embedding-001（768次元）をサポート
- **画像分析**: 画像をアップロードして内容の分析・説明
- **マルチモーダル処理**: テキストと画像を組み合わせた複合処理

> **デフォルトプロバイダー：** GeminiはStarnionのデフォルトプロバイダーです。追加のAPIキー設定なしで無料枠で始められます。

---

## 対応モデル

| モデル | 特徴 | 推奨用途 |
|--------|------|----------|
| Gemini 2.0 Flash | 高速応答、最新モデル | 日常会話、素早いQ&A |
| Gemini 1.5 Pro | 長いコンテキスト（最大1Mトークン） | 長文ドキュメント分析、複雑な推論 |
| Gemini 1.5 Flash | バランスの取れた性能 | 汎用会話 |
| gemini-embedding-001 | 768次元エンベディング | ドキュメントベクトル化、類似検索 |

## 事前準備：Google AI Studio APIキーの発行

### ステップ1：APIキーの作成

1. [Google AI Studio](https://aistudio.google.com/)にアクセスします。
2. Googleアカウントでログインします。
3. **Get API key** ボタンをクリックします。
4. **Create API key** → プロジェクトを選択または新規作成。
5. 生成されたキーをコピーします（`AIza...` 形式）。

> **無料枠：** 1分あたり15リクエスト、1日1,500リクエスト（2025年時点）。

---

## 設定方法

### Web UIでAPIキーを登録

1. Starnion Web UIにログインします。
2. 左メニュー → **Settings** → **Models** タブをクリック。
3. **Google Gemini** プロバイダーを選択します。
4. **API Key** フィールドにコピーしたキー（`AIza...`）を貼り付けます。
5. 使用するモデルをチェックします（複数選択可能）。
6. **保存** ボタンをクリック。

APIキーは保存時に自動検証されます。

## 使用方法

### 会話でモデルを選択

会話開始時にモデル選択ドロップダウンからGeminiモデルを選択するか、ペルソナにGeminiモデルを紐付けて自動適用できます。

### 画像分析

```
自分：（画像添付）この写真に何が写っている？
ボット：この画像には富士山の美しい景色が写っています。
       雪をかぶった山頂が青空に映えていますね。

自分：（グラフ画像添付）このチャートを分析して
ボット：この棒グラフは月別売上推移を示しています。
       3月から売上が着実に増加し、6月に最高を記録しています。
```

### 長文ドキュメント分析

```
自分：（PDF添付）このレポートを要約して
ボット：（Gemini 1.5 Pro使用）
       このレポートの主要ポイント：
       1. 2026年第1四半期の売上は前年比15%増
       2. 新サービスのリリースによりMAUが30%成長
       ...
```

## エンベディング設定

Geminiのエンベディングモデルは、ドキュメント検索や類似度分析などに使用されます。

- **モデル**: gemini-embedding-001
- **次元**: 768
- **用途**: 会話履歴のベクトル化、ドキュメント検索、パターン分析

エンベディング設定は `starnion.yaml` または環境変数で構成できます。

## トラブルシューティング

### 「Gemini APIキーが無効です」

- APIキーが `AIza...` 形式か確認します。
- [Google AI Studio](https://aistudio.google.com/)でキーがアクティブか確認します。

### 「リクエスト上限を超えました」（429エラー）

- 無料枠（1分15回、1日1,500回）を超えています。
- しばらく待つか、他のプロバイダーに切り替えてください。

### 画像分析が動作しない

- Gemini 2.0 FlashまたはGemini 1.5 Proモデルを使用しているか確認します。
- 画像形式がサポートされているか確認します（JPEG、PNG、GIF、WebP）。

## FAQ

**Q: Geminiをずっと無料で使えますか？**
A: はい、Google AI Studioの無料枠内であれば無料で使い続けられます。上限を超えると有料になる場合があります。

**Q: Geminiと他のプロバイダーを同時に設定できますか？**
A: はい、複数のプロバイダーのAPIキーを同時に登録し、会話ごとにモデルを選択できます。

**Q: Geminiエンベディングの利用に別途設定は必要ですか？**
A: Gemini APIキーが登録されていれば、エンベディングも自動的に利用可能です。`starnion.yaml`で詳細設定を調整できます。
