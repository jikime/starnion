# スキルリスト

Starnionには現在**34**の組み込みスキルがあります。各スキルは独立したモジュールで、必要に応じて個別に有効/無効を切り替えることができます。

---

## 財務管理

| スキルID | スキル名 | 説明 | コマンド例 | デフォルト有効 |
|---------|---------|------|-----------|-------------|
| `finance` | 費用管理 | 収入と支出を自動的に記録し、月次合計を取得します。 | `昼食1,200円`、`今月いくら使った？` | ✓ |
| `budget` | 予算管理 | カテゴリー別に月次予算を設定して使用状況を確認します。 | `食費の予算を30,000円に設定`、`残りの予算はいくら？` | ✓ |
| `pattern` | 支出パターン分析 | 曜日別の支出傾向、定期的な請求、異常な支出をバックグラウンドで自動的に分析します。 | `私の支出パターンは？`、`異常な支出はある？` | ✓ |
| `proactive` | プロアクティブ通知 | 予算超過、異常な支出などのAI主導のアラート。 | （自動的に送信されるため、直接の呼び出しは不要） | ✓ |
| `currency` | 為替レート | リアルタイムの為替レートを検索し、通貨を換算します。 | `ドルの為替レートを教えて`、`100ドルは何円？` | ✓ |

## スキルごとの必要な権限

一部のスキルは外部APIキーまたはOAuth認証が必要です。

| スキル | 必要な設定 | 設定場所 |
|-------|-----------|---------|
| `weather` | 天気APIキー | サーバー `.env` |
| `websearch` | Tavily APIキー | **設定 > 連携** |
| `naver_search` | Naver クライアントID / シークレット | **設定 > 連携** |
| `google` | Google OAuth2認証 | **設定 > 連携** |
| `image` | Gemini APIキー（画像生成用） | **設定 > モデル** |
| `audio` | Gemini APIキー（TTS用） | **設定 > モデル** |
| `currency` | 為替レートAPI（Frankfurter、無料） | 自動 |
| `browser` | Playwrightのインストールが必要 | サーバー環境 |

## 自然言語によるスキルの呼び出し

スキル名を直接指定する必要はありません。自然に話しかけるだけで、AIが自動的に適切なスキルを選択します。

```
私: 50,000ウォンはドルでいくらですか？
Bot: 現在の為替レート（1,340 KRW/USD）で、50,000ウォン ≈ 37.31ドルです。

私: ロンドンは今何時ですか？
Bot: ロンドン（GMT）の現在時刻は午前6時42分です。

私: 今日は本当にひどい気分です。上司に怒られました。
Bot: それは辛い一日でしたね。日記に記録しました。

私: 明日の午後3時に医者の予約をリマインドしてください。
Bot: 「医者の予約」のリマインダーを明日の午後3時に設定しました。

私: 1〜45から6つの宝くじの数字を抽選して。
Bot: 🎰 結果：7、14、23、31、38、42

私: この画像には何がありますか？（写真を添付）
Bot: コーヒーショップのレシートのようです — アメリカーノが450円です。費用管理に記録しますか？

私: https://news.ycombinator.com を要約してください。
Bot: そのページを要約しました。現在のトップ投稿：...
```

## 複数スキルの組み合わせ

1つのメッセージで複数のスキルを順番に起動できます。

| 例 | 起動されるスキル |
|---|--------------|
| レシートの写真を送信 | `image` → 金額を抽出 → `finance` → 自動記録 |
| 音声メッセージ「昼食1,000円」 | `audio` → テキスト変換 → `finance` → 費用管理に記録 |
| 「異常な支出パターンはある？」 | `pattern` の分析結果 → `memory` 検索 → 応答 |
| 「このレポートをPDFで作成して」 | LLMがコンテンツを書く → `documents` → PDFファイルを提供 |
