# Telegram Bot

StarnionをTelegramで利用すると、スマートフォンからどこでもAIアシスタントにアクセスできます。Web UIとTelegramを同時に使用でき、会話履歴は単一アカウントに統合されます。

---

## 概要

Telegramチャンネルでは以下のことができます：

- スマートフォンのTelegramアプリでStarnionとチャットする
- 画像、音声メッセージ、ドキュメントファイルを送信する
- グループチャットでボットをメンションして使用する
- Web UIと同じスキルとメモリを使用する

---

## ステップ1：Telegramボットを作成する

TelegramボットはBotFatherを通じて作成します。

1. Telegramで[@BotFather](https://t.me/BotFather)を検索してチャットを開きます。
2. `/start` を送信します。
3. `/newbot` を送信します。
4. ボットの名前を入力します（例：`My Starnion`）。
5. ボットのユーザー名を入力します。`bot` で終わる必要があります（例：`my_starnion_bot`）。
6. BotFatherが**トークン**を発行します。

```
BotFather: Done! Congratulations on your new bot. You will find it at t.me/my_starnion_bot.
Use this token to access the HTTP API:
1234567890:ABCDefGHIjklMNOpqrsTUVwxyz1234567890
```

発行されたトークンをコピーしてください。次のステップで必要になります。

> **セキュリティに関する注意：** トークンはパスワードのようなものです。公開しないでください。

---

## ステップ2：ボットをStarnionに登録する

1. Starnion Web UIにログインします。
2. 左メニューの**設定**をクリックします。
3. **チャンネル**タブを選択します。
4. **Telegram**セクションで、コピーしたトークンを**Botトークン**フィールドに貼り付けます。
5. **保存**をクリックします。
6. **ボットを有効にする**トグルをオンにします。

正常に保存されると、ステータスが**running**と表示されます。

> **注意：** 1つのボットトークンは1つのアカウントにのみ登録できます。すでに別のアカウントに登録されているトークンを入力するとエラーが発生します。

---

## ステップ3：TelegramアカウントをリンクするAI（ペアリング）

ボットを登録するだけでは、TelegramとStarnionアカウントはリンクされません。「このTelegramアカウントが自分のStarnionアカウント」であることを関連付けるために**ペアリング**を完了する必要があります。

### ペアリングの方法

1. Telegramで自分のボット（`@my_starnion_bot`）を検索してチャットを開きます。
2. 何かメッセージを送信します（例：`こんにちは`）。
3. ボットの**DMポリシー**が `pairing` に設定されている場合、ボットが「ペアリングリクエストを送信しました」と応答します。
4. Starnion Web UI > **設定 > チャンネル > Telegram** に移動して、**ペアリングリクエスト**リストを確認します。
5. Telegramアカウント名を確認して**承認**をクリックします。
6. これでTelegramでボットと通常にチャットできます。

---

## ペアリングポリシー

チャンネル設定で**DMポリシー**と**グループポリシー**を設定できます。

### DMポリシー（ダイレクトメッセージ）

| ポリシー | 動作 |
|---------|------|
| `allow` | ボットにDMを送った誰でも即座に応答を受け取ります。 |
| `pairing` | ペアリング済みのアカウントのみ応答を受け取ります。未ペアリングのユーザーはペアリングリクエストを送信できます。 |
| `deny` | すべてのDMが無視されます。 |

### グループポリシー（グループチャット）

| ポリシー | 動作 |
|---------|------|
| `allow` | グループ内のすべてのメッセージに応答します。 |
| `mention` | ボットがメンションされた場合（`@bot_name`）のみ応答します。 |
| `deny` | すべてのグループメッセージを無視します。 |

> **推奨設定：** 個人利用の場合、DMポリシーを `pairing`、グループポリシーを `mention` に設定すると、セキュリティを維持しながら便利に使用できます。

## ボットの使い方

### 基本的な会話

ペアリングが完了したら、通常のメッセージアプリと同じようにチャットします。

```
私: 今日の天気はどうですか？
Bot: 東京の現在の天気：晴れ、22℃。...

私: 昼食に1,200円使いました
Bot: 昼食の費用1,200円を記録しました。今月の食費合計：8,750円

私: Pythonのチュートリアルを探してください
Bot: インターネットでPythonチュートリアルを検索しました...
```

### 画像の送信

送信した画像は自動的に分析されます。

```
（レシートの写真を送信）
Bot: レシートを分析しました。
    スーパー合計：3,560円
    内容：牛乳280円、パン450円、...
    これを費用管理に自動記録しますか？
```

### 音声メッセージ

Telegramの音声メッセージ（ボイスメモ）を送信すると、テキストに変換されて処理されます。

```
（音声メッセージ：「今夜友人と夕食でサムギョプサルを食べました、1人2,000円」）
Bot: 音声メッセージを認識しました。
    夕食（サムギョプサル）の費用2,000円を記録しました。
```

### ドキュメントファイル

PDF、Word、テキストファイルなどを送信すると、ボットがコンテンツを分析したり、ナレッジベースに保存したりします。

```
（PDFファイルを送信）
Bot: ドキュメント「contract_2024.pdf」を処理しました。
    全15ページ、主な内容：賃貸契約...
    検索用に保存しました。
```

## 複数デバイスの同時使用

Starnionは、Web UIとTelegramを同時に使用することをサポートしています。両方のチャンネルは同じAIエージェントに接続されているため、一方のチャンネルに記録されたものは他方から取得できます。

```
[Web UIで]
私: 読書目標：今月3冊読む

[Telegramで]
私: 読書目標は何でしたっけ？
Bot: 目標は今月3冊読むことです。現在1冊完了しています。
```

---

## グループチャットでの使用

ボットをグループに招待してメンションすることで使用できます。

1. グループチャットで**メンバーを追加**を選択し、ボットのユーザー名を検索して招待します。
2. グループポリシーが `mention` の場合、`@bot_name メッセージ` の形式でメンションします。

```
私: @my_starnion_bot チームランチの合計が15,000円でした — 5人でいくらずつになりますか？
Bot: 1人あたり3,000円です。

私: @my_starnion_bot ニューヨークの現在時刻は？
Bot: ニューヨーク（EST）の現在時刻は午前2時15分です。
```

## 重要な注意事項

- **1トークン = 1アカウント：** 1つのボットトークンは1つのStarnionアカウントにのみリンクできます。
- **ボットトークンのセキュリティ：** トークンが漏洩した場合、BotFatherの `/revoke` コマンドで即座に再発行し、Starnion設定で新しいトークンに更新してください。
- **サーバー再起動：** Starnionゲートウェイが再起動すると、有効なボットは自動的に再開されます。

## トラブルシューティング

### ボットが応答しない

1. 設定 > チャンネル > Telegram でステータスが **running** と表示されていることを確認します。
2. ステータスが **configured** の場合、ボットは無効になっています。**ボットを有効にする**トグルをオンにしてください。
3. **ステータスがrunningでも**応答がない場合：
   - DMポリシーが `deny` に設定されていないか確認します。
   - ペアリングが必要な場合は、ペアリング手順を完了してください。

### 409 Conflict エラー

同じボットトークンで2つのサーバーが同時に動作しているときに発生します。

- DockerとローカルAI開発環境を同時に実行しないでください。
- 新しいサーバーを起動する前に、以前のサーバーを完全にシャットダウンしてください。
- Starnionは競合を防ぐためにPostgreSQLアドバイザリロックを使用しています。

### ペアリングが機能しない

- ボットがアクティブな状態であることを確認します。
- DMポリシーが `deny` の場合、ペアリングリクエストは完全にブロックされます。`pairing` または `allow` に変更してください。
- Web UIのペアリングリクエストリストに表示されたリクエストを承認したことを確認します。

## よくある質問

**Q. TelegramボットなしでStarnionを使用できますか？**
はい、Web UIだけですべての機能を使用できます。Telegramはオプションの追加チャンネルです。

**Q. 複数人が同じTelegramボットを使用できますか？**
いいえ。各ユーザーは独自の別のボットを作成する必要があります。ただし、グループチャットでは複数のペアリング済みユーザーが1つのボットをメンションして共有できます。

**Q. ボットトークンを変更するとどうなりますか？**
既存のボットは自動的に停止し、新しいトークンで再起動します。以前にペアリングされたアカウント情報は保持されます。

**Q. 会話はTelegramのサーバーに保存されますか？**
TelegramメッセージはTelegramのサーバーを経由しますが、Starnionでの会話処理と保存はセルフホスト型の自分のサーバー上で行われます。
