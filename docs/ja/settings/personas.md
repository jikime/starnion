# ペルソナ

## 概要

ペルソナ機能は、Starnion の AI の名前・性格・役割をカスタマイズできる AI プロファイルシステムです。複数のペルソナを作成して状況に応じて切り替えることで、まるで異なる性格の AI と会話しているように活用できます。

**主な機能:**
- ペルソナのカスタマイズ: 名前・役割・性格説明・挨拶の設定
- 複数ペルソナ: 複数作成して状況に応じて切り替え
- 組み込みペルソナ: 5種類のプリセットペルソナを標準搭載
- チャット中の切り替え: 会話中にいつでも変更可能

---

## ペルソナとは?

ペルソナは **AI の性格と役割を定義するプロファイル**です。

| 要素 | 説明 | 例 |
|------|------|----|
| **名前** | ペルソナの呼び名 | "Nion"、"家計コーチ"、"英語の先生" |
| **役割** | 専門分野 | "個人ファイナンスアドバイザー" |
| **性格説明** | 会話スタイルと態度 | "親切で節約を重視し..." |
| **挨拶** | 会話開始時のメッセージ | "こんにちは！今日の家計を確認しましょう。" |

---

## 組み込みペルソナ

Starnion には **5種類の組み込みペルソナ**が用意されており、すぐに利用できます。

| ペルソナ | 役割 | 特徴 |
|---------|------|------|
| **こころの友** | 感情サポートパートナー | 共感力が高く、温かい言葉で気持ちを受け止める |
| **基本アシスタント** | 汎用 AI アシスタント | 親切で温かみがあり、日常全般をサポート |
| **金融エキスパート** | 個人ファイナンスアドバイザー | 節約・投資中心、財務状況の分析と助言 |
| **データアナリスト** | データインサイト専門家 | 数値と統計を明確に分析し可視化をサポート |
| **ライフコーチ** | 目標達成コーチ | モチベーション向上と習慣形成を体系的に支援 |

## ペルソナの作成

1. **設定 > ペルソナ** に移動します。
2. **+ 新規ペルソナ** ボタンをクリックします。
3. 名前・役割・性格説明・挨拶を入力します。
4. **保存**をクリックします。

### ペルソナの例

**家計コーチ**
```
名前: 家計コーチ
役割: 個人ファイナンスアドバイザー
性格説明: 節約と投資に集中し、無駄な出費をやさしく指摘する。
挨拶: こんにちは！今日の家計状況を一緒に確認しましょうか？
```

**日記の友達**
```
名前: なごみ
役割: 日記のサポーター
性格説明: 共感力が高く、ユーザーの感情に深く寄り添う。温かい言葉を使う。
挨拶: 今日一日はどうだった？話を聞かせて。
```

**英語の先生**
```
名前: Teacher Emma
役割: 英会話教師
性格説明: 英語のみで会話し、文法エラーをやさしく訂正する。
挨拶: Hi! Ready for some English practice today?
```

## ペルソナの切り替え

### Web UI から

1. **設定 > ペルソナ** で有効にしたいペルソナの **有効化** ボタンをクリックします。
2. またはチャット画面上部のペルソナ名をクリックします。

### チャット中の切り替え

```
ユーザー: 英語の先生に切り替えて
AI:       Hi! I'm Teacher Emma. Let's practice English together!

ユーザー: Nion に戻して
AI:       戻ってきたよ！何か手伝えることはある？
```

## よくある質問

**Q. ペルソナはいくつ作れますか?**
制限なく作成できます。ただし、有効にできるペルソナは一度に一つです。

**Q. ペルソナを切り替えると以前の会話コンテキストが消えますか?**
いいえ。会話履歴とメモリは保持されます。AI の口調と応答スタイルのみ変わります。

**Q. 組み込みペルソナを編集できますか?**
性格説明と挨拶は変更できます。名前は変更できません。

**Q. 良いペルソナ作成のコツは?**
性格説明を具体的に書くほど、AI がより一貫したキャラクターを維持します。
