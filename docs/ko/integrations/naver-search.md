# 네이버 검색 연동

Starnion과 네이버 검색 API를 연결하면 AI 에이전트가 한국어 컨텐츠에 특화된 검색을 수행할 수 있습니다. 블로그, 뉴스, 카페, 쇼핑, 지식인 등 네이버의 다양한 검색 서비스를 자연어로 활용할 수 있습니다.

---

## 개요

네이버 검색 연동을 사용하면:

- **블로그 검색**: 네이버 블로그 포스팅 검색
- **뉴스 검색**: 최신 한국어 뉴스 기사 검색
- **카페 검색**: 네이버 카페 게시글 검색
- **쇼핑 검색**: 상품 정보 검색
- **지식인 검색**: 질문과 답변 검색

> **옵트인 기능:** 네이버 검색 연동은 기본적으로 비활성화되어 있습니다. 네이버 개발자센터에서 앱을 생성하고 API 키를 등록한 후 스킬을 활성화해야 사용할 수 있습니다.

---

## 지원 검색 타입

| 검색 타입 | 설명 |
|----------|------|
| 블로그 | 네이버 블로그 포스팅 검색 |
| 뉴스 | 한국어 뉴스 기사 검색 |
| 카페 | 네이버 카페 게시글 검색 |

| 지식인 | 질문과 답변 검색 |

## 사전 준비: 네이버 개발자센터 앱 생성

### 1단계: 앱 등록

1. [네이버 개발자센터](https://developers.naver.com/)에 접속합니다.
2. **Application** → **애플리케이션 등록** 클릭.
3. **애플리케이션 이름**을 입력합니다 (예: `Starnion`).
4. **사용 API**에서 **검색**을 선택합니다.
5. **비로그인 오픈 API 서비스 환경**에서 적절한 환경을 추가합니다.
6. **등록하기** 클릭.

### 2단계: 인증 정보 확인

1. 등록 완료 후 **내 애플리케이션** 목록에서 생성한 앱을 클릭합니다.
2. **Client ID**와 **Client Secret**을 복사합니다.

> **무료 사용:** 네이버 검색 API는 일 25,000회까지 무료로 호출할 수 있습니다.

---

## 설정 방법

### 웹 UI에서 API 키 등록

1. Starnion 웹 UI에 로그인합니다.
2. 좌측 메뉴 → **Settings** → **Integrations** 탭 클릭.
3. **네이버 검색** 섹션에서 다음 필드를 입력합니다:
   - **Client ID**: 네이버 개발자센터에서 발급받은 Client ID
   - **Client Secret**: 네이버 개발자센터에서 발급받은 Client Secret
4. **저장** 버튼 클릭.
5. **네이버 검색 스킬 활성화** 토글을 켭니다.

## 사용 방법

네이버 검색 연동이 설정되면 AI에게 자연어로 요청합니다.

### 뉴스 검색

```
나: 네이버에서 오늘 경제 뉴스 찾아줘
봇: (네이버 뉴스 검색)
    최근 경제 뉴스:
    1. "코스피 2,650선 돌파, 반도체주 강세" — 한국경제
    2. "환율 1,300원대 안정세 유지" — 매일경제
    3. "한국은행 기준금리 동결 결정" — 연합뉴스

나: 한국 부동산 최신 뉴스 검색해줘
봇: (네이버 뉴스 검색)
    부동산 관련 최근 뉴스:
    1. "서울 아파트 거래량 3개월 연속 증가" — 조선일보
    ...
```

### 블로그 검색

```
나: 네이버 블로그에서 맥북 프로 후기 찾아줘
봇: (네이버 블로그 검색)
    맥북 프로 관련 블로그 글:
    1. "[리뷰] M4 맥북 프로 한 달 사용 후기" — 테크블로그
    2. "맥북 프로 vs 맥북 에어 비교 정리" — IT리뷰어
    ...
```

### 쇼핑 검색

```
나: 네이버 쇼핑에서 무선 이어폰 가격 검색해줘
봇: (네이버 쇼핑 검색)
    무선 이어폰 검색 결과:
    1. AirPods Pro 2 — 299,000원 (최저가)
    2. 삼성 Galaxy Buds3 Pro — 259,000원
    3. 소니 WF-1000XM5 — 279,000원
```

### 지식인 검색

```
나: 네이버 지식인에서 React vs Vue 비교 찾아줘
봇: (네이버 지식인 검색)
    관련 질문과 답변:
    1. "React와 Vue 중 어떤 프레임워크가 좋을까요?"
    2. "프론트엔드 프레임워크 선택 고민"
    ...
```

## 한국어 검색에 특화된 이유

네이버 검색 API는 한국어 컨텐츠에 최적화되어 있습니다:

- **한국어 형태소 분석**: 한국어 검색어를 정확하게 이해
- **국내 컨텐츠**: 한국 웹사이트, 블로그, 뉴스에 특화
- **로컬 정보**: 한국 쇼핑, 지역 정보, 국내 커뮤니티 검색
- **카페/지식인**: 글로벌 검색 엔진에서는 찾기 어려운 한국 커뮤니티 정보

> **팁:** 한국어 검색은 네이버 검색, 글로벌 검색은 Tavily를 함께 사용하면 더욱 풍부한 정보를 얻을 수 있습니다.

## 연결 해제 방법

1. Settings → Integrations → 네이버 검색 섹션.
2. **연결 해제** 버튼 클릭.
3. 저장된 Client ID와 Client Secret이 즉시 삭제됩니다.

---

## 문제 해결

### "네이버 검색 연동이 되어 있지 않아요"

Settings → Integrations → 네이버 검색에서 Client ID와 Client Secret을 등록했는지 확인하세요.

### "네이버 API 인증 실패" (401 오류)

- Client ID와 Client Secret이 올바른지 확인합니다.
- [네이버 개발자센터](https://developers.naver.com/)에서 앱이 활성 상태인지 확인합니다.

### "일일 호출 한도를 초과했어요" (429 오류)

- 네이버 검색 API의 일일 호출 한도(25,000회)를 초과했습니다.
- 다음 날 자동으로 초기화됩니다.

## FAQ

**Q: 네이버 검색과 Tavily 검색을 함께 사용할 수 있나요?**
A: 네, 두 검색 서비스를 동시에 활성화할 수 있습니다. 닌온이 검색 내용에 따라 적절한 서비스를 자동으로 선택합니다.

**Q: 영어 검색도 네이버로 할 수 있나요?**
A: 가능하지만, 영어 컨텐츠 검색은 Tavily를 사용하는 것을 권장합니다. 네이버는 한국어 컨텐츠에 최적화되어 있습니다.

**Q: 네이버 검색 API는 무료인가요?**
A: 일 25,000회까지 무료입니다. 일반적인 개인 사용에는 충분한 양입니다.
