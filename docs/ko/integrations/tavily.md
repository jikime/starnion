# Tavily 검색 연동

Starnion과 Tavily를 연결하면 AI 에이전트가 실시간 웹 검색을 수행하여 최신 정보를 제공할 수 있습니다. 닌온이 최신 정보가 필요할 때 자동으로 Tavily 검색을 실행합니다.

---

## 개요

Tavily 연동을 사용하면:

- **실시간 웹 검색**: 최신 뉴스, 정보, 데이터를 검색
- **뉴스 검색**: 최근 뉴스 기사 검색
- **학술 검색**: 학술 논문 및 연구 자료 검색

> **옵트인 기능:** Tavily 연동은 기본적으로 비활성화되어 있습니다. API 키를 등록한 후 스킬을 활성화해야 사용할 수 있습니다.

---

## 지원 기능 목록

| 기능 | 설명 |
|------|------|
| 웹 검색 | 실시간 웹 검색으로 최신 정보 제공 |
| 뉴스 검색 | 최근 뉴스 기사 검색 |
| 학술 검색 | 학술 논문 및 연구 자료 검색 |

---

## 사전 준비: Tavily API 키 발급

### 1단계: API 키 생성

1. [tavily.com](https://tavily.com)에 접속합니다.
2. 계정을 생성하거나 로그인합니다.
3. 대시보드에서 **API Keys** 섹션을 찾습니다.
4. **Create API Key** 버튼 클릭.
5. 생성된 API 키를 복사합니다 (`tvly-...` 형식).

> **무료 티어:** Tavily는 월 1,000회 검색까지 무료로 제공합니다.

---

## 설정 방법

### 웹 UI에서 API 키 등록

1. Starnion 웹 UI에 로그인합니다.
2. 좌측 메뉴 → **Settings** → **Integrations** 탭 클릭.
3. **Tavily** 섹션에서 **API Key** 입력 필드를 찾습니다.
4. 복사한 API 키(`tvly-...`)를 붙여넣습니다.
5. **저장** 버튼 클릭.
6. **Tavily 스킬 활성화** 토글을 켭니다.

## 사용 방법

Tavily 연동이 설정되면 닌온이 최신 정보가 필요할 때 자동으로 검색을 실행합니다.

### 실시간 정보 검색

```
나: 오늘 코스피 지수 알려줘
봇: (Tavily 검색 실행)
    오늘 코스피 지수는 2,650.12포인트로 전일 대비 0.3% 상승했어요.
    출처: 한국거래소

나: 최신 AI 뉴스 찾아줘
봇: (Tavily 검색 실행)
    최근 AI 관련 주요 뉴스:
    1. "OpenAI, GPT-5 출시 예정" — TechCrunch
    2. "구글, Gemini 2.0 업데이트 발표" — The Verge
    3. "한국 AI 스타트업 투자 사상 최대" — 매일경제
```

## 자동 검색 동작

닌온은 다음과 같은 경우 자동으로 Tavily 검색을 실행합니다:

- 최신 뉴스나 현재 정보를 요청할 때
- 실시간 데이터(주가, 환율, 날씨 등)를 물어볼 때
- AI 학습 데이터에 포함되지 않은 최근 정보가 필요할 때

> **팁:** "검색해줘", "찾아줘" 등의 표현을 사용하면 Tavily 검색이 실행될 확률이 높아집니다.

---

## 연결 해제 방법

1. Settings → Integrations → Tavily 섹션.
2. **연결 해제** 버튼 클릭.
3. 저장된 API 키가 즉시 삭제됩니다.

---

## 문제 해결

### "Tavily 연동이 되어 있지 않아요"

Settings → Integrations → Tavily에서 API 키를 등록했는지 확인하세요.

### "Tavily API 키가 유효하지 않아요" (401 오류)

- API 키가 올바른지 확인합니다.
- [tavily.com](https://tavily.com) 대시보드에서 키가 활성 상태인지 확인합니다.

### "검색 한도를 초과했어요" (429 오류)

- 무료 티어의 월간 검색 한도(1,000회)를 초과했을 수 있습니다.
- 유료 플랜으로 업그레이드하거나 다음 달까지 기다리세요.

## FAQ

**Q: Tavily 검색 결과가 정확하지 않을 수 있나요?**
A: 웹 검색 결과를 기반으로 하므로 항상 출처를 확인하는 것을 권장합니다. 닌온은 가능한 출처를 함께 표시합니다.

**Q: Tavily 없이도 최신 정보를 알 수 있나요?**
A: Tavily 연동 없이는 AI 학습 데이터 기반으로만 답변합니다. 실시간 정보가 필요하면 Tavily 연동을 권장합니다.

**Q: 무료 티어로 충분한가요?**
A: 개인 사용 목적이라면 월 1,000회 검색이면 대부분 충분합니다. 팀 사용 시 유료 플랜을 검토하세요.
