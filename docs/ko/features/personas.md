# 페르소나

## 개요

페르소나 기능은 Starnion AI의 이름, 성격, 역할을 커스터마이징할 수 있는 AI 프로필 시스템입니다. 다양한 페르소나를 만들어 상황에 맞게 전환하면, 마치 다른 성격의 AI와 대화하는 것처럼 활용할 수 있습니다.

**주요 특징:**
- 페르소나 커스터마이징: 이름, 역할, 성격 설명, 인사말 설정
- 다중 페르소나: 여러 페르소나를 만들어 상황에 맞게 전환
- 기본 제공 페르소나: 5가지 내장 페르소나 제공
- 채팅 중 전환: 대화 도중에도 페르소나 변경 가능

---

## 페르소나란?

페르소나는 AI의 **성격과 역할을 정의하는 프로필**입니다.

| 구성 요소 | 설명 | 예시 |
|----------|------|------|
| **이름** | 페르소나의 호칭 | "닌온", "재무 코치", "영어 선생님" |
| **역할** | 전문 분야 | "개인 재무 어드바이저" |
| **성격 설명** | 대화 스타일과 태도 | "친절하고 절약을 강조하며..." |
| **인사말** | 대화 시작 시 메시지 | "안녕! 오늘 재무 상담을 도와줄게." |

---

## 기본 제공 페르소나

Starnion에는 **5가지 기본 페르소나**가 내장되어 있습니다. 별도 설정 없이 바로 사용할 수 있습니다.

| 페르소나 | 역할 | 특징 |
|---------|------|------|
| **마음 친구** | 감성 대화 파트너 | 공감 능력이 뛰어나며 따뜻한 말투로 감정을 나눔 |
| **기본 비서** | 범용 AI 어시스턴트 | 친절하고 다정한 스타일, 일상 전반을 도움 |
| **금융 전문가** | 개인 재무 어드바이저 | 절약·투자 중심, 재무 상태 분석 및 조언 |
| **데이터 분석가** | 데이터 인사이트 전문가 | 수치와 통계를 명확하게 분석하고 시각화 |
| **라이프코치** | 목표 달성 코치 | 동기 부여와 습관 형성을 체계적으로 지원 |

## 페르소나 만들기

1. **기능 > 페르소나** 메뉴로 이동합니다.
2. **+ 새 페르소나** 버튼을 클릭합니다.
3. 이름, 역할, 성격 설명, 인사말을 입력합니다.
4. **저장**을 클릭합니다.

### 페르소나 예시

**재무 코치**
```
이름: 재무 코치
역할: 개인 재무 어드바이저
성격 설명: 절약과 투자에 집중하며, 불필요한 지출을 발견하면 부드럽게 지적한다.
인사말: 안녕! 오늘의 재무 상태를 함께 점검해볼까?
```

**일기 친구**
```
이름: 다정이
역할: 일기 도우미
성격 설명: 감성적이고 공감 능력이 풍부하다. 따뜻한 말투를 사용한다.
인사말: 오늘 하루는 어땠어? 이야기 들려줘.
```

**영어 선생님**
```
이름: Teacher Emma
역할: 영어 회화 교사
성격 설명: 영어로만 대화하며, 문법 오류를 친절하게 교정해준다.
인사말: Hi! Ready for some English practice today?
```

## 페르소나 전환

### 웹에서 전환

1. **기능 > 페르소나**에서 활성화할 페르소나 옆의 **활성화** 버튼을 클릭합니다.
2. 또는 채팅 화면 상단의 페르소나 이름을 클릭합니다.

### 채팅에서 전환

```
사용자: 영어 선생님 모드로 바꿔줘
AI:     Hi! I'm Teacher Emma. Let's practice English together!

사용자: 다시 닌온으로 돌아가줘
AI:     다시 돌아왔어! 무엇을 도와줄까?
```

## 자주 묻는 질문

**Q. 페르소나는 최대 몇 개까지 만들 수 있나요?**
제한 없이 만들 수 있습니다. 단, 활성화할 수 있는 페르소나는 한 번에 하나입니다.

**Q. 페르소나를 바꾸면 이전 대화 맥락이 사라지나요?**
아닙니다. 대화 이력과 기억은 유지됩니다. 말투와 응답 스타일만 변경됩니다.

**Q. 기본 제공 페르소나를 수정할 수 있나요?**
성격 설명과 인사말은 수정 가능합니다. 이름은 변경할 수 없습니다.

**Q. 좋은 페르소나를 만드는 팁이 있나요?**
성격 설명을 구체적으로 작성할수록 AI가 더 일관된 캐릭터를 유지합니다.
