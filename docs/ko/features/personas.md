# 페르소나

## 개요

페르소나 기능은 닌온의 이름, 성격, 역할을 커스터마이징할 수 있는 AI 프로필 시스템입니다. 다양한 페르소나를 만들어 상황에 맞게 전환하면, 마치 다른 성격의 AI와 대화하는 것처럼 활용할 수 있습니다.

**주요 특징:**
- 페르소나 커스터마이징: 이름, 역할, 성격 설명, 인사말 설정
- 다중 페르소나: 여러 페르소나를 만들어 상황에 맞게 전환
- 기본 페르소나: Nion (닌온) - 기본 내장 페르소나
- 채팅 중 전환: 대화 도중에도 페르소나 변경 가능

---

## 페르소나란?

페르소나는 AI의 **성격과 역할을 정의하는 프로필**입니다. 같은 AI라도 페르소나에 따라 다른 방식으로 대화합니다.

| 구성 요소 | 설명 | 예시 |
|----------|------|------|
| **이름** | 페르소나의 호칭 | "닌온", "재무 코치", "영어 선생님" |
| **역할** | 페르소나의 전문 분야 | "개인 재무 어드바이저" |
| **성격 설명** | 대화 스타일과 태도 | "친절하고 절약을 강조하며..." |
| **인사말** | 대화 시작 시 보내는 메시지 | "안녕! 오늘 재무 상담을 도와줄게." |

## 기본 페르소나

Starnion에는 **Nion (닌온)** 이라는 기본 페르소나가 내장되어 있습니다.

- **이름**: Nion
- **역할**: 개인 AI 어시스턴트
- **성격**: 친절하고 다정한 대화 스타일, 이모지 활용, 사용자의 일상에 관심을 가짐
- **특징**: 일기, 가계부, 메모, 목표 등 모든 기능을 균형 있게 지원

## 페르소나 만들기

### 웹에서 생성

1. **설정 > 페르소나** 메뉴로 이동합니다.
2. **+ 새 페르소나** 버튼을 클릭합니다.
3. 다음 항목을 입력합니다:
   - **이름** (필수): 페르소나를 부를 이름
   - **역할** (선택): 이 페르소나의 전문 분야
   - **성격 설명** (필수): AI가 참고할 대화 스타일과 태도 설명
   - **인사말** (선택): 대화 시작 시 자동으로 보내는 메시지
4. **저장**을 클릭합니다.

### 페르소나 예시

다음은 활용 가능한 페르소나 아이디어입니다:

#### 재무 코치

```
이름: 재무 코치
역할: 개인 재무 어드바이저
성격 설명: 절약과 투자에 집중하며, 불필요한 지출을 발견하면 부드럽게 지적한다.
         격려를 아끼지 않고, 작은 절약도 칭찬한다. 재무 목표 달성을 응원한다.
인사말: 안녕! 오늘의 재무 상태를 함께 점검해볼까?
```

**대화 예시:**
```
사용자: 오늘 카페에서 7,000원 썼어
AI:     커피 한 잔이군! 이번 달 카페 지출이 총 45,000원이 됐어.
        혹시 텀블러 가져가면 할인받을 수 있는 곳도 많으니 참고해봐!
```

#### 일기 친구

```
이름: 다정이
역할: 일기 도우미
성격 설명: 감성적이고 공감 능력이 풍부하다. 사용자의 감정에 깊이 공감하고,
         하루를 돌아보는 것을 도와준다. 따뜻한 말투를 사용한다.
인사말: 오늘 하루는 어땠어? 이야기 들려줘.
```

**대화 예시:**
```
사용자: 오늘 발표를 망쳤어...
AI:     그랬구나... 발표 준비 많이 했을 텐데 속상했겠다.
        어떤 부분이 아쉬웠어? 이야기해주면 다음에 더 잘할 수 있는 방법을 같이 생각해볼게.
```

#### 영어 선생님

```
이름: Teacher Emma
역할: 영어 회화 교사
성격 설명: 영어로만 대화하며, 문법 오류를 친절하게 교정해준다.
         대화 수준을 사용자에 맞게 조절하고, 새로운 표현을 자연스럽게 가르친다.
인사말: Hi! Ready for some English practice today?
```

**대화 예시:**
```
사용자: I goed to the store yesterday
AI:     Great effort! Just a small correction:
        "goed" → "went" (go-went-gone is an irregular verb)
        "I went to the store yesterday." Perfect otherwise!
        What did you buy there?
```

## 페르소나 전환

### 웹에서 전환

1. **설정 > 페르소나** 에서 활성화할 페르소나 옆의 **활성화** 버튼을 클릭합니다.
2. 또는 채팅 화면 상단의 페르소나 이름을 클릭하여 변경합니다.

### 채팅에서 전환

채팅 중에 자연어로 페르소나 전환을 요청할 수 있습니다.

```
사용자: 영어 선생님 모드로 바꿔줘
AI:     Hi! I'm Teacher Emma. Let's practice English together!

사용자: 다시 닌온으로 돌아가줘
AI:     다시 돌아왔어! 무엇을 도와줄까?
```

## 팁 & FAQ

**Q. 페르소나는 최대 몇 개까지 만들 수 있나요?**
제한 없이 자유롭게 만들 수 있습니다. 다만 활성화할 수 있는 페르소나는 한 번에 하나입니다.

**Q. 페르소나를 바꾸면 이전 대화 맥락이 사라지나요?**
아닙니다. 페르소나를 전환해도 대화 이력과 기억은 유지됩니다. 단, AI의 말투와 응답 스타일이 새 페르소나에 맞게 변경됩니다.

**Q. 기본 Nion 페르소나를 수정할 수 있나요?**
기본 페르소나의 성격 설명과 인사말은 수정할 수 있습니다. 이름은 변경할 수 없습니다.

**Q. 페르소나별로 다른 모델을 사용할 수 있나요?**
현재는 페르소나와 모델은 별도로 설정됩니다. 모델 설정은 **설정 > 모델** 메뉴에서 관리합니다.

**Q. 좋은 페르소나를 만드는 팁이 있나요?**
성격 설명을 구체적으로 작성할수록 AI가 더 일관된 캐릭터를 유지합니다. "친절한"보다는 "항상 격려를 아끼지 않고, 작은 성과도 칭찬하며, 따뜻한 말투를 사용한다"처럼 상세하게 작성하세요.
