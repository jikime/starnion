# 이미지

## 개요

Starnion의 이미지 기능은 사진을 업로드하여 AI가 내용을 분석하고, 영수증을 자동으로 인식해 가계부에 기록하며, 텍스트 설명만으로 새 이미지를 생성하는 기능을 제공합니다.

채팅에서 이미지를 첨부하면 "이 영수증 가계부에 기록해줘", "이 사진에 뭐가 있어?", "이 차트 분석해줘" 같은 자연어 명령으로 다양하게 활용할 수 있습니다. 업로드된 이미지는 갤러리에서 모아볼 수 있으며 AI 분석 결과도 함께 저장됩니다.

## 이미지 업로드

### 채팅에서 이미지 첨부

채팅 입력창의 파일 첨부 버튼을 클릭하거나, 이미지를 채팅창에 드래그앤드롭하여 업로드할 수 있습니다.

```
사용자: [이미지 첨부] 이게 뭔지 설명해줘
AI:    이 사진은 한국의 전통 가옥(한옥)입니다. 기와지붕과
       나무 기둥이 특징적이며, 처마의 곡선이 아름답게
       표현되어 있습니다. 배경으로 보이는 산과 어우러져
       전통 건축의 조화로운 모습을 보여주고 있습니다.
```

### 이미지 갤러리에서 업로드

상단 메뉴 > **이미지** 탭에서 이미지 업로드 버튼을 클릭하거나 드래그앤드롭으로 추가할 수 있습니다.

## 이미지 분석

AI가 이미지의 내용을 설명하고 다양한 질문에 답할 수 있습니다.

**사진 내용 분석:**
```
사용자: [식당 메뉴판 사진 첨부] 여기서 뭘 먹으면 좋을까?
AI:    메뉴를 보니 이 식당은 한식 위주네요. 추천 메뉴:
       - 된장찌개 (8,000원): 기본에 충실한 메뉴
       - 제육볶음 (9,000원): 인기 메뉴로 보입니다
       - 비빔밥 (8,500원): 건강식으로 좋은 선택
       점심 특선이 11,000원으로 합리적입니다.
```

**차트 및 그래프 분석:**
```
사용자: [매출 그래프 이미지 첨부] 이 그래프 분석해줘
AI:    이 막대그래프는 월별 매출 추이를 보여줍니다.
       - 1월~3월: 상승세 (평균 230만원)
       - 4월: 급격한 하락 (180만원, 전월 대비 -22%)
       - 5월~6월: 회복 추세
       4월의 하락 원인 파악이 필요해 보입니다.
```

**문서/텍스트 추출 (OCR):**
```
사용자: [명함 사진 첨부] 연락처 정리해줘
AI:    명함에서 추출한 정보:
       이름: 김민준
       직책: 마케팅 팀장
       회사: (주)스타니온
       이메일: minjun@starnion.com
       전화: 010-1234-5678
```

## 영수증 OCR — 가계부 자동 연동

영수증 사진을 올리면 AI가 자동으로 금액과 항목을 인식하여 가계부에 기록합니다.

```
사용자: [편의점 영수증 사진 첨부] 가계부에 저장해줘
AI:    영수증을 인식했어요:
       - 삼각김밥 1,200원
       - 아메리카노 1,500원
       - 음료수 1,800원
       합계: 4,500원 (2025-03-08 14:32)

       식비로 4,500원 기록할게요. 맞나요?

사용자: 응, 맞아
AI:    편의점 / 4,500원 기록 완료! 오늘 식비 합계: 12,800원
```

**영수증 일괄 처리:**
```
사용자: [여러 영수증 사진들 첨부] 이번 주 영수증 다 기록해줘
AI:    3장의 영수증을 인식했어요:
       1. 마트 영수증 - 45,200원 (3/5)
       2. 카페 영수증 - 8,500원 (3/6)
       3. 식당 영수증 - 32,000원 (3/7)
       총 85,700원을 가계부에 기록했습니다.
```

## 이미지 갤러리

**이미지** 탭에서 지금까지 사용한 모든 이미지를 한눈에 볼 수 있습니다.

### 갤러리 필터

| 필터 | 설명 |
|------|------|
| 전체 | 모든 이미지 |
| 생성됨 | AI가 생성한 이미지 |
| 편집됨 | AI가 편집한 이미지 |
| 분석됨 | 업로드 후 AI 분석한 이미지 |
| 웹 채팅 | 웹에서 첨부한 이미지 |
| 텔레그램 | 텔레그램에서 전송한 이미지 |

### 갤러리에서 할 수 있는 것

- 이미지 클릭 시 원본 크기로 보기
- AI 분석 결과 텍스트 확인
- 생성에 사용된 프롬프트 확인
- 이미지 삭제

## AI 이미지 생성

텍스트 설명(프롬프트)으로 이미지를 생성할 수 있습니다. 생성된 이미지는 갤러리에 자동으로 저장됩니다.

```
사용자: 한강에서 바라보는 서울 야경 그림 그려줘
AI:    이미지를 생성 중입니다...
       [이미지 생성됨]
       한강에서 바라보는 서울 야경 이미지를 생성했어요.
       갤러리에서도 확인할 수 있습니다.
```

**다양한 스타일 지정:**
```
사용자: 수채화 스타일로 제주도 올레길 풍경 그려줘
AI:    [수채화 스타일 제주 올레길 이미지 생성됨]

사용자: 귀여운 고양이 캐릭터 스티커 만들어줘
AI:    [고양이 스티커 이미지 생성됨]
```

## 이미지 편집

기존 이미지를 AI로 수정할 수 있습니다. 편집된 이미지는 갤러리에 `edited_` 접두사로 저장됩니다.

```
사용자: [사진 첨부] 이 사진 배경을 흐리게 해줘 (보케 효과)
AI:    배경을 흐리게 처리한 이미지를 생성했어요.
       [편집된 이미지]

사용자: [사진 첨부] 이 사진에서 사람 지워줘
AI:    배경을 복원하여 사람을 제거했어요.
       [편집된 이미지]
```

## 활용 예시

**업무 활용:**
```
사용자: [화이트보드 사진 첨부] 회의 내용 텍스트로 정리해줘
AI:    화이트보드 내용:
       [프로젝트 일정]
       - 기획: 3/1 ~ 3/15
       - 개발: 3/16 ~ 4/30
       - QA: 5/1 ~ 5/15
       - 출시: 5/20
```

**일상 활용:**
```
사용자: [냉장고 사진 첨부] 오늘 저녁 뭐 해먹을 수 있을까?
AI:    냉장고에 있는 재료로 만들 수 있는 요리:
       1. 계란볶음밥 - 계란, 밥, 대파 있음
       2. 된장국 - 두부, 애호박, 된장 있음
       3. 닭볶음탕 - 닭다리, 감자, 양파 있음
```

## 💬 텔레그램 사용 예시

> 니온에게 자연어로 말하면 됩니다. 정해진 명령어 없이 일상 언어로 대화하세요!

### 이미지 분석

```
[영수증 사진 첨부] 이거 정리해줘
```
→ 영수증을 분석하여 품목과 금액을 자동으로 가계부에 기록합니다

```
[음식 사진 첨부] 칼로리 얼마나 될까?
```
→ 음식 사진을 분석하여 예상 칼로리를 알려줍니다

### 이미지 생성

```
고양이가 우주복 입고 달에 서 있는 그림 그려줘
```
→ AI가 설명에 맞는 이미지를 생성합니다

### 업무 활용

```
[화이트보드 사진 첨부] 회의 내용 텍스트로 정리해줘
```
→ 화이트보드 내용을 텍스트로 변환합니다

## 팁 & FAQ

**Q. 영수증 인식이 잘 안 돼요.**
A. 밝은 환경에서 정면으로 촬영하고, 영수증 전체가 화면 안에 들어오도록 해주세요. 구겨진 영수증은 펴서 찍으면 인식률이 높아집니다.

**Q. 이미지를 여러 장 한 번에 올릴 수 있나요?**
A. 채팅에서는 한 번에 여러 장 첨부가 가능합니다. 단, AI 분석은 한 번에 처리하므로 너무 많은 이미지는 응답이 느릴 수 있습니다.

**Q. 생성된 이미지를 다운로드할 수 있나요?**
A. 갤러리에서 이미지를 클릭하면 원본 크기로 볼 수 있고, 오른쪽 클릭 또는 길게 누르면 저장할 수 있습니다.

**Q. 개인 정보가 담긴 이미지도 안전한가요?**
A. 업로드된 이미지는 암호화되어 저장되며, 본인만 접근할 수 있습니다. 단, 민감한 정보가 포함된 이미지 업로드 시에는 주의하세요.
