# 소비 분석

## 개요

소비 분석 기능은 가계부에 기록된 지출 데이터를 **카테고리별, 기간별로 시각화**하여 소비 패턴을 파악할 수 있는 분석 대시보드입니다. 파이차트, 라인차트 등 다양한 차트로 지출 트렌드를 한눈에 확인하고, AI가 절약 인사이트를 제공합니다.

> 돈이 어디로 가는지 알아야 절약할 수 있습니다. 니온에게 "이번 달 소비 패턴 분석해줘"라고 말해보세요.

---

## 주요 차트

### 카테고리별 지출 비율 (파이차트)

전체 지출에서 각 카테고리가 차지하는 비율을 파이차트로 보여줍니다.

| 항목 | 설명 |
|------|------|
| **카테고리 색상** | 식비(초록), 교통(파랑), 쇼핑(분홍), 여가(주황) 등 |
| **비율 표시** | 각 카테고리의 백분율 (%) |
| **금액 표시** | 호버 시 실제 금액 표시 |

### 월별 지출 트렌드 (라인차트)

최근 6개월간의 월별 총 지출을 라인차트로 보여줍니다.

- X축: 월 (예: 10월, 11월, 12월, 1월, 2월, 3월)
- Y축: 지출 금액
- 전월 대비 증감 방향 표시

### 최다 지출 카테고리 TOP 5

이번 달 가장 많이 지출한 카테고리 5개를 순위별로 보여줍니다.

| 순위 | 표시 정보 |
|------|----------|
| 1위 | 카테고리명 + 금액 + 전월 대비 변화율 |
| 2위 | 카테고리명 + 금액 + 전월 대비 변화율 |
| ... | ... |
| 5위 | 카테고리명 + 금액 + 전월 대비 변화율 |

### 전월 대비 변화율

이번 달과 지난 달의 지출을 비교하여 변화율을 표시합니다.

| 표시 | 의미 |
|------|------|
| **↑ 15%** (빨강) | 지난달 대비 15% 증가 |
| **↓ 8%** (초록) | 지난달 대비 8% 감소 |
| **— 0%** (회색) | 변화 없음 |

## 기간별 조회

상단 필터를 사용하여 분석 기간을 변경할 수 있습니다.

| 기간 | 설명 |
|------|------|
| **이번 주** | 최근 7일 지출 분석 |
| **이번 달** | 당월 1일부터 오늘까지 |
| **지난 달** | 전월 전체 |
| **최근 3개월** | 최근 3개월 누적 |
| **최근 6개월** | 최근 6개월 누적 |
| **올해** | 1월 1일부터 오늘까지 |

## AI 분석 인사이트

니온이 소비 데이터를 분석하여 맞춤형 인사이트를 제공합니다.

### 인사이트 예시

```
📊 이번 달 소비 분석 결과:

1. 총 지출: 1,520,000원 (전월 대비 ↑12%)
2. 최다 지출: 식비 480,000원 (31.6%)
3. 증가 항목: 쇼핑 +45% (세일 시즌 영향 추정)
4. 감소 항목: 교통비 -20% (재택근무 영향 추정)
5. 절약 제안: 식비를 주 3회 집밥으로 전환하면 월 약 8만원 절약 가능
```

## 💬 텔레그램 사용 예시

> 니온에게 자연어로 말하면 됩니다. 정해진 명령어 없이 일상 언어로 대화하세요!

### 전체 분석

```
이번 달 소비 패턴 분석해줘
```
→ 카테고리별 지출 비율, 전월 대비 변화, 절약 제안을 알려줍니다

### 카테고리별 분석

```
식비 지출 추이 보여줘
```
→ 최근 6개월간 식비 변화 추이를 분석합니다

```
이번 달 교통비 얼마 썼어?
```
→ 교통비 카테고리 지출 합계와 상세 내역을 보여줍니다

### 비교 분석

```
지난달이랑 이번 달 비교해줘
```
→ 전월 대비 카테고리별 증감 내역을 정리합니다

```
올해 소비 트렌드 알려줘
```
→ 월별 지출 추이와 연간 총 지출을 분석합니다

### 절약 제안

```
어디서 절약할 수 있을까?
```
→ 지출 패턴을 분석하여 구체적인 절약 방법을 제안합니다

## FAQ

**Q. 분석에 사용되는 데이터는 어디서 오나요?**

가계부에 기록된 모든 수입/지출 데이터가 분석에 사용됩니다. 가계부 기록이 많을수록 더 정확한 분석이 가능합니다.

**Q. 수입도 분석에 포함되나요?**

소비 분석은 지출 데이터만 분석합니다. 수입 데이터는 가계부의 수입/지출 요약에서 확인할 수 있습니다.

**Q. 차트 데이터를 다운로드할 수 있나요?**

현재 차트 이미지 또는 데이터의 직접 다운로드는 지원하지 않습니다. 채팅에서 니온에게 분석 결과를 텍스트로 요청하면 복사해서 활용할 수 있습니다.

**Q. 특정 기간의 분석만 볼 수 있나요?**

네, 상단 필터에서 기간을 선택하면 해당 기간의 분석 결과만 표시됩니다. 채팅에서도 "3월 소비 분석해줘"처럼 특정 기간을 지정할 수 있습니다.

**Q. 카테고리는 직접 추가할 수 있나요?**

카테고리는 AI가 지출 기록 시 자동으로 분류합니다. 현재 사용자가 직접 카테고리를 추가하는 기능은 지원하지 않지만, AI의 분류가 잘못된 경우 수정할 수 있습니다.
