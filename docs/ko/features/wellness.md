# 마음 정원

## 개요

마음 정원은 Starnion의 감정 웰니스 대시보드입니다. 일기에 기록된 감정을 시각화하고, AI 캐릭터 니온(Nion)이 오늘의 마음 상태에 맞는 메시지를 전달합니다. 단순한 통계 화면이 아닌 "내 감정을 돌보는 공간"을 지향합니다.

> 일기를 쓸수록 감정 레이어가 채워지고, 치유의 나무가 성장합니다.

---

## 화면 구성

마음 정원은 세 개의 컬럼으로 구성됩니다.

```
┌─────────────────┬──────────────────┬─────────────────┐
│  치유의 나무     │     니온 센터     │    상담 공간     │
│  감정 레이어     │  (오늘의 마음)    │  (세션 통계)    │
│  월간 감정 달력  │                  │  (마음의 꽃)    │
│                 │                  │  (진행 중 목표) │
└─────────────────┴──────────────────┴─────────────────┘
```

## 왼쪽 컬럼: 치유의 나무 & 감정 달력

### 치유의 나무

일기 기록이 쌓일수록 나무가 성장합니다.

| 요소 | 설명 |
|------|------|
| **가지** | 5가지 감정 색상으로 에너지 흐름 표시 |
| **하트 열매** | 감정 종류당 하나씩, 최대 5개 박동 애니메이션 |
| **기억 잎사귀** | 일기 10개당 잎사귀 1개 생성, 최대 8개 |
| **중앙 오브** | 금색 구체 ₩ 심볼 — 활력을 나타냄 |

### 감정 레이어 차트

기록된 일기의 감정 분포를 가로 막대 그래프로 표시합니다.

- 감정별 기록 횟수 시각화
- 상위 5개 감정 표시
- 하단에 **"총 N개의 기억"** 카운터

### 월간 감정 달력

날짜별 감정을 색상 도트로 한눈에 볼 수 있는 월간 달력입니다.

| UI 요소 | 설명 |
|---------|------|
| **‹ / ›** 버튼 | 이전 달 / 다음 달로 이동 (미래 이동 불가) |
| **컬러 도트** | 해당 날짜의 일기 감정 색상으로 표시 |
| **오늘 강조** | 금색 테두리 + 금색 폰트 |
| **요일 뱃지** | 일(핑크) / 토(파란) / 평일(흰색) 원형 뱃지 |
| **범례** | Joy · Peace · Calm · Sad · Down 도트 색상 설명 |

## 가운데 컬럼: 니온 센터

### 니온 캐릭터

AI 동반자 니온이 현재 시간대에 맞는 상태로 등장합니다.

| 시간 | 니온 상태 |
|------|---------|
| 0시 (자정 기준) | 치유 (healing) |
| 이후 매 1시간 순환 | 식물 가꾸기 → 관찰 → 광내기 → 청소 |

니온 주변에는 하트·별·도트 오브젝트가 궤도 회전하며 공간을 채웁니다.

### 오늘의 마음

- 가장 최근 일기의 감정이 자동 표시됩니다.
- 하단 **감정 선택 버튼** (5개)으로 직접 오늘 감정을 지정할 수도 있습니다.
- 선택한 감정에 따라 니온의 힐링 메시지가 변경됩니다.

### 힐링 메시지

감정별 위로·격려 메시지가 표시됩니다.

| 감정 | 예시 메시지 |
|------|------------|
| 매우좋음 | "오늘은 정말 빛나는 날이네요! 그 기쁨을 오래 간직하세요 ✨" |
| 좋음 | "마음이 평화롭군요. 이 순간을 충분히 누려요 🌿" |
| 보통 | "평범한 하루도 소중한 기록이에요. 오늘도 잘 하고 있어요 💜" |
| 나쁨 | "힘든 감정도 괜찮아요. 니온이 옆에 있을게요 💙" |
| 매우나쁨 | "많이 지쳐있군요. 천천히, 조금씩 나아가요 🌙" |

## 오른쪽 컬럼: 상담 공간

### 세션 통계 카드

| 항목 | 설명 |
|------|------|
| **도넛 차트** | 목표 달성률 (%) |
| **이번 주 기록** | 최근 7일 일기 작성 횟수 |

### 마음의 꽃

세 가지 마음 챙김 활동 바로가기 버튼입니다.

| 버튼 | 이동 경로 |
|------|---------|
| 🧘 명상 | 일기 페이지 |
| 📖 일기 | 일기 페이지 |
| 💬 대화 | 채팅 페이지 |

### 진행 중 목표

현재 진행 중(`in_progress`) 상태인 목표를 최대 3개 표시합니다.

- 아이콘 + 제목 + 진행률 바 + % 숫자
- 보라→핑크 그라디언트 진행 바

## 감정 시스템

마음 정원은 5가지 정규 감정 값을 사용합니다.

| 키 | 레이블 | 색상 | 이모지 |
|----|-------|------|-------|
| `매우좋음` | Joy | 금색 `#fbbf24` | ✨ |
| `좋음` | Peace | 초록 `#34d399` | 🌿 |
| `보통` | Calm | 보라 `#a78bfa` | 💜 |
| `나쁨` | Sad | 파랑 `#60a5fa` | 💙 |
| `매우나쁨` | Down | 인디고 `#818cf8` | 🌙 |

채팅에서 AI가 자유 형식 감정("기쁨", "피곤", "최악" 등)을 저장해도 자동으로 위 5가지 중 하나로 정규화됩니다.

## 데이터 흐름

```
채팅 / 일기 작성
    ↓
diary_entries 테이블 저장 (mood 정규화 적용)
    ↓
/api/diary?limit=100  ←── 마음 정원 로드 시 호출
/api/goals
    ↓
감정 카운트 집계 → 감정 레이어 차트
날짜별 mood 맵 → 월간 감정 달력
최근 mood → 니온 힐링 메시지
진행 중 목표 → 상담 공간 목표 섹션
```

## FAQ

**Q. 감정 달력에 도트가 표시되지 않아요.**

일기에 저장된 감정이 `매우좋음 / 좋음 / 보통 / 나쁨 / 매우나쁨` 중 하나가 아닌 경우 표시되지 않을 수 있습니다. 채팅을 통해 저장된 기존 일기는 v1.3.3 마이그레이션 이후 자동 정규화됩니다.

**Q. 니온 이미지가 계속 바뀌어요.**

니온은 시간대별로 상태가 변합니다. 매 정시에 자동 전환되며, 이는 정상 동작입니다.

**Q. 감정 선택 버튼을 눌렀는데 저장이 안 돼요.**

감정 선택 버튼은 **현재 화면에서만 감정을 변경**합니다. 실제 일기로 저장하려면 채팅이나 일기 메뉴를 이용해 주세요.

**Q. 목표가 표시되지 않아요.**

목표 메뉴에서 목표를 추가하고 상태를 "진행 중"으로 설정하면 상담 공간에 표시됩니다.
