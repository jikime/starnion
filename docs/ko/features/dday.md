# 디데이 (D-Day)

## 개요

디데이 기능은 중요한 날짜까지 얼마나 남았는지 실시간으로 카운트다운하는 기능입니다. 생일, 기념일, 시험, 여행 출발일 등 기억하고 싶은 날짜를 등록해두면 AI가 "시험까지 며칠 남았어?"라는 질문에 바로 답해드립니다.

최대 30개의 디데이를 등록할 수 있으며, 미래 날짜(D-N)와 과거 날짜(D+N) 모두 지원합니다.

---

## 디데이 등록하기

### 웹에서 등록

1. 사이드바에서 **디데이** 메뉴를 클릭합니다.
2. **+ 디데이 추가** 버튼을 클릭합니다.
3. 아래 정보를 입력합니다:
   - **이름** (필수): 디데이 제목 (예: 생일, 토익 시험)
   - **날짜** (필수): 목표 날짜 (달력으로 선택)
   - **아이콘** (선택): 대표 이모지 (기본값: 📅)
   - **설명** (선택): 부가 설명
4. **저장** 버튼을 클릭합니다.

디데이 목록은 날짜 오름차순(가장 가까운 날짜 먼저)으로 정렬됩니다.

### 채팅으로 등록

AI에게 자연어로 말하면 자동으로 디데이를 설정합니다.

**예시 대화:**

> 사용자: "크리스마스 디데이 설정해줘"
>
> AI: "크리스마스 디데이를 설정했어요! 2025년 12월 25일까지 D-292입니다 🎄"

> 사용자: "결혼기념일 매년 반복으로 설정해줘, 날짜는 5월 20일"
>
> AI: "결혼기념일 디데이를 매년 반복으로 설정했어요! 다음 기념일(5/20)까지 D-73이에요 💍"

> 사용자: "토익 시험이 6월 15일이야, 디데이 잡아줘"
>
> AI: "토익 시험 디데이를 설정했어요! 시험일(6/15)까지 D-99일 남았어요. 화이팅! 📝"

**날짜 표현 방법:**
AI는 자연어 날짜 표현을 자동으로 변환합니다:

| 표현 | 변환 |
|------|------|
| "다음 주 월요일" | 다음 주 월요일 날짜 |
| "이번 달 말" | 해당 월의 마지막 날 |
| "크리스마스" | 12월 25일 |
| "6월 15일" | YYYY-06-15 |
| "내 생일 3월 10일" | YYYY-03-10 |

## D-Day vs D+Day

디데이 계산 방식을 이해하면 더 효과적으로 활용할 수 있습니다.

### D-N (남은 날)

미래 날짜까지 남은 일수를 표시합니다.

```
오늘: 2025년 3월 8일
대상 날짜: 2025년 6월 15일
표시: D-99
```

### D-Day! (당일)

오늘이 등록한 날짜와 같은 경우 표시됩니다.

```
오늘: 2025년 6월 15일
대상 날짜: 2025년 6월 15일
표시: D-Day!
```

### D+N (지난 날)

과거 날짜(이미 지난 날)부터 경과한 일수를 표시합니다. 기념일 등 과거 날짜를 등록했을 때 유용합니다.

```
오늘: 2025년 3월 8일
대상 날짜: 2024년 3월 8일
표시: D+365
```

### 활용 예시

| 목적 | 날짜 예시 | 표시 예시 |
|------|----------|----------|
| 시험 D-Day | 2025-06-15 | D-99 |
| 결혼기념일 (오늘) | 2025-03-08 | D-Day! |
| 연인과 만난 날 기념 | 2024-01-01 | D+431 |
| 여행 출발일 | 2025-04-01 | D-24 |

## 알림

현재 디데이 알림은 채팅을 통해 능동적으로 조회하거나, 주간 리포트를 통해 확인하는 방식으로 작동합니다.

### 주간 리포트 포함

매주 생성되는 주간 리포트에 가까운 디데이 정보가 포함됩니다:

```
[다가오는 디데이]
- D-7: 토익 시험 (6/15)
- D-30: 친구 생일 (4/7)
- D-93: 제주도 여행 (6/9)
```

### 채팅으로 확인

언제든지 AI에게 물어보면 현재 디데이 현황을 알 수 있습니다:

> 사용자: "다음 디데이 언제야?"
>
> AI: "가장 가까운 디데이는 **토익 시험**이에요. D-7, 6월 15일이에요. 마지막 스퍼트 화이팅!"

> 사용자: "이번 달 디데이 있어?"
>
> AI: "이번 달 디데이가 2개 있어요:\n- 3/10 (D-2): 부모님 결혼기념일\n- 3/20 (D-12): 프로젝트 마감"

## 예시: 다양한 디데이 활용

### 생일

```
이름: 엄마 생신
날짜: 2025-04-15
아이콘: 🎂
설명: 선물 준비 잊지 말 것!
```

> 사용자: "엄마 생신 언제야?"
> AI: "엄마 생신은 4월 15일이에요. D-38 남았어요. 선물 미리 준비해두세요! 🎂"

### 기념일

```
이름: 우리 만난 날
날짜: 2023-11-25
아이콘: 💕
```

> 사용자: "우리 만난 지 며칠 됐어?"
> AI: "2023년 11월 25일부터 오늘까지 D+468일이에요. 1년 넘게 함께하셨네요! 💕"

### 시험

```
이름: IELTS 시험
날짜: 2025-05-10
아이콘: 📝
설명: 목표 점수: 7.0
```

> 사용자: "시험까지 얼마나 남았어?"
> AI: "IELTS 시험까지 D-63 남았어요. 5월 10일이에요. 준비 잘 되고 있나요? 📝"

### 여행

```
이름: 일본 여행 출발
날짜: 2025-05-01
아이콘: ✈️
설명: 오사카 4박 5일
```

> 사용자: "여행 출발 며칠 남았어?"
> AI: "일본 여행 출발까지 D-54 남았어요. 설레겠다! 짐 준비 시작해야겠네요 ✈️"

### 프로젝트 마감

```
이름: Q2 보고서 마감
날짜: 2025-03-31
아이콘: 📊
설명: 팀장님께 제출
```

## AI로 조회하기

채팅에서 다양한 방식으로 디데이를 조회할 수 있습니다.

```
사용자: "다음 기념일 언제야?"
AI: "등록된 기념일 중 가장 가까운 건 **결혼기념일**이에요. D-73, 5월 20일이에요 💍"

사용자: "시험까지 며칠 남았어?"
AI: "토익 시험까지 D-99 남았어요. 6월 15일이에요. 매일 조금씩 준비하면 충분해요!"

사용자: "디데이 전체 목록 보여줘"
AI: "등록된 디데이 목록이에요:
  1. D-7: 토익 시험 (6/15) 📝
  2. D-30: 친구 생일 (4/7) 🎂
  3. D-54: 일본 여행 (5/1) ✈️
  4. D-73: 결혼기념일 (5/20) 💍
  5. D+5: 회사 창립기념일 (3/3) 🏢"

사용자: "이미 지난 디데이도 보여줘"
AI: "지난 디데이 포함 전체 목록이에요:
  [진행 중]
  - D-7: 토익 시험
  ...
  [지난 날]
  - D+5: 회사 창립기념일 (3/3)"

사용자: "크리스마스 디데이 삭제해줘"
AI: "크리스마스 디데이를 삭제했어요."
```

## FAQ

**Q. 디데이는 최대 몇 개까지 등록할 수 있나요?**
최대 30개까지 등록할 수 있습니다. 30개가 가득 찬 경우 오래된 디데이를 삭제해야 새로 추가할 수 있습니다.

**Q. 매년 반복되는 기념일은 어떻게 설정하나요?**
채팅에서 "매년 반복으로 설정해줘"라고 하거나, 웹 등록 화면에서 **반복** 옵션을 활성화하면 됩니다. 설정된 날짜가 지나면 자동으로 다음 해 날짜로 업데이트됩니다.

**Q. 오늘 날짜를 기준으로 자동 계산되나요?**
네, 디데이 값은 매일 자동으로 재계산됩니다. 어제 D-10이었다면 오늘은 D-9로 표시됩니다.

**Q. 디데이 날짜를 수정할 수 있나요?**
웹에서 해당 디데이를 클릭하면 수정 화면에서 날짜, 이름, 아이콘, 설명을 변경할 수 있습니다.

**Q. 과거 날짜를 디데이로 등록하면 어떻게 표시되나요?**
과거 날짜는 D+N 형식으로 표시됩니다. 예를 들어, 연인과 처음 만난 날이나 회사 입사일 같이 경과 일수를 기념하고 싶을 때 유용하게 활용할 수 있습니다.

**Q. 주간 리포트에 디데이가 자동으로 포함되나요?**
네, 주간 리포트 생성 시 다가오는 디데이(D-30 이내)가 자동으로 포함됩니다. 리포트를 통해 한 주를 시작하기 전에 중요한 날짜를 미리 확인할 수 있습니다.
