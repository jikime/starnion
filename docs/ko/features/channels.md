# 채널

## 개요

채널 관리 페이지에서는 Starnion의 AI 비서 니온(Nion)과 대화할 수 있는 **커뮤니케이션 채널**들을 설정하고 관리합니다. 현재 **Telegram Bot**과 **Web Chat** 두 가지 채널을 지원하며, 각 채널의 연결 상태를 한눈에 확인할 수 있습니다.

> 채널은 니온과 소통하는 통로입니다. Telegram에서든 웹에서든, 어디서나 동일한 AI 경험을 제공합니다.

---

## 지원 채널

| 채널 | 설명 | 상태 표시 |
|------|------|----------|
| **Telegram Bot** | Telegram 메신저를 통한 대화 | 연결됨 / 연결 끊김 |
| **Web Chat** | 웹 브라우저 대시보드 내 채팅 | 항상 사용 가능 |

---

## Telegram 봇 연결하기

### 1단계: BotFather에서 봇 생성

1. Telegram에서 [@BotFather](https://t.me/BotFather)를 검색합니다.
2. `/newbot` 명령을 입력합니다.
3. 봇 이름과 사용자명을 설정합니다.
4. 발급받은 **API 토큰**을 복사합니다.

### 2단계: Starnion에 토큰 등록

1. 웹 대시보드 사이드바에서 **설정** 메뉴로 이동합니다.
2. **Telegram** 섹션에서 봇 토큰을 입력합니다.
3. **연결** 버튼을 클릭합니다.
4. 연결 성공 시 채널 페이지에서 "연결됨" 상태로 표시됩니다.

### 3단계: 대화 시작

Telegram에서 생성한 봇을 검색한 뒤 `/start` 명령을 입력하면 니온과의 대화가 시작됩니다.

---

## 채널 상태 확인

채널 페이지에서는 각 채널의 현재 상태를 실시간으로 확인할 수 있습니다.

| 상태 | 의미 |
|------|------|
| **연결됨** (초록) | 정상적으로 작동 중 |
| **연결 끊김** (빨강) | 토큰 만료 또는 설정 오류 |

연결이 끊긴 경우 설정 페이지에서 토큰을 재입력하거나, 봇을 다시 생성해야 합니다.

## FAQ

**Q. Telegram 봇이 응답하지 않아요.**

봇 토큰이 올바른지 확인하고, 설정 페이지에서 연결 상태를 확인해 주세요. 토큰이 만료된 경우 BotFather에서 `/token` 명령으로 재발급받을 수 있습니다.

**Q. 여러 Telegram 계정에서 같은 봇을 사용할 수 있나요?**

하나의 Starnion 계정에는 하나의 Telegram 봇만 연결할 수 있습니다. 다른 Telegram 계정에서 동일 봇에 메시지를 보내도 첫 번째 연결된 계정으로만 동작합니다.

**Q. Web Chat과 Telegram 대화 기록이 공유되나요?**

네, 모든 채널의 대화는 동일한 대화 기록에 저장됩니다. Telegram에서 한 대화를 웹에서 이어서 할 수 있습니다.
