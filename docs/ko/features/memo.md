# 메모

## 개요

메모 기능은 기억해두어야 할 정보를 빠르게 저장하고 태그로 분류해 관리하는 간편 노트 시스템입니다. 일기가 감정과 일상을 기록하는 개인 저널이라면, 메모는 정보와 사실을 보관하는 개인 데이터베이스입니다.

**주요 특징:**
- 태그 기반 분류로 빠른 검색
- 최대 100개까지 저장 가능
- AI가 관련 메모를 자동으로 찾아줌
- 제목 + 내용 구조로 간단명료하게 관리

> 메모는 기억 스킬(memory)과 다릅니다. 메모는 명시적으로 저장하고 조회하는 구조화된 정보 저장소이고, 기억은 AI가 대화 맥락을 자동으로 학습하는 장기 메모리입니다.

## 메모 작성하기

### 웹에서 작성

1. 사이드바에서 **메모** 메뉴를 클릭합니다.
2. 오른쪽 상단의 **+ 새 메모** 버튼을 클릭합니다.
3. 다음을 입력합니다:
   - **제목** (필수): 메모를 빠르게 식별할 수 있는 이름
   - **내용** (선택): 메모의 상세 내용
   - **태그** (선택): 분류 카테고리 (기본값: 개인)
4. **저장** 버튼을 클릭합니다.

저장된 메모는 최근 수정일 기준으로 목록에 표시됩니다.

### 채팅으로 작성

채팅창에서 AI에게 메모 저장을 요청하면 즉시 저장됩니다.

**예시 대화:**

> 사용자: "우유랑 계란 사야 해, 메모해줘"
>
> AI: "메모 저장했어요! '장보기 목록'으로 기록해 뒀어요."

> 사용자: "회의록 저장해줘 — 다음 주 화요일 출시 예정, 마케팅팀 승인 필요"
>
> AI: "회의록 메모를 저장했어요. 태그는 '업무'로 설정했어요."

> 사용자: "새 아이디어 메모 — 앱에 다크모드 추가하면 좋겠다"
>
> AI: "아이디어 메모를 저장했어요! '아이디어' 태그로 기록했어요."

**저장 트리거 표현:**
- "메모해줘", "기록해줘", "저장해줘"
- "메모 남겨줘 —", "노트해줘"
- "회의록 저장", "X 적어줘"

## 태그 시스템

태그는 메모를 분류하고 나중에 빠르게 찾기 위한 핵심 기능입니다.

### 기본 태그

메모 저장 시 태그를 지정하지 않으면 자동으로 **개인** 태그가 적용됩니다. AI는 메모 내용을 분석해 적절한 태그를 자동으로 제안합니다:

| 내용 예시 | 자동 태그 |
|----------|----------|
| 회의 내용, 프로젝트 관련 | 업무 |
| 장보기, 음식 메모 | 개인 |
| 새로운 아이디어, 기획 | 아이디어 |
| 공부 내용, 강의 노트 | 학습 |
| 여행 계획, 방문 장소 | 여행 |

### 태그 추가 방법

**웹에서:** 태그 입력란에 직접 입력 후 저장
**채팅에서:** "업무 태그로 메모해줘" / "아이디어 태그로 저장해줘"

### 태그로 검색

**웹에서:** 메모 목록 상단의 태그 필터 버튼 클릭 → 원하는 태그 선택
**채팅에서:**

> 사용자: "업무 메모만 보여줘"
>
> AI: "업무 태그 메모 목록이에요:\n1. 회의록 (3/5)\n2. 프로젝트 계획 (3/3)\n3. 마감 일정 (2/28)"

## 메모 검색

### 키워드 검색 (웹)

웹 메모 화면의 검색창에서 제목이나 내용에 포함된 키워드로 검색할 수 있습니다. 대소문자를 구분하지 않으며, 한글도 부분 일치로 검색됩니다.

**예시:** "마케팅" 검색 → 제목이나 내용에 "마케팅"이 포함된 모든 메모 표시

### AI를 통한 검색 (채팅)

AI에게 찾고 싶은 메모를 설명하면 관련 메모를 찾아줍니다. 정확한 키워드가 기억나지 않아도 됩니다.

> 사용자: "지난번에 저장한 회의 관련 메모 찾아줘"
>
> AI: "회의 관련 메모를 찾았어요:\n1. 3/5 회의록 — 다음 주 출시 예정\n2. 2/28 주간 회의 — 마케팅팀 피드백"

> 사용자: "프로젝트 관련 메모 다 보여줘"
>
> AI: "프로젝트 관련 메모 3개를 찾았어요:\n..."

> 사용자: "음식 관련 메모 있어?"
>
> AI: "음식 관련 메모가 있어요:\n- 장보기 목록: 우유, 계란\n- 맛집 리스트: 파스타집 (홍대)"

## AI 활용 예시

메모 기능을 AI와 함께 활용하는 다양한 방법을 소개합니다.

### 빠른 메모 저장

```
사용자: "나중에 볼 링크 저장해줘 — https://example.com/article"
AI: "링크 메모를 저장했어요. 제목은 '참고 링크'로 설정했어요."

사용자: "김철수 전화번호 010-1234-5678 메모해줘"
AI: "연락처 메모를 저장했어요. '개인' 태그로 기록했어요."
```

### 구조화된 메모 저장

```
사용자: "회의록 저장해줘:
  참석자: 김팀장, 이대리, 박사원
  안건: Q2 마케팅 전략
  결정사항: SNS 광고 예산 200만원 증액"
AI: "회의록을 저장했어요! '업무' 태그로 기록했어요."
```

### 메모 조회

```
사용자: "메모 보여줘"
AI: "최근 메모 5개예요:
  1. 회의록 (업무) — 3/8
  2. 장보기 목록 (개인) — 3/7
  3. 여행 준비물 (여행) — 3/5
  4. 아이디어 노트 (아이디어) — 3/3
  5. 공부 요약 (학습) — 3/1"

사용자: "여행 메모 자세히 보여줘"
AI: "[여행 준비물 메모]
  여권, 충전기, 여행자 보험증서, 현지 유심..."
```

### 메모 삭제

```
사용자: "장보기 목록 메모 삭제해줘"
AI: "장보기 목록 메모를 삭제했어요."
```

## 팁 & FAQ

**Q. 메모와 일기는 어떻게 구분해서 사용하면 좋나요?**

| 상황 | 권장 기능 |
|------|----------|
| "오늘 기분이 어땠고 무슨 일이 있었어" | 일기 |
| "나중에 참고할 정보 저장" | 메모 |
| "회의 내용, 할 일 목록" | 메모 |
| "감정적인 하루 돌아보기" | 일기 |

**Q. 메모는 최대 몇 개까지 저장할 수 있나요?**
최대 100개까지 저장할 수 있습니다. 100개가 꽉 찬 경우 오래된 메모를 삭제해야 새 메모를 저장할 수 있습니다.

**Q. 태그를 여러 개 붙일 수 있나요?**
현재는 메모당 태그 하나만 지정할 수 있습니다. 여러 주제에 걸친 내용이라면 내용 안에 키워드를 포함해두면 검색 시 찾기 수월합니다.

**Q. 채팅에서 저장한 메모가 웹에서도 보이나요?**
네, 채팅과 웹은 동일한 데이터를 공유합니다. 채팅에서 저장한 메모는 웹 메모 목록에서도 확인하고 수정할 수 있습니다.

**Q. 메모를 수정할 수 있나요?**
웹에서 메모를 클릭하면 수정 화면으로 이동합니다. 채팅에서는 현재 수정 기능을 직접 지원하지 않으므로, 기존 메모를 삭제하고 새로 저장하는 방법을 사용하세요.

**Q. 메모 내용이 AI 기억에도 저장되나요?**
메모 자체는 AI 장기 메모리에 직접 저장되지 않습니다. 다만 채팅 중 메모를 조회하거나 내용을 언급하면, 해당 대화 맥락이 AI 기억에 영향을 줄 수 있습니다. 명시적으로 기억시키려면 "이걸 기억해줘"라고 요청하세요.
