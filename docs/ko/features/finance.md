# 가계부 & 예산

## 개요

Starnion의 가계부는 **자연어로 말하듯 기록하는 AI 가계부**입니다. "점심 김치찌개 9천원"이라고 입력하면 AI가 카테고리를 자동 분류하고 날짜와 금액을 파악해 즉시 기록합니다. 영수증 사진을 찍어서 첨부하는 것도 가능합니다.

복잡한 앱을 배울 필요 없이, 평소 대화하듯 기록하고 "이번 달 얼마 썼어?"라고 물으면 분석까지 해 줍니다. 카테고리별 예산을 설정하면 초과 시 경고를 받을 수 있고, 소비 패턴 분석을 통해 절약 인사이트도 제공합니다.

## 지출/수입 입력

### 자연어로 바로 입력

채팅창에 자연스럽게 입력하면 됩니다. 형식을 지킬 필요가 없습니다.

**지출 입력 예시:**

```
사용자: 점심 김치찌개 9000원
AI:    식비 / 점심 김치찌개 / 9,000원 기록했어요 🍜 (3월 8일)

사용자: 카페 아메리카노 4500원 카드
AI:    카페 / 아메리카노 / 4,500원 기록했어요 ☕ (3월 8일, 카드)

사용자: 버스 1250원
AI:    교통 / 버스 / 1,250원 기록했어요 🚌 (3월 8일)

사용자: 쿠팡에서 샴푸 12800원 결제함
AI:    쇼핑 / 샴푸 / 12,800원 기록했어요 🛒 (3월 8일)

사용자: 넷플릭스 월정액 17000원
AI:    구독 / 넷플릭스 / 17,000원 기록했어요 📺 (3월 8일)

사용자: 병원 내과 진료비 5000원
AI:    의료 / 내과 진료비 / 5,000원 기록했어요 🏥 (3월 8일)

사용자: 어제 친구랑 치킨 먹었어, 내가 32000원 냈어
AI:    식비 / 치킨 / 32,000원 기록했어요 🍗 (3월 7일)
```

**수입 입력 예시:**

```
사용자: 월급 들어왔어. 2850000원
AI:    수입 / 월급 / 2,850,000원 기록했어요 💰 (3월 8일)

사용자: 프리랜서 작업비 받았어 450000원
AI:    수입 / 프리랜서 / 450,000원 기록했어요 (3월 8일)

사용자: 중고나라에서 의자 팔아서 30000원 받음
AI:    수입 / 중고판매 / 30,000원 기록했어요 (3월 8일)
```

**여러 건 한 번에 입력:**

```
사용자: 오늘 지출 정리
        - 편의점 5200원
        - 점심 8500원
        - 커피 4000원
        - 지하철 1250원

AI:    4건 기록했어요 ✅
       식비 / 편의점 / 5,200원
       식비 / 점심 / 8,500원
       카페 / 커피 / 4,000원
       교통 / 지하철 / 1,250원
       ────────────────
       오늘 총 지출: 18,950원
```

**영수증 사진으로 입력:**

```
[영수증 이미지 첨부]
사용자: 이 영수증 가계부에 추가해줘

AI:    영수증을 분석했어요 🧾
       - 스타벅스 아메리카노 Tall 5,500원
       - 블루베리 머핀 4,800원
       총 10,300원을 카페 항목으로 기록할까요?

사용자: 응
AI:    카페 / 스타벅스 / 10,300원 기록 완료! (3월 8일)
```

### 날짜 지정 입력

날짜를 명시하면 해당 날짜로 기록합니다.

```
사용자: 그저께 마트에서 장봐서 67,400원 썼어
AI:    쇼핑 / 마트 / 67,400원 (3월 6일)로 기록했어요 🛒

사용자: 지난 월요일 헬스장 3개월 등록했어. 270,000원
AI:    문화 / 헬스장 등록 / 270,000원 (3월 3일)로 기록했어요 💪

사용자: 2월 28일에 외식비 45000원
AI:    식비 / 외식 / 45,000원 (2월 28일)로 기록했어요 🍽️
```

## 카테고리 시스템

지출을 입력하면 AI가 내용을 분석해 자동으로 카테고리를 분류합니다.

### 기본 카테고리

| 카테고리 | 포함 항목 예시                                     |
|----------|---------------------------------------------------|
| 식비     | 점심, 저녁, 마트 장보기, 편의점 음식, 배달음식      |
| 카페     | 커피숍, 카페, 음료, 디저트                          |
| 교통     | 지하철, 버스, 택시, 주유, 고속도로 통행료           |
| 쇼핑     | 의류, 전자제품, 생활용품, 온라인쇼핑                |
| 구독     | 넷플릭스, 유튜브 프리미엄, 음악 스트리밍, 앱 구독   |
| 의료     | 병원 진료비, 약국, 건강검진, 의료용품               |
| 문화     | 영화, 공연, 전시, 헬스장, 취미 활동                 |
| 기타     | 위 카테고리에 해당하지 않는 항목                    |

### 커스텀 카테고리

기본 카테고리 외에도 직접 카테고리를 지정할 수 있습니다.

```
사용자: 반려동물 사료 35000원 "펫" 카테고리로
AI:    펫 / 사료 / 35,000원 기록했어요 🐾

사용자: 영어학원 월 수업료 150000원, 카테고리는 교육
AI:    교육 / 영어학원 / 150,000원 기록했어요 📚
```

### 카테고리 자동 분류 규칙

```
입력: "스타벅스 아메리카노 5500원"
  ↓
AI 분석:
  - "스타벅스" → 카페 브랜드 인식
  - "아메리카노" → 음료 종류 인식
  → 카테고리: 카페 (신뢰도: 높음)

입력: "약국 타이레놀 8200원"
  ↓
AI 분석:
  - "약국" → 의료 관련 장소
  - "타이레놀" → 의약품
  → 카테고리: 의료 (신뢰도: 높음)

입력: "OO마트 물건 삼"
  ↓
AI 분석:
  - "마트" → 쇼핑 관련
  - 구체적 품목 불명
  → 카테고리: 쇼핑 (신뢰도: 보통)
  AI: "어떤 물건을 구매하셨나요? 카테고리를 더 정확하게 분류해 드릴게요."
```

## 예산 설정

카테고리별로 월간 예산 한도를 설정해 소비를 관리할 수 있습니다.

### 채팅으로 예산 설정

```
사용자: 식비 예산 40만원으로 설정해줘
AI:    식비 예산을 400,000원으로 설정했어요.
       현재 이번 달 식비 지출: 342,500원 (85.6% 사용)
       남은 예산: 57,500원

사용자: 카페는 10만원, 교통은 15만원으로 해줘
AI:    예산 설정 완료!
       카페:  100,000원
       교통:  150,000원

사용자: 전체 예산 현황 알려줘
AI:    🔧 `budget_status` 실행 중...

       📋 3월 예산 현황
       ──────────────────────────────────
       카테고리  예산       사용    잔여    비율
       식비      400,000   342,500  57,500  85.6% ⚠️
       카페      100,000   127,000 -27,000 127.0% ❌ 초과!
       교통      150,000    89,400  60,600  59.6% ✅
       쇼핑      200,000    67,400 132,600  33.7% ✅
       ──────────────────────────────────
       전체      850,000   626,300 223,700  73.7%
```

### 웹 UI에서 예산 설정

설정 > 예산 관리 메뉴에서 각 카테고리별 예산을 직접 입력할 수 있습니다.

- **경고 임계값**: 예산 대비 소비 비율이 이 값을 넘으면 주의 표시 (기본: 70%)
- **위험 임계값**: 예산 대비 소비 비율이 이 값을 넘으면 위험 표시 (기본: 90%)

```
예산 설정 예시:

카테고리  월 예산
──────────────────
식비      400,000원
카페      100,000원
교통      150,000원
쇼핑      200,000원
구독       70,000원
의료       50,000원
문화       80,000원
기타      100,000원
──────────────────
합계    1,150,000원
```

## 예산 알림

예산을 초과하거나 임계값에 근접하면 알림을 받을 수 있습니다.

### 알림 동작 방식

```
[70% 경고 - 주의 단계]
AI 알림: "식비 예산의 70%를 사용했어요.
          이번 달 남은 예산: 120,000원
          앞으로 22일 동안 하루 5,455원 이내로 쓰면 돼요."

[90% 경고 - 위험 단계]
AI 알림: "⚠️ 교통 예산의 92%를 사용했어요!
          남은 예산이 12,000원밖에 없어요.
          이번 달 교통 지출을 줄여보세요."

[100% 초과]
AI 알림: "❌ 카페 예산을 초과했어요.
          예산: 100,000원 / 지출: 127,000원 (27,000원 초과)
          이번 달 카페 예산을 재검토해보세요."
```

### Telegram 알림 연동

Telegram 채널을 연결하면 예산 알림을 텔레그램으로도 받을 수 있습니다. 외출 중에도 실시간으로 소비 현황을 파악할 수 있습니다.

### 가계부를 더 잘 활용하는 팁

1. **그때그때 기록하기**: 지출 직후 바로 입력하면 잊지 않고 정확하게 기록할 수 있습니다.

2. **영수증 사진 활용**: 큰 지출이나 항목이 많은 경우 영수증을 사진으로 찍어 첨부하면 한 번에 여러 건을 자동 입력할 수 있습니다.

3. **월초 예산 설정**: 매월 초에 카테고리별 예산을 설정해두면 지출 관리가 훨씬 쉬워집니다.

4. **주간 리뷰 습관**: 매주 한 번 "이번 주 지출 요약해줘"라고 물어보면 소비 패턴을 빠르게 파악할 수 있습니다.

5. **인사이트 활용**: 정기적으로 "내 소비 패턴 분석해줘"를 요청하면 절약 포인트를 발견할 수 있습니다.

## 자주 묻는 질문

**Q. 금액을 잘못 입력했어요. 어떻게 수정하나요?**

A. "방금 입력한 [항목] [금액] 수정해줘"라고 말하면 됩니다. 또는 웹 UI의 거래 목록에서 직접 클릭해서 수정할 수도 있습니다.

```
사용자: 아까 입력한 편의점 5200원 4200원으로 수정해줘
AI:    편의점 지출을 5,200원 → 4,200원으로 수정했어요.
```

**Q. 카테고리 자동 분류가 틀렸어요.**

A. "카테고리 [원래 카테고리]에서 [원하는 카테고리]로 바꿔줘"라고 말하면 즉시 수정됩니다. AI가 자주 틀리는 패턴이 있다면 피드백을 주시면 개선에 반영됩니다.

**Q. 이전 달 지출을 뒤늦게 입력할 수 있나요?**

A. 네, 날짜를 명시하면 됩니다. "2월 15일 마트 32,000원"처럼 입력하면 해당 날짜로 기록됩니다.

**Q. 지출 데이터를 엑셀로 내보낼 수 있나요?**

A. 현재 CSV 내보내기 기능을 준비 중입니다. "이번 달 지출 내역 보내줘"라고 요청하면 텍스트 형식으로 받을 수 있습니다.

**Q. 공동 지출(가족, 커플) 관리도 되나요?**

A. 현재는 개인 계정 기반으로 운영되며, 공유 가계부 기능은 추후 업데이트 예정입니다.

**Q. 예산을 초과하면 어떤 알림을 받나요?**

A. 예산의 70%(경고), 90%(위험), 100% 초과(긴급) 시점에 각각 알림을 받습니다. Telegram을 연결하면 모바일로도 알림을 받을 수 있습니다.

**Q. 수입은 어떻게 기록하나요?**

A. "월급 300만원 들어왔어" 또는 "수입 300만원"처럼 입력하면 자동으로 수입으로 분류됩니다. 금액이 양수이면 수입, 음수이면 지출로 처리됩니다.

**Q. 구독 서비스 정기결제를 자동으로 기록할 수 있나요?**

A. 현재는 수동 입력이 기본이지만, 정기적으로 동일한 금액이 결제되는 경우 "매달 넷플릭스 17,000원 자동 입력해줘"라고 설정하면 매월 자동으로 기록됩니다.
