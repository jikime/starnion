# 목표 관리

## 개요

Starnion의 목표 관리 기능은 일상의 크고 작은 목표를 체계적으로 추적하고 달성을 도와주는 AI 기반 시스템입니다. 단순히 할 일을 나열하는 것이 아니라, AI가 목표 달성 과정을 함께 따라가며 격려하고 진행 상황을 분석합니다.

지원하는 목표 유형:
- **할 일/프로젝트** — 기한이 있는 단발성 과제 (보고서 제출, 프로젝트 완료)
- **습관** — 매일 반복하는 행동 (운동, 독서, 명상)
- **저축 목표** — 특정 금액 모으기 (여행 자금, 비상금)
- **지출 제한** — 예산 내 지출 유지 (이번 달 식비 30만원)
- **일반 목표** — 위 분류에 속하지 않는 개인 목표

## 목표 만들기

### 웹에서 만들기

1. 사이드바에서 **목표** 메뉴를 클릭합니다.
2. **+ 새 목표** 버튼을 클릭합니다.
3. 아래 항목을 입력합니다:
   - **목표명** (필수): 달성하려는 목표를 간략하게
   - **아이콘**: 목표를 상징하는 이모지 (기본값 🎯)
   - **카테고리**: 건강, 재정, 자기계발, 업무 등
   - **목표 수치**: 달성 기준이 되는 숫자 (예: 30회, 10만원)
   - **단위**: 회, 일, 원, km 등
   - **시작일 / 종료일**: 기간 설정
   - **설명**: 목표에 대한 부연 설명 (선택)

### 채팅으로 만들기

자연어로 이야기하면 AI가 자동으로 목표를 설정합니다. 별도 확인 없이 즉시 저장됩니다.

**예시 대화:**

> 사용자: "이번 달 독서 목표 5권으로 설정해줘"
>
> AI: "목표 저장했어요! '이번 달 독서 5권' 목표를 세웠어요. 응원할게요 📚"

> 사용자: "매일 아침 운동하기로 했어"
>
> AI: "훌륭한 습관 목표네요! '매일 아침 운동' 목표를 저장했어요. 오늘부터 시작이에요 💪"

> 사용자: "이번 달 식비 30만원 이내로 쓸 거야"
>
> AI: "재정 목표를 저장했어요! 이번 달 식비를 30만원 이하로 유지하는 목표예요. 함께 관리해볼게요 👍"

**트리거 패턴** — 다음 표현이 포함되면 자동으로 목표가 생성됩니다:

| 표현 예시 | 목표 유형 |
|----------|----------|
| "X 하기로 했어", "X 목표 세워줘" | 일반/습관 |
| "매일 X 해야지" | 습관 |
| "이번 주 X 끝내야 해" | 할 일 |
| "X 절약할 거야", "X 이내로 쓸 거야" | 재정 |
| "X 모으기로 했어" | 저축 |

## 체크인 시스템

체크인은 습관형 목표에서 특히 중요합니다. 매일 목표 달성 여부를 기록해 연속 달성(스트릭)을 쌓을 수 있습니다.

### 체크인 방법

**웹에서 체크인:**
1. 목표 목록에서 해당 목표 카드를 찾습니다.
2. 카드 하단의 **오늘 체크인** 버튼을 클릭합니다.
3. 체크인 완료 표시와 함께 스트릭이 업데이트됩니다.

**채팅으로 체크인:**

> 사용자: "오늘 운동 했어"
>
> AI: "훌륭해요! 운동 목표 오늘 체크인 완료했어요. 현재 스트릭: 7일 연속 🔥"

> 사용자: "독서 목표 오늘 체크"
>
> AI: "체크인 완료! 이번 달 독서 진행 현황: 3/5권 (60%)"

### 스트릭 카운터

스트릭은 연속으로 체크인한 날의 수입니다. 어제 체크인을 놓쳤더라도 오늘 체크인하면 스트릭이 유지됩니다 (1일 유예).

| 스트릭 | 상태 |
|--------|------|
| 0일 | 시작 전 또는 리셋 |
| 1~6일 | 시작 중 |
| 7~29일 | 진행 중 🔥 |
| 30일 이상 | 습관 완성 🏆 |

체크인을 2일 이상 연속으로 놓치면 스트릭이 0으로 초기화됩니다.

## 목표 평가

### 진행률 업데이트

수치형 목표(저축, 지출 등)는 직접 진행률을 업데이트할 수 있습니다.

> 사용자: "여행 저축 목표 50% 달성했어"
>
> AI: "절반을 달성했네요! 여행 저축 목표를 50%로 업데이트했어요. 목표까지 절반 남았어요 ✈️"

> 사용자: "운동 목표 진행률 70%야"
>
> AI: "대단해요! 운동 목표 진행률을 70%로 업데이트했어요. 이번 달 안에 달성할 수 있을 것 같아요!"

### 목표 상태 변경

목표의 상태는 3가지입니다:

| 상태 | 의미 | 변경 표현 |
|------|------|----------|
| `진행 중` | 현재 도전 중 | 기본 상태 |
| `달성` | 목표 완료 | "달성했어", "성공했어", "완료했어" |
| `포기` | 더 이상 진행 안 함 | "포기할래", "그만둘게", "취소할게" |

> 사용자: "독서 목표 5권 다 읽었어!"
>
> AI: "축하해요! 🎉 이번 달 독서 목표를 달성하셨어요! 5권 모두 읽으셨군요. 정말 대단해요!"

## 알림

목표 관련 알림은 주간 리포트를 통해 전달됩니다. 설정된 종료일이 다가올수록 채팅에서 자동으로 상기시켜 드립니다.

**자동 알림 상황:**
- 종료일 7일 전: "목표 마감까지 7일 남았어요"
- 종료일 1일 전: "내일이 목표 마감일이에요"
- 스트릭 위기: "어제 체크인을 놓쳤어요, 오늘 체크인하면 스트릭이 유지돼요"

---

## 주간 리포트와의 연동

매주 생성되는 주간 리포트에는 목표 현황 섹션이 포함됩니다:

```
[이번 주 목표 현황]
진행 중인 목표: 3개
- 매일 운동 (스트릭: 5일) — 이번 주 5/7일 체크인
- 독서 5권 (진행률: 60%) — 3/5권 완료, 마감 18일 후
- 식비 절약 (진행률: 45%) — 예산 13만 5천원 남음

이번 주 달성: 없음
이번 주 신규: 독서 5권 (3/1 시작)
```

## 목표 예시

### 운동 습관

```
목표명: 매일 30분 운동
카테고리: 건강
아이콘: 🏃
목표 수치: 30 (일)
단위: 일
시작일: 2025-03-01
종료일: 2025-03-31
```

채팅: "오늘 조깅 30분 했어" → 자동 체크인

### 독서 목표

```
목표명: 이번 달 책 5권 읽기
카테고리: 자기계발
아이콘: 📚
목표 수치: 5 (권)
단위: 권
종료일: 이번 달 말
```

채팅: "토지 1권 다 읽었어" → 진행률 업데이트

### 저축 목표

```
목표명: 유럽 여행 자금 모으기
카테고리: 재정
아이콘: ✈️
목표 수치: 3,000,000 (원)
단위: 원
종료일: 2025-12-31
```

채팅: "이번 달 저축 50만원 했어" → 진행률 업데이트

## 💬 텔레그램 사용 예시

> 니온에게 자연어로 말하면 됩니다. 정해진 명령어 없이 일상 언어로 대화하세요!

### 목표 생성

```
매일 운동하기 목표 만들어줘
```
→ 습관 추적 목표가 생성됩니다

```
이번 달 책 3권 읽기 목표 설정해줘
```
→ 수치 기반 목표가 생성됩니다

### 진행률 업데이트

```
오늘 운동 완료!
```
→ 운동 목표에 오늘 체크인이 기록됩니다

```
"데미안" 다 읽었어
```
→ 읽기 목표 진행률이 업데이트됩니다

### 목표 조회

```
내 목표 진행 상황 알려줘
```
→ 현재 활성 목표들의 진행률을 보여줍니다

```
달성한 목표 보여줘
```
→ 완료된 목표 목록을 보여줍니다

## 팁 & FAQ

**Q. 목표는 최대 몇 개까지 만들 수 있나요?**
진행 중인 목표는 최대 20개까지 동시에 관리할 수 있습니다. 달성하거나 포기한 목표는 제한이 없습니다.

**Q. 스트릭이 끊기면 처음부터 다시 시작해야 하나요?**
네, 스트릭이 끊기면 0부터 다시 시작됩니다. 하지만 당일 체크인 후 어제 체크인을 놓친 경우, 내일 체크인하면 스트릭이 보존됩니다 (1일 유예 정책).

**Q. 채팅에서 목표를 설정하면 수치 정보도 자동으로 설정되나요?**
"매일 운동" 같은 습관 목표는 자동으로 단위가 '일'로 설정됩니다. "책 5권 읽기" 같이 수치가 명시된 경우 목표 수치도 자동으로 설정됩니다.

**Q. 완료된 목표는 어떻게 확인하나요?**
웹 목표 화면 상단의 필터에서 **완료**를 선택하면 달성한 목표 목록을 볼 수 있습니다. 채팅에서는 "달성한 목표 보여줘"라고 하면 됩니다.

**Q. 재정 목표와 가계부 기능이 겹치지 않나요?**
재정 목표는 목표 금액을 설정하고 큰 그림을 추적하는 용도입니다. 가계부는 일별 지출을 상세하게 기록하는 기능입니다. 두 기능은 서로 보완 관계로, 가계부 데이터가 재정 목표 달성 여부 분석에 활용될 수 있습니다.
