# 예산 관리

## 개요

예산 관리 기능은 **월별, 카테고리별 예산을 설정하고 실시간으로 모니터링**하는 기능입니다. 예산 대비 실제 지출을 프로그레스 바로 시각화하여, 한눈에 소비 상황을 파악할 수 있습니다. 예산의 80%를 초과하면 경고 알림을 통해 과지출을 방지합니다.

> 예산을 설정하면 가계부 기록과 자동으로 연동됩니다. 니온에게 "이번 달 식비 얼마 남았어?"라고 물어보세요.

---

## 예산 설정하기

### 웹에서 설정

1. 사이드바에서 **예산 관리** 메뉴를 클릭합니다.
2. **+ 예산 추가** 버튼을 클릭합니다.
3. 아래 정보를 입력합니다:
   - **카테고리** (필수): 식비, 교통, 쇼핑, 여가, 의료, 교육 등
   - **월 예산 금액** (필수): 해당 카테고리의 월별 예산
4. **저장** 버튼을 클릭합니다.

총 예산은 각 카테고리 예산의 합계로 자동 계산됩니다.

### 채팅으로 설정

AI에게 자연어로 말하면 예산을 설정할 수 있습니다.

**예시 대화:**

> 사용자: "이번 달 식비 예산 30만원으로 설정해줘"
>
> AI: "식비 예산을 300,000원으로 설정했어요! 현재 식비 지출은 128,000원으로, 예산의 43%를 사용했어요."

> 사용자: "교통비 예산 10만원 추가해줘"
>
> AI: "교통비 예산을 100,000원으로 추가했어요. 이번 달 총 예산은 400,000원이에요."

## 예산 현황 보기

### 대시보드 화면

예산 관리 페이지에서는 각 카테고리별로 다음 정보를 확인할 수 있습니다:

| 표시 항목 | 설명 |
|----------|------|
| **카테고리명** | 예산이 설정된 카테고리 |
| **프로그레스 바** | 예산 대비 지출 비율 시각화 |
| **지출 금액 / 예산** | 현재 지출 / 설정 예산 (예: 128,000 / 300,000원) |
| **사용률 (%)** | 예산 대비 지출 백분율 |
| **상태 아이콘** | 정상(초록), 주의(주황), 초과(빨강) |

### 프로그레스 바 색상

| 사용률 | 색상 | 의미 |
|--------|------|------|
| 0–60% | 초록 | 안전 구간 |
| 60–80% | 주황 | 주의 구간 |
| 80–100% | 빨강 | 경고 구간 |
| 100%+ | 진한 빨강 | 초과 |

## 예산 초과 경고

예산의 80% 이상을 사용하면 자동으로 경고를 받을 수 있습니다.

### 경고 방식

- **웹 대시보드**: 해당 카테고리의 프로그레스 바가 빨간색으로 변경
- **채팅 알림**: 가계부 기록 시 예산 초과 가능성을 AI가 안내
- **주간 리포트**: 주간 리포트에 예산 현황 포함

### 경고 예시

> 사용자: "오늘 저녁 3만원 썼어"
>
> AI: "저녁 식사 30,000원을 식비에 기록했어요. 참고로, 이번 달 식비 예산(300,000원)의 87%를 사용했어요. 남은 예산이 39,000원이에요. 조금 조절하면 좋을 것 같아요!"

## 예산 수정 및 삭제

### 웹에서 수정

- 예산 항목을 클릭하면 수정 화면이 열립니다.
- 금액을 변경하고 **저장**을 클릭합니다.

### 채팅으로 수정

> 사용자: "식비 예산 40만원으로 올려줘"
>
> AI: "식비 예산을 400,000원으로 수정했어요. 현재 사용률이 32%로 줄었어요."

### 삭제

> 사용자: "교통비 예산 삭제해줘"
>
> AI: "교통비 예산을 삭제했어요. 총 예산이 300,000원으로 변경됐어요."

## 💬 텔레그램 사용 예시

> 니온에게 자연어로 말하면 됩니다. 정해진 명령어 없이 일상 언어로 대화하세요!

### 예산 설정

```
이번 달 식비 예산 30만원으로 설정해줘
```
→ 식비 카테고리에 300,000원 예산이 설정됩니다

```
쇼핑 예산 15만원 추가
```
→ 쇼핑 카테고리 예산이 추가됩니다

### 예산 현황 확인

```
예산 현황 알려줘
```
→ 전체 카테고리별 예산 대비 지출 현황을 보여줍니다

```
식비 얼마나 남았어?
```
→ 식비 카테고리의 남은 예산을 알려줍니다

### 예산 수정

```
교통비 예산 8만원으로 줄여줘
```
→ 교통비 예산이 80,000원으로 변경됩니다

## FAQ

**Q. 예산은 매달 자동으로 초기화되나요?**

네, 예산은 매월 1일에 자동으로 초기화됩니다. 지출 금액만 리셋되며, 설정한 예산 금액은 유지됩니다.

**Q. 카테고리별 예산을 설정하지 않고 총 예산만 설정할 수 있나요?**

현재는 카테고리별 예산 설정만 지원합니다. "기타" 카테고리에 총 예산을 설정하는 방식으로 활용할 수 있습니다.

**Q. 예산을 초과하면 지출 기록이 차단되나요?**

아니요, 예산 초과 시에도 지출 기록은 정상적으로 저장됩니다. 초과 경고만 표시되며, 기록 자체는 제한하지 않습니다.

**Q. 가계부 데이터와 자동 연동되나요?**

네, 가계부에 기록된 지출이 해당 카테고리 예산에 자동으로 반영됩니다. 별도의 연동 설정이 필요 없습니다.
