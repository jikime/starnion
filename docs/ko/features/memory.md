# AI 기억

## 개요

AI 기억은 Nion이 대화를 통해 자동으로 학습한 패턴과 인사이트를 시각화하는 기능입니다. 사용자가 직접 입력하는 메모와 달리, AI가 대화 맥락을 분석하여 스스로 생성하는 장기 메모리입니다.

**주요 특징:**
- 대화 패턴에서 자동으로 추출된 인사이트
- 시간대별 감정 및 행동 패턴 분석
- 주간 대화 요약 및 주제 추이 파악
- 저장된 기억 항목 직접 삭제 가능

> AI 기억은 메모와 다릅니다. 메모는 사용자가 명시적으로 저장하는 정보이고, AI 기억은 Nion이 대화를 분석하여 자동으로 생성하는 장기 학습 데이터입니다.

---

## 세 가지 탭

### 대화 인사이트

AI가 최근 대화를 분석하여 추출한 인사이트 목록입니다.

**표시 정보:**
- **날짜 목록**: 분석이 완료된 날짜들 (왼쪽 사이드바 또는 드롭다운)
- **인사이트 카드**: 각 분석 항목의 요약 내용
- **감정 지표**: 해당 시점의 전반적인 감정 상태
- **주제 태그**: 대화에서 자주 등장한 주요 주제

**인사이트 유형:**
| 유형 | 설명 |
|------|------|
| 대화 패턴 | 자주 하는 말투, 질문 방식 등 |
| 관심 주제 | 반복적으로 언급되는 관심사 |
| 감정 흐름 | 대화 중 감정 변화 추이 |
| 행동 지표 | 루틴, 습관, 자주 하는 행동 |

### 행동 패턴

대화 데이터를 기반으로 분석된 행동 및 선호도 패턴입니다.

**표시 정보:**
- **패턴 카드**: 각 행동 패턴의 제목과 설명
- **카테고리 배지**: 패턴의 분류 (생활, 건강, 재정, 관계 등)
- **신뢰도 지표**: AI 분석의 신뢰도 수준

**패턴 카테고리:**
| 카테고리 | 예시 |
|----------|------|
| 생활 습관 | 수면 패턴, 식사 시간 등 |
| 감정 패턴 | 스트레스 요인, 긍정 요인 |
| 재정 행동 | 소비 경향, 저축 패턴 |
| 사회 관계 | 대화 빈도, 관계 유형 |

### 주간 기억

매주 자동으로 생성되는 대화 요약 및 주요 사건 기록입니다.

**표시 정보:**
- **주간 요약**: 해당 주의 주요 대화 내용과 활동
- **주요 사건**: 특별히 기록된 중요한 순간들
- **삭제 기능**: 특정 주간 기억을 선택하여 삭제 가능

## AI 기억이 생성되는 조건

AI 기억 데이터는 자동으로 생성되며, 특정 조건이 충족되어야 분석이 실행됩니다.

### 대화 인사이트 생성 조건

- 텔레그램 또는 채팅에서 **30분~2시간** 이상 대화가 없는 경우 자동 분석 실행
- **일기 스킬**이 활성화되어 있어야 함
- 분석 주기: 약 10분마다 조건을 확인

### 행동 패턴 생성 조건

- **매일 오전 6시**에 전날까지의 데이터를 분석
- **7일 이상**의 대화 데이터가 쌓여야 의미 있는 패턴 생성
- 재정 데이터가 있는 경우 소비 패턴도 함께 분석

### 주간 기억 생성 조건

- **매주 월요일 오전 5시**에 이전 주 데이터 자동 정리
- **30일 이상**의 대화 기록이 있어야 주간 압축 실행

> **데이터가 보이지 않는다면?** 위 조건이 아직 충족되지 않았을 수 있습니다. 텔레그램이나 채팅에서 자연스럽게 대화를 나누다 보면 AI가 점차 패턴을 학습하기 시작합니다.

## 데이터 삭제

저장된 AI 기억 항목은 직접 삭제할 수 있습니다.

### 삭제 방법

**주간 기억 삭제:**
1. **주간 기억** 탭을 클릭합니다.
2. 삭제할 주차 항목의 **삭제** 버튼을 클릭합니다.
3. 확인 팝업에서 **삭제** 버튼을 클릭합니다.

**대화 인사이트 / 행동 패턴 삭제:**
1. 해당 탭에서 삭제할 항목의 **삭제** 아이콘을 클릭합니다.
2. 즉시 삭제됩니다.

> 삭제된 기억은 복구할 수 없습니다. 신중하게 삭제하세요.

## 팁 & FAQ

**Q. AI 기억과 메모의 차이는 무엇인가요?**

| 항목 | AI 기억 | 메모 |
|------|--------|------|
| 생성 방식 | AI가 자동 생성 | 사용자가 직접 입력 |
| 데이터 출처 | 대화 분석 결과 | 사용자 입력 내용 |
| 용도 | 장기 패턴 학습 | 정보 저장 및 조회 |
| 수정 | 불가 (삭제만 가능) | 웹에서 수정 가능 |

**Q. 왜 AI 기억에 아무 데이터도 없나요?**
AI 기억은 충분한 대화 데이터가 쌓인 후에 생성됩니다. 텔레그램이나 채팅에서 꾸준히 대화를 나누면 2~7일 후부터 데이터가 생성되기 시작합니다.

**Q. 일기 스킬이 없으면 대화 인사이트가 생성되지 않나요?**
네, 대화 인사이트는 일기 스킬이 활성화된 사용자에게만 생성됩니다. 설정 > 스킬에서 일기 스킬을 활성화하세요.

**Q. 기억을 삭제하면 AI가 다시 학습하나요?**
삭제된 기억은 복구되지 않으며, 해당 데이터 기반의 패턴은 더 이상 표시되지 않습니다. 단, 이후 새로운 대화가 쌓이면 AI가 새로 학습하여 기억을 다시 생성합니다.

**Q. AI 기억 데이터는 안전한가요?**
모든 AI 기억 데이터는 사용자 계정에 종속되어 저장되며, 다른 사용자와 공유되지 않습니다. 언제든지 직접 삭제할 수 있습니다.
