# 데이터 가든

## 개요

데이터 가든은 Starnion에 저장된 모든 데이터를 **정원 생태계로 시각화**하는 대시보드입니다. 가계부, 목표, 일기 감정, 파일, 스킬 등 각종 데이터가 살아있는 정원의 구성 요소로 표현됩니다.

> 저축률이 높을수록 나무가 튼튼해지고, 지출이 과해지면 구름에서 비가 내립니다.

## 하늘 배경 — 감정 팔레트

최근 7일 일기 감정에 따라 하늘 색상이 변합니다.

| 감정 | 하늘 색상 | 분위기 |
|------|---------|-------|
| 매우좋음 | 황금 그라디언트 | 따뜻한 노을빛 |
| 좋음 | 파란 그라디언트 | 맑고 차분한 하늘 |
| 보통 | 보라 그라디언트 | 신비로운 우주 |
| 나쁨 | 짙은 보라 | 어두운 밤하늘 |
| 매우나쁨 | 심우주 블랙 | 칠흑 같은 밤 |

## 중앙: 자산 나무

정원의 핵심 요소. 저축률에 따라 나무 상태가 변합니다.

### 건강 단계

| 저축률 | 나무 상태 | 가지 색상 | 열매 |
|-------|---------|---------|-----|
| ≥ 30% | 건강 | 색상 유지 | 별 모양 최대 5개 |
| 15–29% | 시들기 시작 | 채도 감소 + 처짐 | 열매 감소 |
| < 15% | 심각 | 회갈색 + 강한 처짐 | 열매 없음 |

### 페르소나 오라

활성화된 AI 페르소나에 따라 나무 뿌리 주변에 컬러 오라가 나타납니다.

| 페르소나 | 오라 색상 |
|---------|---------|
| assistant | 은백색 |
| finance | 초록 |
| buddy | 핑크 |
| coach | 주황 |
| analyst | 파랑 |
| counselor | 보라 |

### 예산 팝업 (나무 클릭)

나무를 클릭하면 예산 상세 팝업이 열립니다.

| 표시 정보 | 설명 |
|---------|------|
| **잔여 예산** | 이번 달 남은 예산 (컬러 코딩: 녹색/주황/빨강) |
| **예산 별자리** | 카테고리를 별로 배치한 SVG 시각화 |
| **총 예산 / 지출** | 숫자 및 진행 바 |
| **지출률 / 저축률** | 백분율 표시 |
| **상태 메시지** | 예산 건강 상태에 따른 맥락 메시지 |

## 지출 구름 (우상단)

지출이 예산을 초과한 카테고리가 있을 때 비가 내립니다.

- **비 없음**: 모든 카테고리 예산 내 유지
- **비 내림**: 초과 카테고리에서 애니메이션 빗방울 표시
- **가위(✂️) 애니메이션**: 과지출 시 나무에 가위가 등장

클릭하면 초과 카테고리 목록과 각 통계를 팝오버로 확인할 수 있습니다.

## 목표 꽃밭 (하단 좌측)

진행 중인 목표를 튤립 꽃으로 표현합니다. 최대 4개 꽃이 표시됩니다.

### 성장 단계

| 단계 | 진행률 | 모양 |
|------|-------|------|
| 씨앗 | 0–10% | 반짝이는 점 하나 |
| 새싹 | 10–30% | 줄기 + 꽃봉오리 |
| 성장 | 30–55% | 줄기 + 잎 2장 |
| 개화 | 55–80% | 꽃잎 3장 피어남 |
| 만개 | 80–100% | 완전 개화 + 흔들림 애니메이션 |

클릭하면 전체 목표 목록과 달성률을 팝오버로 볼 수 있습니다.

---

## 감정 씨앗 (하단 좌측)

최근 7일 일기의 감정을 작은 씨앗으로 표현합니다.

- 최대 7개 씨앗
- 각 씨앗은 감정 색상으로 표시
- 활짝 피는 블룸 애니메이션

## 특수 효과

### 수입 별가루

수입이 기록되면 하늘에서 금빛 별이 떨어집니다.

- 별 개수: `수입 ÷ 200,000`원, 최대 12개
- 별가루(Star Dust) 낙하 애니메이션
- 하늘의 수입 별(Income Star)도 수입 규모에 비례 생성

### 유성

D-Day가 7일 이내로 남으면 유성이 대각선으로 날아갑니다.

- 최대 3개 유성
- 각도 35°, 4.5초 주기
- D-Day 수에 비례

## 데이터 흐름

마음 정원 진입 시 아래 12개 API를 동시에 호출합니다.

| API 엔드포인트 | 용도 |
|-------------|------|
| `/api/budget` | 예산 카테고리 및 지출 |
| `/api/goals` | 목표 진행률 |
| `/api/diary?limit=7` | 최근 7일 감정 |
| `/api/finance/summary` | 수입 및 저축률 |
| `/api/dday` | D-Day 목록 |
| `/api/images?limit=50` | 이미지 파일 수 |
| `/api/audios?limit=50` | 오디오 파일 수 |
| `/api/documents` | 문서 파일 수 |
| `/api/skills` | 활성 스킬 수 |
| `/api/usage?days=30` | 월간 토큰 사용량 |
| `/api/profile/persona` | 현재 페르소나 |
| `/api/settings/providers` | 연동 서비스 수 |

## FAQ

**Q. 나무가 회색으로 변했어요.**

저축률이 15% 미만일 때 나무가 시들어 보입니다. 지출을 줄이거나 수입을 늘려 저축률을 높이면 회복됩니다.

**Q. 구름에서 비가 계속 내려요.**

예산을 초과한 카테고리가 있을 때 발생합니다. 구름을 클릭해 어느 카테고리가 초과됐는지 확인하고 예산을 조정해 보세요.

**Q. 꽃이 4개밖에 안 보여요.**

목표 꽃밭은 진행 중인 목표를 최대 4개까지 표시합니다. 전체 목표를 보려면 꽃밭을 클릭하거나 목표 메뉴로 이동하세요.

**Q. 에너지 충전기가 빨간색이에요.**

월간 토큰 사용량이 1,000만 토큰의 80% 이상일 때 빨간색으로 표시됩니다. 채팅 사용 빈도를 조절하면 충전 상태가 개선됩니다.

**Q. 하늘 색상이 어두워요.**

최근 7일 일기의 감정이 나쁨 또는 매우나쁨으로 기록된 경우 하늘이 어두워집니다. 니온에게 마음을 털어놓거나 일기에 오늘의 기분을 적어보세요.

**Q. 유성은 언제 나타나나요?**

7일 이내에 도래하는 D-Day가 있을 때 유성이 나타납니다. D-Day 메뉴에서 일정을 등록하면 됩니다.
