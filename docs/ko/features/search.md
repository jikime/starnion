# 검색

Starnion은 두 가지 검색 방식을 제공합니다. **로컬 검색**은 내 데이터(일기, 메모, 가계부 등)를 빠르게 찾고, **웹 검색**은 인터넷에서 최신 정보를 가져옵니다. 두 방식 모두 AI 채팅을 통해 자연어로 사용할 수도 있습니다.

---

## 상단 검색바

화면 상단의 검색바를 이용하면 타이핑과 동시에 내 데이터를 빠르게 검색할 수 있습니다.

### 사용 방법

1. 화면 상단의 검색 아이콘(🔍) 또는 검색 입력란을 클릭합니다.
2. 검색어를 입력합니다.
3. 실시간으로 결과가 나타납니다.
4. 결과를 클릭하면 해당 항목으로 이동합니다.

### 검색 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+K` (Windows/Linux) | 검색창 열기 |
| `Cmd+K` (macOS) | 검색창 열기 |
| `ESC` | 검색창 닫기 |
| `↑` `↓` | 결과 탐색 |
| `Enter` | 선택한 항목으로 이동 |

## 로컬 검색

내 데이터에서 검색합니다. 외부 인터넷 연결 없이 빠르게 검색됩니다.

### 검색 대상

| 데이터 종류 | 검색 가능 내용 | 예시 |
|-----------|------------|------|
| **일기** | 날짜, 내용, 감정 | "지난 주 일기", "슬펐던 날" |
| **메모** | 제목, 내용, 태그 | "파이썬 메모", "#업무 태그" |
| **가계부** | 항목명, 금액, 날짜, 카테고리 | "삼겹살", "5만원 이상 지출" |
| **문서** | 파일명, 문서 내용 | "계약서", "2024년 보고서" |
| **대화 내역** | 이전 채팅 내용 | "저번에 물어봤던 레시피" |

### 검색 예시

```
검색어: "삼겹살"
결과:
  📒 가계부  2024-03-15  삼겹살 (친구들) 32,000원
  📒 가계부  2024-02-28  삼겹살집 야식 18,000원
  📔 일기    2024-03-15  "친구들이랑 삼겹살 먹었다..."

검색어: "파이썬"
결과:
  📝 메모  파이썬 기초 정리  #공부 #개발
  📄 문서  Python_tutorial.pdf
  💬 대화  "파이썬 리스트 컴프리헨션 사용법 알려줘..."
```

## 웹 검색

Starnion은 **Tavily API**와 **네이버 검색 API**를 통해 인터넷에서 최신 정보를 검색합니다.

### Tavily 웹 검색

AI 친화적으로 설계된 검색 API로, 복잡한 질문에도 관련성 높은 결과를 제공합니다.

**활성화 방법:** 서버 관리자가 `.env`에 `TAVILY_API_KEY`를 설정하면 `websearch` 스킬이 자동으로 활성화됩니다.

**사용 예시:**

```
나: 2024년 최신 AI 모델 비교해줘
봇: [웹 검색 중...]
    인터넷 검색 결과를 정리했습니다.

    GPT-4o (OpenAI): 멀티모달 지원, 빠른 응답...
    Gemini 1.5 Pro (Google): 100만 토큰 컨텍스트...
    Claude 3.5 Sonnet (Anthropic): 코딩 성능 우수...

    출처: techcrunch.com, arxiv.org ...
```

### 네이버 검색

한국어 콘텐츠를 위한 검색입니다. 블로그, 뉴스, 쇼핑, 지식iN 등을 검색합니다.

**활성화 방법:** `.env`에 `NAVER_CLIENT_ID`와 `NAVER_CLIENT_SECRET` 설정 후, 설정 > 스킬에서 `naver_search` 스킬을 활성화합니다.

**지원 검색 유형:**

| 유형 | 설명 | 예시 요청 |
|------|------|---------|
| 블로그 | 네이버 블로그 검색 | "김치찌개 맛있게 끓이는 법 블로그에서 찾아줘" |
| 뉴스 | 최신 뉴스 | "오늘 국내 경제 뉴스" |
| 쇼핑 | 상품·가격 비교 | "에어팟 프로 최저가 알려줘" |
| 지식iN | Q&A 검색 | "지식iN에서 전세 계약 주의사항 찾아줘" |
| 지역 | 지역 정보 | "강남 맛집 찾아줘" |
| 책 | 도서 검색 | "클린코드 책 정보 알려줘" |

## 검색 히스토리

최근 검색어와 결과를 확인할 수 있습니다.

- 상단 검색바에서 검색 후 결과가 자동으로 기록됩니다.
- **설정 > 검색 히스토리** 에서 전체 기록을 조회하거나 삭제할 수 있습니다.
- 검색 히스토리도 AI 채팅에서 참조할 수 있습니다.

## AI 채팅으로 검색하기

상단 검색바 외에도, 채팅창에서 자연어로 검색 요청을 할 수 있습니다. AI가 로컬 검색과 웹 검색을 적절히 조합하여 답변합니다.

### 로컬 데이터 검색 (memory 스킬)

내 기록을 자연어로 조회합니다.

```
나: 지난달에 뭐 먹었지?
봇: 지난 달(2024년 2월) 식비 내역입니다.
    - 02-03: 점심 (편의점) 8,500원
    - 02-08: 저녁 (삼겹살집) 22,000원
    - 02-14: 발렌타인 디너 (이탈리안) 67,000원
    ...
    총 식비: 312,400원

나: 파이썬 관련 메모 찾아줘
봇: 파이썬 관련 메모 3개를 찾았습니다.
    1. "파이썬 기초 정리" (2024-01-15, #공부)
    2. "FastAPI 사용법" (2024-02-03, #개발)
    3. "pandas 데이터프레임 정리" (2024-02-18, #공부)

나: 저번에 천장 수리 견적 얼마였다고 했더라?
봇: 2024년 1월 15일 대화에서 찾았습니다.
    "천장 수리 견적 250만원" 이라고 기록되어 있습니다.
```

### 웹 검색 (websearch / naver_search 스킬)

```
나: 요즘 전세사기 예방법 알려줘
봇: [웹 검색 중...]
    최신 전세사기 예방 정보를 정리했습니다.

    1. 전입신고 + 확정일자 반드시 받기
    2. 전세가율 80% 이상이면 위험 신호
    3. 등기부등본에서 근저당 확인
    ...

나: 강남에 점심 먹을 만한 곳 추천해줘
봇: [네이버 지역 검색 중...]
    강남 맛집을 검색했습니다.

    1. OO 한정식 - 블로그 리뷰 2,400건, 평점 4.7
    2. XX 스시 - 점심 특선 15,000원, 예약 필요
    ...
```

### 로컬 + 웹 혼합 검색

```
나: 내가 기록한 이달 식비랑, 서울 평균 식비 비교해줘
봇: 이달 내 식비를 조회하고, 서울 평균 식비를 웹에서 검색했습니다.

    내 이달 식비: 287,000원
    서울 직장인 평균 월 식비: 약 350,000~400,000원 (출처: ○○연구원)

    내 식비가 서울 평균보다 약 20% 절약되고 있습니다.
```

## 시맨틱 검색

Starnion의 검색은 단순 키워드 매칭이 아니라 **의미(시맨틱) 기반** 검색을 지원합니다.

### 시맨틱 검색이란?

- 입력한 단어와 **정확히 일치하지 않아도** 의미가 비슷하면 검색됩니다.
- 텍스트를 벡터(임베딩)로 변환하여 PostgreSQL(pgvector)에 저장하고, 검색 시 유사한 벡터를 찾습니다.

### 시맨틱 검색 예시

```
검색어: "밥 먹은 거"
실제 매칭: "점심 식사", "저녁 외식", "삼겹살집", "이탈리안 레스토랑"
→ "밥"이라는 단어가 없어도 식사 관련 기록을 찾아냄

검색어: "돈 나간 것"
실제 매칭: "지출", "결제", "12,000원", "구매"
→ "돈"이라는 단어가 없어도 금전 관련 기록을 찾아냄

검색어: "기분 좋았던 날"
실제 매칭: "오늘 정말 신났어", "행복한 하루", "기분 최고"
→ 감정 표현을 의미로 파악하여 관련 일기를 찾아냄
```

### 시맨틱 검색이 적용되는 데이터

| 데이터 | 임베딩 적용 | 비고 |
|--------|:----------:|------|
| 일일 대화 로그 | ✓ | 매 대화 후 자동 저장 |
| 가계부 항목 | ✓ | 항목명 + 카테고리 |
| 일기 | ✓ | 전문 임베딩 |
| 메모 | ✓ | 제목 + 내용 |
| 문서 섹션 | ✓ | 문서 청크 단위 |
| 지식 베이스 | ✓ | 패턴 분석 결과 |

## FAQ

**Q. 로컬 검색은 얼마나 빠른가요?**
대부분의 검색은 100ms 이내에 결과를 반환합니다. 시맨틱 벡터 검색의 경우 데이터 양에 따라 다소 시간이 걸릴 수 있지만 통상 1초 이내입니다.

**Q. 검색 결과가 엉뚱하게 나올 때는 어떻게 하나요?**
더 구체적인 키워드를 입력하거나, AI 채팅에서 "2024년 2월에 기록한 점심 식비" 처럼 날짜와 조건을 함께 명시하면 더 정확한 결과를 얻을 수 있습니다.

**Q. 웹 검색 결과는 저장되나요?**
웹 검색 결과 자체는 저장되지 않습니다. 단, 채팅 대화 내역으로는 저장됩니다. 이후 "저번에 찾아줬던 전세사기 예방법 다시 알려줘"와 같이 AI에게 요청하면 대화 내역에서 찾아줍니다.

**Q. 오래된 데이터도 검색되나요?**
네. 모든 데이터는 영구적으로 저장되며 검색 가능합니다. 다만 오래된 데이터일수록 관련성 점수가 낮게 평가될 수 있습니다.

**Q. 파일 내용을 검색하려면 어떻게 해야 하나요?**
PDF나 Word 파일을 Starnion에 업로드하면 `documents` 스킬이 내용을 파싱하여 벡터 DB에 저장합니다. 이후 상단 검색바나 AI 채팅에서 파일 내용으로 검색할 수 있습니다.
