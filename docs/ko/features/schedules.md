# 일정 & 알림

## 개요

Starnion의 일정 & 알림 기능은 반복 알림과 일회성 알림을 설정하고 중요한 일정을 놓치지 않도록 도와주는 기능입니다. "매일 오전 8시에 운동 알림 줘"처럼 채팅으로 자연스럽게 말하면 알림이 바로 등록됩니다.

시스템이 자동으로 보내는 알림(예산 초과 경고, 디데이 알림, 목표 달성 현황 등)도 있고, 직접 만드는 개인 알림도 있습니다. 모든 알림은 텔레그램으로 전송되며, 웹 채팅에서도 확인할 수 있습니다.

> 알림은 하루 최대 3회까지 발송되며, 오후 10시 ~ 오전 8시(KST) 사이의 조용한 시간대에는 전송되지 않습니다.

## 알림 만들기

### 채팅으로 만들기

가장 쉬운 방법입니다. 채팅에서 원하는 알림을 자연어로 말하면 됩니다.

```
사용자: 매일 오전 8시에 운동 알림 줘
AI:    매일 오전 8시에 운동 알림을 등록했어요!
       "지금 운동할 시간이에요! 💪"

사용자: 매주 월요일 오전 9시에 주간 계획 세우라고 알려줘
AI:    매주 월요일 09:00에 알림을 등록했어요!
       "이번 주 계획을 세워보세요! 📅"

사용자: 매월 25일 오후 6시에 월세 납부 알림 줘
AI:    매월 25일 18:00에 알림을 등록했어요!
       "월세 납부일이에요! 🏠"

사용자: 3월 20일 오전 10시에 병원 예약 알림 줘
AI:    2025년 3월 20일 10:00에 일회성 알림을 등록했어요!
       "오늘 오전 10시에 병원 예약이 있어요! 🏥"
```

### 알림 관리 메뉴에서 만들기

상단 메뉴 > **스케줄** 탭 > **알림 추가** 버튼을 클릭하면 양식을 통해 알림을 등록할 수 있습니다.

## Cron 표현식

알림 스케줄은 Cron 표현식으로 관리됩니다. 직접 입력할 필요는 없지만, 원하는 패턴이 있다면 채팅에서 상세하게 요청하세요.

**기본 패턴 예시:**

| 패턴 | Cron 표현식 | 설명 |
|------|-------------|------|
| 매일 오전 8시 | `0 8 * * *` | 매일 08:00 |
| 매일 오후 9시 | `0 21 * * *` | 매일 21:00 |
| 매주 월요일 오전 9시 | `0 9 * * 1` | 월요일 09:00 |
| 매주 금요일 오후 6시 | `0 18 * * 5` | 금요일 18:00 |
| 매월 1일 오전 9시 | `0 9 1 * *` | 매월 1일 09:00 |
| 매월 말일 오후 8시 | `0 20 28-31 * *` | 매월 28~31일 20:00 |
| 평일(월~금) 오전 7시 | `0 7 * * 1-5` | 평일 07:00 |
| 주말(토,일) 오전 10시 | `0 10 * * 6,0` | 토·일 10:00 |
| 매 시간 정각 | `0 * * * *` | 매시 정각 |
| 매 15분 | `*/15 * * * *` | 15분마다 |

## 알림 종류

### 개인 알림 (사용자 생성)

```
사용자: 매일 저녁 10시에 일기 쓰라고 알림 줘
AI:    매일 22:00에 일기 알림을 등록했어요!

사용자: 2주마다 금요일 오후 2시에 정기 보고서 제출 알림 줘
AI:    격주 금요일 14:00에 알림을 등록했어요.
```

### 시스템 자동 알림

Starnion이 자동으로 보내는 알림입니다. 별도 설정 없이 조건이 충족되면 발송됩니다.

| 알림 | 발송 조건 | 시간 |
|------|-----------|------|
| 주간 리포트 | 지출 기록이 있는 사용자 | 매주 월요일 09:00 |
| 예산 경고 | 카테고리 예산 90% 초과 시 | 1시간마다 확인 |
| 일일 소비 요약 | 당일 지출 기록이 있는 사용자 | 매일 21:00 |
| 소비 이상 감지 | 평균 대비 급격한 지출 발생 시 | 3시간마다 확인 |
| 비활성 알림 | 3일 이상 미접속 시 | 매일 20:00 |
| 월말 결산 알림 | 매월 28~31일 | 매일 21:00 |
| 목표 달성 현황 | 목표가 있는 사용자 | 매주 수요일 12:00 |
| 패턴 분석 결과 | 분석 데이터 충분한 사용자 | 매일 14:00 |
| 디데이 알림 | 등록된 디데이가 있는 사용자 | 매일 08:00 |

## 시스템 자동 알림 예시

**예산 경고 알림:**
```
⚠️ 예산 알림

식비 예산을 거의 다 사용했어요!
사용: 81,200원 / 90,000원 (90%)
남은 금액: 8,800원
```

**목표 달성 현황 (매주 수요일):**
```
🎯 목표 달성 현황 (3월 12일 기준)

[여행 자금 모으기] 650,000원 / 1,000,000원 (65%)
[식비 절약] 이번 달 진행률: 78% - 잘 하고 있어요!
```

**디데이 알림:**
```
📅 디데이 알림

D-7: 팀 발표 (3월 15일)
D-30: 여름 휴가 출발 (4월 7일)
D+365: 입사 1주년 기념일 (어제였어요! 🎉)
```

**비활성 사용자 알림:**
```
안녕하세요! 며칠 동안 안 보이셨네요 😊
가계부 기록이나 궁금한 게 있으면 언제든지 말해주세요.
오늘 지출은 어떠셨나요?
```

## 알림 관리

### 알림 목록 조회

```
사용자: 내 알림 목록 보여줘
AI:    등록된 알림 목록:

       1. 운동 알림 - 매일 08:00 [활성]
       2. 주간 계획 - 매주 월요일 09:00 [활성]
       3. 월세 납부 - 매월 25일 18:00 [활성]
       4. 독서 알림 - 매일 22:30 [비활성]
```

### 알림 활성화/비활성화

```
사용자: 운동 알림 끄기
AI:    운동 알림을 비활성화했어요. 다시 켜려면 "운동 알림 켜줘"라고 해주세요.

사용자: 독서 알림 다시 켜줘
AI:    독서 알림을 다시 활성화했어요!
```

### 알림 수정

```
사용자: 운동 알림 시간을 오전 7시로 바꿔줘
AI:    운동 알림을 오전 7시로 변경했어요! (08:00 → 07:00)

사용자: 월세 알림 메시지를 "월세 납부하기! 이번 달도 잊지 말자"로 바꿔줘
AI:    알림 메시지를 수정했어요.
```

### 알림 삭제

```
사용자: 병원 예약 알림 삭제해줘
AI:    병원 예약 알림을 삭제했어요.

사용자: 모든 알림 목록 보여주고 독서 알림 지워줘
AI:    독서 알림을 삭제했어요.
```

## 텔레그램 알림

모든 알림은 텔레그램 메신저로 전송됩니다. 텔레그램을 사용하려면 먼저 연동이 필요합니다.

텔레그램 연동 방법:
1. Starnion 텔레그램 봇(`@starnion_bot`)을 검색합니다
2. `/start` 명령을 입력합니다
3. 연결 코드를 확인하고 웹에서 계정을 연동합니다

> 자세한 텔레그램 연동 방법은 [텔레그램 채널](../channels/telegram.md) 문서를 참고하세요.

## 실생활 알림 예시 10가지

```
1. 기상 알림
   사용자: 매일 오전 7시에 기상 알림 줘
   → 매일 07:00 "좋은 아침이에요! 오늘도 활기차게 시작해봐요! ☀️"

2. 수분 섭취 알림
   사용자: 2시간마다 물 마시라고 알려줘
   → 2시간마다 "물 한 잔 마실 시간이에요! 💧"

3. 약 복용 알림
   사용자: 매일 오전 8시, 오후 8시에 약 먹으라고 알림 줘
   → 매일 08:00, 20:00 "약 복용 시간이에요! 💊"

4. 점심 식비 체크
   사용자: 평일 오후 1시에 오늘 점심 가계부 기록했는지 확인해줘
   → 평일 13:00 "오늘 점심 지출 기록하셨나요? 가계부 한 번 확인해볼게요."

5. 저녁 운동 알림
   사용자: 평일 저녁 7시에 운동 알림 줘
   → 평일 19:00 "운동할 시간이에요! 오늘 목표 달성 화이팅! 💪"

6. 주간 청소 알림
   사용자: 매주 토요일 오전 10시에 청소하라고 알려줘
   → 매주 토요일 10:00 "주간 청소 타임! 깔끔한 주말 시작해요! 🧹"

7. 카드값 납부 알림
   사용자: 매월 15일 오전 9시에 카드값 확인하라고 알림 줘
   → 매월 15일 09:00 "이번 달 카드 청구액을 확인해보세요! 💳"

8. 독서 알림
   사용자: 매일 밤 11시에 독서 알림 줘
   → 매일 23:00 "잠들기 전 독서 시간이에요! 📚"

9. 주간 목표 점검
   사용자: 매주 일요일 저녁 9시에 이번 주 목표 점검하라고 알림 줘
   → 매주 일요일 21:00 "이번 주 목표 달성도를 점검해볼 시간이에요! ✅"

10. 연간 기념일 알림
    사용자: 매년 12월 31일 오전 9시에 연말 정산 알림 줘
    → 매년 12월 31일 09:00 "연말 정산 서류 준비하세요! 📋"
```

## 동작 방식

사용자가 등록한 스케줄은 `knowledge_base` 테이블에 JSON 형태로 저장됩니다. 시스템은 15분마다 활성 스케줄을 조회하여 현재 시각과 예약 시각이 15분 윈도우 안에 있으면 알림을 발송합니다.

```
스케줄 동작 흐름:

1. 등록     → AI가 채팅 요청을 파싱해 knowledge_base에 스케줄 저장
2. 점검     → 15분마다 활성 스케줄 전체 조회
3. 시각 확인 → 현재 시각이 예약 시각의 ±15분 윈도우 내인지 확인
4. 피로도 검사 → 조용한 시간대(22:00~08:00) 및 일일 최대(3회) 체크
5. 발송     → 텔레그램으로 메시지 전송
6. 상태 갱신 → 일회성 → 'completed', 반복 → last_sent 갱신
```

### 스케줄 타입

| 타입 | 설명 | 예시 |
|------|------|------|
| `one_time` | 지정 날짜·시각에 한 번만 | "3월 20일 10시에 병원 예약 알림" |
| `recurring` | 매일·매주·매월 반복 | "매주 월요일 9시에 주간 계획 알림" |

### 알림 발송 제한

자동 알림 과부하를 방지하기 위해 다음 조건에서는 알림이 차단됩니다:

- **조용한 시간대**: 오후 10시(22:00) ~ 오전 8시(08:00) KST
- **일일 한도**: 하루 최대 3회 (자동 알림 기준, 사용자 요청 응답 제외)
- **대화 중 차단**: 최근 1시간 내에 채팅 메시지를 보낸 경우 (사용자가 활성 상태이므로)

## FAQ

**Q. 알림을 몇 개까지 만들 수 있나요?**
A. 현재 개인 알림 수에 제한은 없습니다. 단, 너무 많은 알림은 관리가 어려울 수 있으니 필요한 것만 등록하는 것을 권장합니다.

**Q. 텔레그램 없이도 알림을 받을 수 있나요?**
A. 텔레그램 알림은 텔레그램 연동이 필요합니다. 웹 채팅을 통해서도 AI가 알림을 언급하지만, 능동적인 푸시 알림은 텔레그램을 통해서만 가능합니다.

**Q. 알림 발송 시간을 분 단위로 설정할 수 있나요?**
A. 가능합니다. "매일 오전 8시 30분에 알림 줘"처럼 분 단위로 지정할 수 있습니다.

**Q. 시스템 자동 알림(예산 경고, 소비 이상 감지 등)을 끌 수 있나요?**
A. 현재는 시스템 자동 알림을 개별적으로 끄는 기능이 제공되지 않습니다. 해당 기능(예산, 가계부 등)을 비활성화하면 관련 알림도 자동으로 중단됩니다.

**Q. 특정 날짜에만 한 번 알림을 보낼 수 있나요?**
A. 가능합니다. "3월 20일 오전 10시에 알림 줘"처럼 특정 날짜를 지정하면 일회성 알림이 등록됩니다. 해당 시간이 지나면 자동으로 비활성화됩니다.
