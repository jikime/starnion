# 리포트

## 개요

Starnion의 리포트 기능은 AI가 여러분의 일기, 가계부, 목표, 소비 패턴 데이터를 종합 분석하여 주간·월간 요약 보고서를 자동으로 생성하는 기능입니다.

매주 월요일 오전 9시(KST)에 지난 주 활동 요약이 텔레그램으로 자동 발송됩니다. "주간 리포트 만들어줘"라고 채팅에서 요청하면 언제든지 즉시 생성할 수도 있습니다. 생성된 모든 리포트는 리포트 메뉴에서 과거 기록까지 조회할 수 있습니다.

> 리포트는 나를 돌아보는 주간·월간 회고의 출발점입니다. AI가 먼저 데이터를 정리해 드리니, 내용을 확인하고 다음 주 계획을 세워보세요.

---

## 리포트 종류

| 종류 | 타입 | 설명 | 자동 생성 시각 (KST) |
|------|------|------|---------------------|
| **주간 리포트** | `weekly` | 지난 7일간 종합 분석 | 매주 월요일 09:00 |
| **일간 요약** | `daily` | 오늘 하루 지출·활동 요약 | 매일 21:00 (기록 있는 날) |
| **월간 리포트** | `monthly` | 이번 달 전체 분석 | 매월 마지막 날 21:00 |
| **소비 이상 감지** | `anomaly` | 평균 대비 200% 초과 지출 경보 | 3시간마다 자동 점검 |
| **패턴 인사이트** | `pattern` | 개인화된 소비·행동 패턴 통찰 | 매일 14:00 (조건 충족 시) |
| **목표 달성 현황** | `goal` | 활성 목표들의 진척도 요약 | 매주 수요일 12:00 |

## 자동 발송 리포트

### 주간 리포트 — 매주 월요일 09:00 KST

매주 월요일 오전 9시, 지난 한 주의 활동을 분석한 리포트가 텔레그램으로 자동 전송됩니다. 지난 30일 이내에 가계부 기록이 있는 사용자에게만 발송됩니다.

**주간 리포트 예시:**
```
📊 2025년 03월 2주차 주간 리포트

💰 이번 주 지출 총계: 187,400원

카테고리별 지출:
🍽 식비: 78,500원 (41.9%)
🚌 교통: 24,200원 (12.9%)
☕ 카페: 18,700원 (10.0%)
🛒 쇼핑: 42,000원 (22.4%)
🎬 여가: 24,000원 (12.8%)

지난 주 대비: +12,300원 (▲7.0%)
이번 달 누계: 312,800원

💡 인사이트
카페 지출이 지난 주보다 35% 늘었어요.
식비는 예산(90,000원)의 87%를 사용했습니다.
```

### 일간 요약 — 매일 21:00 KST

당일 지출 기록이 있는 사용자에게 오늘 하루 요약을 전송합니다.

```
📋 2025년 3월 8일 일간 요약

오늘 지출: 42,300원
- 아침 식사: 6,500원
- 점심 식사: 9,000원
- 카페: 4,500원
- 교통: 4,200원
- 저녁 식사: 18,100원

이번 주 누계: 127,400원 / 300,000원 (42.5%)
```

### 월간 리포트 — 매월 마지막 날 21:00 KST

매월 마지막 날 저녁, 이번 달 전체 지출 분석을 전송합니다. 이번 달 기록이 있는 사용자를 대상으로 합니다.

### 소비 이상 감지 — 3시간마다 자동 점검

오늘 지출이 최근 30일 일평균의 200%를 초과하면 즉시 알림을 보냅니다. 최소 7일 이상의 데이터가 축적된 경우에만 동작합니다.

**소비 이상 감지 예시:**
```
⚠️ 지출 이상 감지

오늘 총 지출: 185,000원
최근 30일 일평균: 31,200원 (593%)

평소보다 많이 사용하고 있어요. 한번 확인해 보세요!
```

### 패턴 인사이트 — 매일 14:00 KST

AI가 사용자의 소비 패턴을 학습하고, 조건이 충족되면 개인화된 통찰을 전송합니다. 패턴 분석은 매일 오전 6시에 백그라운드에서 먼저 실행됩니다.

```
💡 패턴 인사이트

매주 금요일 저녁 지출이 평소의 2.3배에요.
지난 4주간 금요일 평균 지출: 48,200원
(평일 평균: 21,000원)

금요일 저녁 예산을 따로 설정해보는 건 어떨까요?
```

### 목표 달성 현황 — 매주 수요일 12:00 KST

활성화된 목표가 있는 사용자에게 주간 진척도를 전송합니다.

```
🎯 목표 달성 현황 (3월 3주차)

✅ 독서 챌린지: 4/5권 (80%) — 1권만 더!
🔄 운동 습관화: 8/12회 (67%) — 이번 주 2회 달성 필요
💰 여행 저축: 150,000/200,000원 (75%)

전주 대비 종합 달성률: 74% (+8%)
이번 주도 화이팅이에요! 💪
```

## 수동 리포트 생성

자동 생성 주기를 기다리지 않고 즉시 리포트를 만들 수 있습니다.

### 채팅에서 요청

```
사용자: 주간 리포트 만들어줘
AI:    이번 주 리포트를 생성했어요. [리포트 내용 표시]

사용자: 이번 달 월간 리포트 만들어줘
AI:    3월 월간 리포트를 생성했어요.

사용자: 오늘 소비 패턴 분석해줘
AI:    패턴 인사이트 리포트를 생성했어요.

사용자: 저축 목표 달성 현황 알려줘
AI:    목표 달성 현황 리포트를 생성했어요.
```

### 리포트 메뉴에서 생성

상단 메뉴 > **리포트** 탭 > **리포트 생성** 버튼을 클릭하고, 원하는 리포트 종류를 선택하면 됩니다. 생성에는 약 20~60초 소요됩니다.

> 수동 생성된 리포트도 리포트 목록에 저장되어 나중에 다시 볼 수 있습니다.

## 리포트 상세 내용

### 주간 리포트 (weekly)

- 이번 주 총 지출액
- 카테고리별 지출 금액 및 비율
- 지난 주 대비 증감
- 이번 달 누계
- 예산 사용률 (예산 설정 시)
- AI 인사이트 및 절약 팁

### 월간 리포트 (monthly)

- 이번 달 총 지출액
- 카테고리별 월간 분석
- 전월 대비 비교
- 일별 지출 추이
- 최대 지출일/최소 지출일
- 절약 목표 달성 여부

### 패턴 인사이트 (pattern)

```
💡 소비 패턴 인사이트 (2025-03-08)

1. 카페 지출 패턴
   평일 오전(9-11시) 카페 방문 빈도가 높아요.
   주 평균 카페 지출: 23,500원

2. 주말 지출 특징
   주말 쇼핑 지출이 평일의 3.2배입니다.
   주말 전 쇼핑 목록 작성을 추천해요.

3. 절약 기회
   편의점 지출을 마트 구매로 바꾸면
   한 달 약 15,000원 절약 가능해요.
```

### 목표 달성 현황 (goal)

```
🎯 목표 달성 현황 (2025-03-08)

[목표 1] 비상금 100만원 모으기
현재: 650,000원 / 1,000,000원 (65%)
예상 달성일: 2025년 5월 15일

[목표 2] 식비 월 15만원 이내
3월 식비: 89,200원 / 150,000원 (59.5%)
이번 달 목표 달성 가능성: ✅ 높음
```

## 리포트 조회

### 리포트 목록

상단 메뉴 > **리포트** 탭에서 과거에 생성된 모든 리포트를 최신 순으로 볼 수 있습니다. 최대 20개씩 표시되며 더 불러오기 버튼으로 추가 조회할 수 있습니다.

### 리포트 필터

| 필터 | 설명 |
|------|------|
| 전체 | 모든 리포트 |
| 주간 | weekly 타입만 |
| 월간 | monthly 타입만 |
| 패턴 | pattern 타입만 |
| 목표 | goal 타입만 |
| 이상 감지 | anomaly 타입만 |

### 과거 리포트 조회

```
사용자: 지난 달 월간 리포트 보여줘
AI:    2025년 2월 월간 리포트:
       [2월 리포트 내용 표시]

사용자: 지난 주 주간 리포트랑 이번 주 비교해줘
AI:    지난 주(2월 4주) vs 이번 주(3월 1주) 비교:
       지출 총액: 175,200원 → 187,400원 (+12,200원)
       가장 늘어난 항목: 쇼핑 (+18,000원)
       가장 줄어든 항목: 식비 (-5,800원)
```

## 동작 방식

```
리포트 생성 흐름:

1. 트리거        → 스케줄러(자동) 또는 사용자 요청(수동)
2. 데이터 수집   → 일기, 가계부, 목표, 메모 데이터 조회
3. AI 분석       → gRPC로 에이전트에 분석 요청 (최대 120초)
4. 내용 생성     → LLM이 종합 보고서 텍스트 생성
5. 저장          → reports 테이블에 영구 저장
6. 전달          → 웹 UI에 표시 + 텔레그램으로 전송 (텔레그램 연동 시)
```

리포트 타입별 제목 형식:

| 타입 | 제목 예시 |
|------|---------|
| `daily` | `2025년 03월 08일 일간 요약` |
| `weekly` | `2025년 03월 2주차 주간 리포트` |
| `monthly` | `2025년 03월 월간 리포트` |
| `anomaly` | `2025-03-08 소비 이상 감지` |
| `pattern` | `2025-03-08 패턴 인사이트` |
| `goal` | `2025-03-08 목표 달성 현황` |

## API 엔드포인트

### 리포트 목록 조회

```
GET /api/v1/reports?user_id={user_id}&type={type}&limit=20&offset=0
```

| 파라미터 | 필수 | 설명 |
|----------|------|------|
| `user_id` | 필수 | 사용자 ID |
| `type` | 선택 | 리포트 종류 (daily, weekly, monthly, anomaly, pattern, goal) |
| `limit` | 선택 | 최대 조회 수 (기본 20) |
| `offset` | 선택 | 오프셋 (기본 0) |

**응답 예시:**
```json
[
  {
    "id": 42,
    "report_type": "weekly",
    "title": "2025년 03월 2주차 주간 리포트",
    "created_at": "2025-03-10 09:00"
  }
]
```

### 리포트 상세 조회

```
GET /api/v1/reports/{id}?user_id={user_id}
```

**응답 예시:**
```json
{
  "id": 42,
  "report_type": "weekly",
  "title": "2025년 03월 2주차 주간 리포트",
  "content": "📊 이번 주 지출 요약...",
  "created_at": "2025-03-10 09:00"
}
```

### 리포트 수동 생성

```
POST /api/v1/reports/generate
Content-Type: application/json

{
  "user_id": "...",
  "report_type": "weekly"
}
```

`report_type` 허용 값: `daily`, `weekly`, `monthly`, `anomaly`, `pattern`, `goal`

## 자동 발송 스케줄 요약

| 리포트 | 발송 시각 (KST) | 대상 조건 |
|--------|----------------|----------|
| 주간 리포트 | 매주 월요일 09:00 | 최근 30일 내 가계부 기록 있는 사용자 |
| 일간 요약 | 매일 21:00 | 당일 지출 기록 있는 사용자 |
| 월간 리포트 | 매월 마지막 날 21:00 | 이번 달 기록 있는 사용자 |
| 소비 이상 감지 | 3시간마다 자동 점검 | 일평균의 200% 초과 지출 발생 시 |
| 패턴 인사이트 | 매일 14:00 | 패턴 트리거 충족 시 |
| 목표 달성 현황 | 매주 수요일 12:00 | 활성 목표 있는 사용자 |

> 알림 과부하를 방지하기 위해 하루 최대 3회까지만 발송됩니다.
> 오후 10시 ~ 오전 8시(KST) 사이의 조용한 시간대에는 발송되지 않습니다.

## FAQ

**Q. 주간 리포트를 매주 월요일이 아닌 다른 요일에 받고 싶어요.**
A. 현재 주간 리포트 자동 발송은 매주 월요일 오전 9시로 고정되어 있습니다. 다른 시간에 받고 싶다면 채팅에서 직접 "주간 리포트 만들어줘"라고 요청해 주세요.

**Q. 텔레그램을 사용하지 않아도 리포트를 받을 수 있나요?**
A. 자동 발송은 텔레그램을 통해서만 이루어집니다. 웹에서는 리포트 메뉴에서 직접 생성하거나 채팅에서 요청할 수 있습니다.

**Q. 지출 기록이 없어도 리포트가 생성되나요?**
A. 지난 30일 이내에 지출 기록이 없으면 주간 리포트 자동 발송 대상에서 제외됩니다. 수동으로 요청하면 "기록이 없다"는 내용의 리포트가 생성될 수 있습니다.

**Q. 생성된 리포트는 얼마나 보관되나요?**
A. 생성된 모든 리포트는 영구 보관됩니다. 리포트 메뉴에서 과거 리포트를 언제든 조회할 수 있습니다.

**Q. 리포트 생성이 너무 느려요.**
A. AI가 모든 기록 데이터를 분석하는 과정에서 20~60초 정도 소요됩니다. 기록이 많을수록 처리 시간이 늘어날 수 있습니다.

**Q. 소비 이상 감지가 너무 자주 와요.**
A. 소비 이상 감지는 하루 최대 알림 횟수(3회) 내에서만 발송됩니다. 평소 지출이 커서 기준치가 낮다면, 가계부 기록을 지속적으로 쌓아 30일 평균이 실제 소비 패턴을 반영하게 되면 오탐이 줄어듭니다.
