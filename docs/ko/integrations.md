# 통합

| 연동 서비스 | 설명 |
|-----------|------|
| [Google Workspace](integrations/google.md) | Gmail, 캘린더, Drive 연동 |
| [Notion](integrations/notion.md) | 페이지 검색, 생성, 편집 |
| [GitHub](integrations/github.md) | 저장소, 이슈, PR 관리 |
| [Tavily](integrations/tavily.md) | 실시간 웹 검색 |
| [네이버 검색](integrations/naver-search.md) | 한국어 뉴스·블로그 검색 |
| [Gemini](integrations/gemini.md) | Google Gemini AI 모델 연동 |
