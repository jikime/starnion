# 스킬 시스템

스킬(Skill)은 Starnion AI 에이전트의 능력 모듈입니다. 각 스킬은 특정 기능을 담당하는 독립된 도구로, 필요에 따라 켜고 끌 수 있습니다.

---

## 스킬이란?

AI가 단순히 대화만 하는 것이 아니라 실제로 무언가를 **실행**할 수 있도록 해주는 기능 단위입니다.

예를 들어:
- `날씨 어때?` → **weather** 스킬이 날씨 API를 호출
- `점심 12,000원` → **finance** 스킬이 가계부에 기록
- `영수증 사진` → **image** 스킬이 금액을 추출 → **finance** 스킬이 자동 기록

사용자가 명시적으로 스킬 이름을 언급하지 않아도, AI가 문맥을 파악하여 적절한 스킬을 자동으로 선택합니다.

---

## 스킬 활성화 / 비활성화

**설정 > 스킬(Skills)** 메뉴에서 각 스킬을 개별적으로 켜고 끌 수 있습니다.

- 비활성화된 스킬은 AI가 요청을 받아도 해당 기능을 실행하지 않습니다.
- 일부 핵심 스킬(시스템 스킬)은 비활성화할 수 없습니다.
- 스킬 설정은 계정별로 저장됩니다.

---

## 스킬 상세 패널

**설정 > 스킬**의 각 스킬 카드에는 **ⓘ** 버튼이 있습니다. 클릭하면 사이드 패널이 열리며 다음 정보를 확인할 수 있습니다:

- **설명** — 스킬이 하는 일
- **트리거 키워드** — 메시지에 이 단어가 포함되면 스킬이 자동으로 활성화됨
- **사용 예시** — 채팅에 바로 보낼 수 있는 복사 가능한 예시

상세 패널에서 스킬을 바로 활성화하거나 비활성화할 수도 있습니다.

## 자연어로 스킬 호출하기

스킬을 직접 명령하지 않아도 됩니다. 자연스럽게 말하면 AI가 적절한 스킬을 선택합니다.

```
나: 달러 5만원으로 바꾸면 얼마야?
봇: 현재 환율(1,340원/달러) 기준으로 50,000원 = 약 37.31달러입니다.

나: 런던 지금 몇 시야?
봇: 런던(GMT) 현재 시간은 오전 6:42입니다.

나: 오늘 기분이 너무 안 좋아. 팀장한테 혼났어.
봇: 오늘 힘드셨겠네요. 일기에 기록해 두었습니다.

나: 내일 오후 3시에 병원 예약 잊지 말라고 알려줘.
봇: 내일 오후 3시에 "병원 예약" 알림을 설정했습니다.

나: 이 이미지에 뭐가 있어? (사진 첨부)
봇: 이미지를 분석했습니다. 커피숍 영수증으로 보이며, 아메리카노 4,500원이 결제되었습니다.

나: https://news.ycombinator.com 요약해줘.
봇: 해당 페이지를 요약했습니다. 현재 주요 게시물: ...
```

## 스킬 권한 안내

일부 스킬은 외부 API 키 또는 OAuth 인증이 필요합니다.

| 스킬 | 필요한 설정 |
|------|-----------|
| `weather` | 날씨 API 키 (환경변수 설정) |
| `websearch` | Tavily API 키 |
| `naver_search` | 네이버 Client ID / Secret |
| `google` | Google OAuth2 인증 완료 |
| `image` | OpenAI API 키 (이미지 생성 시) |
| `currency` | 환율 API 키 |
| `notion` | Notion 인테그레이션 토큰 |
| `github` | GitHub 개인 액세스 토큰 |
| `coding_agent` | Claude Code CLI (서버 환경) |

API 키 설정은 서버 관리자가 `.env` 파일에서 구성합니다.

## 자주 묻는 질문

**Q. 비활성화한 스킬의 데이터는 삭제되나요?**
아니요. 스킬을 비활성화해도 기존 데이터는 그대로 유지됩니다. 다시 활성화하면 이전 데이터를 그대로 사용할 수 있습니다.

**Q. 시스템 스킬은 무엇인가요?**
`memory`, `proactive` 등 에이전트의 핵심 동작에 필요한 스킬입니다. 이 스킬들은 비활성화할 수 없습니다.

**Q. 스킬을 여러 개 동시에 사용할 수 있나요?**
네. 하나의 메시지로 여러 스킬이 순서대로 실행될 수 있습니다. 예를 들어 영수증 이미지를 보내면 `image` 스킬이 금액을 인식하고, `finance` 스킬이 자동으로 가계부에 기록합니다.
