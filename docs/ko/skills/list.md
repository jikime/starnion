# 스킬 목록

Starnion에는 현재 **34개**의 스킬이 내장되어 있습니다. 각 스킬은 독립된 모듈로, 필요에 따라 개별적으로 활성화하거나 비활성화할 수 있습니다.

---

## 재무 (Finance)

| 스킬 ID | 스킬 이름 | 설명 | 예시 명령 | 기본 활성화 |
|---------|----------|------|----------|:-----------:|
| `finance` | 가계부 | 지출·수입을 자동 기록하고 월별 합계를 조회합니다. | `점심 12,000원`, `이번 달 얼마 썼어?` | ✓ |
| `budget` | 예산 관리 | 카테고리별 월 예산을 설정하고 사용 현황을 조회합니다. | `식비 예산 30만원`, `예산 얼마 남았어?` | ✓ |
| `pattern` | 소비 패턴 분석 | 요일별 소비 경향, 반복 결제, 이상 소비를 백그라운드에서 자동 분석합니다. | `내 소비 패턴 어때?`, `이상한 지출 있어?` | ✓ |
| `proactive` | 능동 알림 | 예산 초과 경고, 이상 소비 감지 등 AI가 먼저 알림을 보냅니다. | (자동 발송, 직접 호출 없음) | ✓ |
| `currency` | 환율·환전 | 실시간 환율을 조회하고 통화를 변환합니다. | `달러 환율 알려줘`, `100달러 원화로 얼마야?` | ✓ |

## 스킬별 필요 권한

일부 스킬은 외부 API 키 또는 OAuth 인증이 필요합니다.

| 스킬 | 필요한 설정 | 설정 위치 |
|------|-----------|----------|
| `weather` | 날씨 API 키 | 서버 `.env` |
| `websearch` | Tavily API 키 | **설정 > 연동** |
| `naver_search` | 네이버 Client ID / Secret | **설정 > 연동** |
| `google` | Google OAuth2 인증 | **설정 > 연동** |
| `image` | Gemini API 키 (이미지 생성 시) | **설정 > 모델** |
| `audio` | Gemini API 키 (TTS 시) | **설정 > 모델** |
| `currency` | 환율 API (Frankfurter, 무료) | 자동 |
| `browser` | Playwright 설치 필요 | 서버 환경 |

## 자연어로 스킬 호출하기

스킬 이름을 직접 언급하지 않아도 됩니다. 자연스럽게 말하면 AI가 적절한 스킬을 자동으로 선택합니다.

```
나: 달러 5만원으로 바꾸면 얼마야?
봇: 현재 환율(1,340원/달러) 기준으로 50,000원 = 약 37.31달러입니다.

나: 런던 지금 몇 시야?
봇: 런던(GMT) 현재 시간은 오전 6:42입니다.

나: 오늘 기분이 너무 안 좋아. 팀장한테 혼났어.
봇: 오늘 힘드셨겠네요. 일기에 기록해 두었습니다.

나: 내일 오후 3시에 병원 예약 잊지 말라고 알려줘.
봇: 내일 오후 3시에 "병원 예약" 알림을 설정했습니다.

나: 1~45에서 로또 번호 6개 뽑아줘.
봇: 🎰 추첨 결과: 7, 14, 23, 31, 38, 42

나: 이 이미지에 뭐가 있어? (사진 첨부)
봇: 커피숍 영수증으로 보이며, 아메리카노 4,500원이 결제되었습니다. 가계부에 기록할까요?

나: https://news.ycombinator.com 요약해줘.
봇: 해당 페이지를 요약했습니다. 현재 주요 게시물: ...
```

## 스킬 복합 활용

하나의 메시지로 여러 스킬이 순서대로 실행될 수 있습니다.

| 예시 | 동작하는 스킬 |
|------|-------------|
| 영수증 사진 전송 | `image` → 금액 추출 → `finance` → 자동 기록 |
| 음성 메시지 "점심 만원" | `audio` → 텍스트 변환 → `finance` → 가계부 기록 |
| "내 소비 패턴 이상한 거 있어?" | `pattern` 분석 결과 → `memory` 검색 → 답변 |
| "보고서 PDF로 만들어줘" | LLM 내용 작성 → `documents` → PDF 파일 전달 |
