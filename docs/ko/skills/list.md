# 스킬 목록

Starnion에는 현재 **4개**의 스킬이 내장되어 있습니다. 각 스킬은 독립된 모듈로, 필요에 따라 개별적으로 활성화하거나 비활성화할 수 있습니다.

---

## 재무 (Finance)

| 스킬 ID | 스킬 이름 | 설명 | 예시 명령 | 기본 활성화 |
|---------|----------|------|----------|:-----------:|
| `finance` | 가계부 | 지출·수입을 자동 기록하고 월별 합계를 조회합니다. | `점심 12,000원`, `이번 달 얼마 썼어?` | ✓ |

---

## 개인 기록 (Personal)

| 스킬 ID | 스킬 이름 | 설명 | 예시 명령 | 기본 활성화 |
|---------|----------|------|----------|:-----------:|
| `diary` | 일기 | 일상·감정·생각을 기록합니다. | `오늘 기분이 너무 안 좋아`, `회의가 길었어` | ✓ |
| `goals` | 목표 관리 | 개인 목표를 설정하고 진행 상황을 추적합니다. | `이번 달 운동 목표 설정`, `목표 달성했어!` | ✓ |

---

## 미디어 (Media)

| 스킬 ID | 스킬 이름 | 설명 | 예시 명령 | 기본 활성화 |
|---------|----------|------|----------|:-----------:|
| `image` | 이미지 분석·생성 | 이미지를 분석하고 AI로 생성합니다. 영수증에서 지출 자동 추출을 지원합니다. | `이미지 분석해줘`, `고양이 이미지 만들어줘`, `영수증 정리해줘` | ✓ |
