# Jiki Web UI — 기능 분석 및 페이지 설계

## 개요

33개 스킬, 63개 도구, 전체 기능을 커버하는 Web UI 설계 문서.
Telegram Bot 기능을 웹 인터페이스로 제공하며, 대시보드·채팅·리포트·통계·설정 등을 포함한다.

**기반 데이터**: `docs/skill-registry.md`, `docs/agent-proactive.md`, `docs/autonomous-scheduling.md`, `docs/hybrid-search.md`, `docs/idle-time-analysis.md`

**지원 리포트 유형**:
| report_type | 설명 | 트리거 |
|-------------|------|--------|
| `daily_summary` | 일간 지출 요약 | 매일 저녁 (스케줄러) |
| `weekly` | 주간 리포트 | 매주 일요일 |
| `monthly_closing` | 월간 마감 | 매월 말일 |
| `spending_anomaly` | 소비 이상 탐지 | 실시간 |
| `budget_warning` | 예산 경고 | 임계값 초과 시 |
| `pattern_analysis` | 패턴 분석 | 매일 오전 6시 |
| `pattern_insight` | 패턴 인사이트 | 주기적 |
| `goal_evaluate` | 목표 평가 | 주기적 |
| `goal_status` | 목표 현황 | 요청 시 |
| `conversation_analysis` | 대화 분석 | 유휴 30분~2시간 후 |
| `memory_compaction` | 메모리 압축 요약 | 매주 월요일 오전 5시 |

**구성 컴포넌트**:
- 리포트 유형 카드 (활성/비활성, 바로 생성 버튼)
- 리포트 목록 테이블
- 마크다운 리포트 뷰어
- PDF/DOCX 다운로드
